```
## C[edit]

This is a simple implementation that uses the `rand()` function to generate random numbers between 0 and 9. It then generates a random number between 0 and 4 for each row and column in the matrix. The resulting matrix is printed out using the `printf()` function.

```
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand((unsigned int) time(NULL)); // seed rand with current time
    int i, j;
    int m[5][5];
    for (i = 0; i < 5; ++i) {
        for (j = 0; j < 5; ++j) {
            m[i][j] = rand() % 10;
        }
    }
    printf("matrix:\n");
    for (i = 0; i < 5; ++i) {
        for (j = 0; j < 5; ++j) {
            printf("%d ", m[i][j]);
        }
        printf("\n");
    }
}
