## C[edit]

This is a simple implementation that uses the `rand()` function to generate random numbers for each cell in the grid. It then prints out the grid and asks the user if they want to continue or not. If yes, it generates another grid and continues until the user says no.

```
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand((unsigned int) time(NULL)); // seed rand with current time
    int i, j;
    int grid[4][4];
    char answer = 'y';
    while (answer == 'y') {
        printf("Grid:\n");
        for (i=0; i<4; i++) {
            for (j=0; j<4; j++) {
                grid[i][j] = rand() % 2 + 1;
                printf("%d ", grid[i][j]);
            }
            printf("\n");
        }
        printf("Continue? [Y/N]: ");
        scanf(" %c", &answer);
        getchar();
    }
}
