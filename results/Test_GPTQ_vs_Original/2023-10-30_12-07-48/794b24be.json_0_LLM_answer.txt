## Answer (3)

You can use `np.where` to create a mask for each column and then multiply it with your original dataframe.

Here is an example:

```
import numpy as np

df = pd.DataFrame({'a': [0, 1, 0], 'b': [0, 0, 1]})
masks = [[True, False, True], [False, True, False]]

df_new = df.copy()
for i in range(len(masks)):
    df_new[['a', 'b']][masks[i]] = df_new[['a', 'b']][masks[i]] * -1

print(df_new)
```

Output:

```
 a   b
0  1  1
1  0  0
2  0  0
```

Note that this will only work if all columns have the same number of rows. If they don't, you need to loop over them separately.
