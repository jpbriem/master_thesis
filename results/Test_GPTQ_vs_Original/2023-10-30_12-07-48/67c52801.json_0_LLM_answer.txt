```
import numpy as np

# create arrays from lists
X = np.array([[0, 0, 0, 0, 0, 0], [2, 2, 0, 6, 6, 0],...])

def f(x):
    return x**2 + x - 1

# apply function to X

# print results
print("X:", X)
print("y:", y)

```
