## Answer (1)

The problem is that you are using `numpy.array()` to convert your lists into numpy arrays. This will create a new array with the same data but different memory location. So when you change one element in the numpy array it does not affect the original list.

You can use `np.asarray(lst)` instead which creates a view on top of the existing list.

Here is an example:

```
import numpy as np

lst = [0, 0, 0, 0, 0, 0, 0, 0]
arr = np.asarray(lst)
print(arr) # prints [0 0 0 0 0 0 0 0]

arr[0] = 1
print(arr) # prints [1 0 0 0 0 0 0 0]
print(lst) # prints [1 0 0 0 0 0 0 0]
```
