You can use `apply()` to apply a function to each column of your data frame. Here is an example that should do what you want:

```
df <- data.frame(a = c(1, 2, 3), b = c(4, 5, 6))

# define a function that returns the sum of its arguments
sum_col <- function(x){sum(x)}

# apply this function to each column of df
df[, lapply(.SDcols, sum_col)]
#>   a b
#> 1 6 15
```

If you want to do this without using `lapply()`, you could use `sapply()` instead:

```
df[, sapply(.SDcols, sum_col)]
#>   a b
#> 1 6 15
```
