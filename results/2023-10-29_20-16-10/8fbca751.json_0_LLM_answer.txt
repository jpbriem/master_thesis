Then, compare each difference with zero using [`np.equal`](https://docs.scipy.org/doc/numpy-1.14.3/user/basics.indexing.html#boolean-array-indexing), which returns a boolean array that tells us whether or not the corresponding element was equal to zero. Finally, we sum up these booleans along the first axis to count how many times they were true for each row. This gives us the number of transitions from `0` to `1`, so we subtract one from them to get the number of transitions from `1` to `0`.
