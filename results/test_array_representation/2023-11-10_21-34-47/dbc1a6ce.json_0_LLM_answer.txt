Here's an example:

```
df = pd.DataFrame({'A': ['a', 'b', 'c', 'd'], 'B': [1, 2, 3, 4]})

df['C'] = df.groupby('A')['B'].transform(max)
print(df)
```

 A   B  C

1  b   2   4

3  d   4   4
```
