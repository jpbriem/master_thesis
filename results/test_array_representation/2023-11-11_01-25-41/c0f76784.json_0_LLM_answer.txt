## Answer (1)

I think you are overthinking this. You can just use a simple `for` loop to iterate through the rows and columns of your matrix. For each cell in the matrix, if it is not zero, then add one to the count for that row and column. Then, at the end, divide by two to get the number of non-zero cells in the row or column. Here's an example implementation:

```
def num_nonzeros(matrix):
    # Initialize counts to zero
    row_counts = [0] * len(matrix)
    col_counts = [0] * len(matrix[0])

    # Iterate through the rows and columns of the matrix
    for i in range(len(matrix)):
        for j in range(len(matrix[i])):
            if matrix[i][j]!= 0:
                row_counts[i] += 1
                col_counts[j] += 1

    # Divide by two to get the number of non-zero cells in each row/column
    return [row / 2 for row in row_counts], [col / 2 for col in col_counts]
```

This function takes a list of lists representing a matrix as input, and returns two lists containing the numbers of non-zero cells in each row and column of the matrix.
