Comment: I'm not sure what you mean by "the network is trained to recognize the pattern". The network doesn't know anything about patterns. It just learns how to map inputs to outputs based on training data. If your training data contains a lot of examples with the same pattern, then it will learn that pattern and be able to reproduce it in test data. But if there are no such examples in the training data, then it won't learn them. So you need to make sure that your training data includes all possible patterns that might occur in the real world.

Answer: You can use `tf.keras.layers.ConvLSTM2D` layer for this purpose.
