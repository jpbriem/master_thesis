Comment: I'm not sure what you mean by "the network is trained to recognize the pattern". The network doesn't know anything about patterns. It just learns how to map inputs to outputs. If it happens that there are some patterns in your data, then yes, the network will learn those patterns and be able to generalize them to new data. But if there aren't any patterns, or they are too complex for the network to find, then no, the network won't do well on test data.

Answer: You can use a convolutional neural network (CNN) with a single layer of filters. This type of CNN is called a \em{filter bank}.

The idea behind this approach is that each filter represents one possible feature that could exist in an image. For example, if we have an image of a cat, then one possible feature might be two black dots near the top left corner of the image. Another possible feature might be three white stripes running horizontally across the middle of the image. And so on...

Each filter has its own set of weights which determine how much importance should be given to each pixel when calculating whether or not it matches up with what was learned during training time. These weights are updated as more examples come into play; eventually they become very accurate at predicting whether something belongs together based solely off their appearance alone!

You can read more about this topic here: https://www.quora.com/How-do-I-use-a-convolutional-neural-network-to-detect-patterns
