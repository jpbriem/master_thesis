Comment: I'm not sure what you mean by "the network is trained to recognize the pattern". The network doesn't know anything about patterns. It just learns how to map inputs to outputs. If your training data contains a lot of examples with the same pattern, then it will learn that mapping. But if there are no such examples in the training set, or only very few, then it won't be able to do that. So you need to make sure that your training set has enough examples for all the different things you want the network to learn.
