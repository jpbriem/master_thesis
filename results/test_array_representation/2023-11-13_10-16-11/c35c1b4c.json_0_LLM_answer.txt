The first thing that jumps out at me is your use of nested loops. This is almost always a bad idea in Python because it makes your code slow and hard to read. Instead, you should be using vectorized operations whenever possible. For example, instead of this:

\begin{blockquote}

\begin{code}
for i in range(len(input_array)):
    for j in range(len(input_array[i])):
        if input_array[i][j] == 0:
            continue
        else:
           ...
\end{code}

\end{blockquote}

You could do something like this:

\begin{code}
import numpy as np
from scipy import spatial

def get_neighbors(x):
    return x - 1, x + 1

# Convert your list-of-lists to an array
input_array = np.asarray(input_array)

# Find all nonzero elements
nonzeros = input_array!= 0

# Get their coordinates
coords = np.where(nonzeros)

# Compute distances from each point to its neighbours
distances = []
for coord in coords:
    # Get the neighbouring points
    neighbors = [get_neighbor(coord[0]) for neighbor in get_neighbors(coord[1])]
    # Compute distance between them
    distances.append([spatial.distance.cdist(coord, n) for n in neighbors])

# Flatten the list of lists
distances = sum(distances, [])
\end{code}
