## Answer (1)

I think you are looking for `np.where()`. It returns the indices where a condition is met. You can then use those to index into your array and set values.

For example:

```
import numpy as np

# Create some data
data = np.array([[0, 0, 0], [0, 2, 3], [0, 3, 3]])

# Find all rows that have a value greater than zero in column 2
indices = np.where(data[:, 2] > 0)[0]

# Set the corresponding values in column 3 to 3
data[indices, 3] = 3

print(data)
```
