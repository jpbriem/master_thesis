## Answer (4)

The problem is that you are using `np.array()` to convert your numpy array into a list. This will create a new list with the same values as the numpy array. However, when you use `numpy.array(lst)`, it creates a new numpy array from the list.

You can fix this by converting your numpy array to a list before passing it to `numpy.array()`. You can do this with `list(numpy.array(lst))`.

Here is an example:

```
import numpy as np

# Create numpy array
arr = np.array([[1, 2], [3, 4]])

# Convert numpy array to list
lst = list(arr)

# Convert list back to numpy array
new_arr = np.array(lst)
print(new_arr)
```

Output:

```
array([[[1, 2],
        [3, 4]],
       [[1, 2],
        [3, 4]]])
```

Note that if you want to keep the original numpy array, you should pass the numpy array directly to `numpy.array()` instead of first converting it to a list and then converting it again.
