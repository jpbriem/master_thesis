I think this is because your `train_output` variable is being overwritten in each iteration of the loop. You should move it outside the loop.

Also, you can simplify your code to something like this:

```
def train(X, y):
    for i in range(len(y)):
        X_i = X[i]
        y_i = y[i]
        if y_i == 0:
            train_output += [0] * len(X_i)
        else:
            train_output += [1] * len(X_i)
    return train_output
```
