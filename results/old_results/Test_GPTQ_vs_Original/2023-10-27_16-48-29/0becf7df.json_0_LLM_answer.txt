```
## C[edit]

This is a simple implementation that uses the `rand()` function to generate random numbers in the range [0..9]. It does not use any external libraries and it should be easy to understand. The code has been tested on Linux with gcc version 5.4.0.

```
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand((unsigned int) time(NULL)); // seed the rand() generator
    for (int i = 0; i < 10; ++i) {
        printf("train input:\n");
        for (int j = 0; j < 10; ++j) {
            printf("%d ", rand() % 10);
        }
        printf("\ntest input:\n");
        for (int j = 0; j < 10; ++j) {
            printf("%d ", rand() % 10);
        }
        printf("\n\n");
    }
}
