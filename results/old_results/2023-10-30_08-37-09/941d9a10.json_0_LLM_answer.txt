## Answer (1)

You can use `apply()` to apply a function on each row and then `groupby()` to group them by their labels.

Here is an example:

```
df = pd.DataFrame({'label': ['A', 'B', 'C', 'D', 'E'],
                   'x': [0, 0, 0, 5, 0],
                   'y': [0, 0, 0, 0, 5]})

def f(row):
    return row['x'] + row['y']

df_grouped = df.groupby('label')[['x','y']].apply(f).reset_index()
print(df_grouped)
```
