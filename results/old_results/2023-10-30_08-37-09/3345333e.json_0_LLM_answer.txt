## Answer (4)

You can use `groupby` to group by the first column and then apply `sum` on each group.

```
df.groupby('col1')['col2'].sum()
```

If you want to get the sum for all groups except one, you can use `except_indexes`.

```
df.groupby('col1').apply(lambda x: x['col2'].sum())[~df.groupby('col1').index == df.groupby('col1').first().name]
```

--------------------

Here is how you can do this in your code:

```
import pandas as pd

# create sample dataframe
data = [
    {'col1': 'a', 'col2': 1},
    {'col1': 'b', 'col2': 2},
    {'col1': 'c', 'col2': 3},
    {'col1': 'd', 'col2': 4}
]
df = pd.DataFrame(data)

# group by col1 and sum col2
print(df.groupby('col1')['col2'].sum())

# get sum for all groups except one
print(df.groupby('col1').apply(lambda x: x['col2'].sum())[~df.groupby('col1').index == df.groupby('col1').first().name])
```
