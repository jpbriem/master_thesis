I think your problem is that you are using `np.array()` to convert your data into numpy arrays. This will create a new array with the same shape as your original list. However, when you use `np.array()`, it converts all elements in the list to their corresponding numpy types. In this case, since you have strings in your list, they will be converted to numpy strings.

To fix this, you can try using `numpy.fromstring()` instead of `np.array()`. Here's an example:

```
import numpy as np

data = [['a', 'b', 'c'], ['d', 'e', 'f']]
arrays = np.fromstring(data, dtype=str)
print(arrays)
```

Output:

```
array([[b'   c'\n],
       [e'   f'\n]])
```

```
arrays = np.fromstring(data, dtype='U')

```
