Comment: I'm not sure what you mean by "the network is unable to learn the pattern". What do you expect it to be able to do? It looks like it can detect a line of ones in any orientation. Is that not correct?

Answer: You are using `tf.nn.relu` as your activation function for all layers except the last one (which uses `sigmoid`). This means that the neurons will only activate if their inputs are positive. If they have negative or zero inputs then they won't fire at all.

This is why your network isn't learning anything - because the inputs are always zero! The first layer has no way of knowing whether there is an edge present or not, so it just outputs zeros. Then the second layer also gets zeros and does nothing with them. And so on...

You need to use a different type of activation function which allows negative values through unchanged. For example, try replacing `tf.nn.relu` with [`tf.tanh`](https://www.tensorflow.org/api_docs/python/tf/math/tanh).
