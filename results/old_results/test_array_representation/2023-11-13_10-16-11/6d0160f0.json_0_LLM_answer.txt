Answer: The problem with your code is that you are using `np.argmax` on an array of shape `(n_samples, n_features)`. This function returns the index of the maximum value along each axis (i.e., for each sample). So if you have two samples, one with all values equal to zero and another with all values equal to one, then this will return `[0, 0]` because both samples have their maximum at position 0.

What you want instead is to find the \em{global} maximum across all samples. To do so, use [`np.amax`](https://docs.scipy.org/doc/numpy-1.15.0/reference/generated/numpy.amax.html) which does exactly that.
