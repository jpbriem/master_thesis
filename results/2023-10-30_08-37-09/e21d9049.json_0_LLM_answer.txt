You can use `groupby` to group your dataframe by the first column and then apply a function on each group. Here is one way to do this:

```
df = df.groupby('A')['B'].apply(lambda x: x[x == x.min()])
```

This will give you a new DataFrame where each row corresponds to a unique value in 'A', and the 'B' column contains only the minimum value for that group.

If you want to keep all values in 'B' instead of just the minimum, you can replace `x[x == x.min()]` with `x`.
