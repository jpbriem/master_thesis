## Answer (4)

You can use `apply()` to apply a function to each row in your data frame and then `group_by()` to group them by their class. Then you can use `summarise()` to calculate the sum of each column for each group. Here is an example code snippet that should do what you want:

```
library(dplyr)

df %>%
  mutate(class = factor(class)) %>% # convert class from numeric to factor
  group_by(class) %>%
  summarise_all(sum) %>%
  ungroup()
```

This will give you a data frame with one row per class containing the sum of all values in each column for that class.

If you don't want to include the class variable in your output, you can remove it using `select(-class)`.
