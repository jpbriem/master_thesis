## Answer (4)

You can use a recursive function to fill in all possible paths from top-left to bottom-right. The base cases are when you reach an edge or hit a wall. If there's no wall and it's not at the edge then we have two choices - either go right or down. We keep track of the maximum value seen so far on this path using `max_val`. When we finally reach the end point, if our current value is greater than max_val, we update max_val with that value. Then we backtrack by restoring the original values and continue filling other paths.

```
def flood(grid):
    def dfs(i, j, val, max_val):
        if i == m or j == n or grid[i][j] == 0: return max_val
        temp = grid[i][j]
        grid[i][j] = val
        max_val = max(max_val, val)
        max_val = dfs(i + 1, j, val + 1, max_val)
        max_val = dfs(i, j + 1, val + 1, max_val)
        grid[i][j] = temp
        return max_val

    m, n = len(grid), len(grid[0])
    ans = []
    for i in range(m):
        for j in range(n):
            if grid[i][j]!= 0:
                ans.append(dfs(i, j, 1, float('-inf')))
    return ans
