You can use a recursive function to find all possible paths from top-left corner to bottom right. For each path you will have two possibilities - either go down or go right. If current cell contains number then add it to sum and mark this position as visited. When you reach bottom-right corner return found sum. In your example there are three such paths so we need to check which one gives us maximum value. Here is an implementation of this algorithm in Python:

```
def maxPathSum(grid):
    def dfs(i, j, grid, visited, cur_sum):
        if i == len(grid) or j == len(grid[0]):
            return cur_sum

        if grid[i][j]!= 0 and not visited[i][j]:
            visited[i][j] = True
            cur_sum += grid[i][j]

            res = max(dfs(i + 1, j, grid, visited, cur_sum),
                      dfs(i, j + 1, grid, visited, cur_sum))

            visited[i][j] = False
            cur_sum -= grid[i][j]

        else:
            res = max(dfs(i + 1, j, grid, visited, cur_sum),
                      dfs(i, j + 1, grid, visited, cur_sum))

        return res

    m, n = len(grid), len(grid[0])
    visited = [[False]*n for _ in range(m)]
    return dfs(0, 0, grid, visited, 0)
