## Answer (1)

You can use a recursive function to find all possible paths from top-left corner to bottom right. For each path you will have two possibilities - either go down or go right. If there are no more moves left then check if current cell has value of 7 and increment counter by one. After finding all paths return maximum found value. Here is an example in Python:

```
def maxPathSum(grid):
    def dfs(x, y):
        if x == len(grid) or y == len(grid[0]):
            return 0
        elif grid[x][y] == 7:
            return 1 + dfs(x+1, y) + dfs(x, y+1)
        else:
            return max(dfs(x+1, y), dfs(x, y+1))

    return dfs(0, 0)
