## Answer (4)

You can use a recursive function to find all paths from top-left corner to bottom right. If you reach a cell with value 3 then return False and if you reach a cell with value 0 or outside of grid then return True. For each path check whether it's possible to move left or up by checking previous cells in that direction. If both are possible then choose one randomly using random module. At last when you have reached bottom-right cell return True. This way you will get all valid paths. Now just replace 0s on those paths with 3s. Here is an implementation -

```
import random
def dfs(i, j, n, m):
    if i == n or j == m or arr[i][j] == 3:
        return False
    elif i == 0 and j == 0:
        return True
    elif i == 0:
        return dfs(i, j + 1, n, m)
    elif j == 0:
        return dfs(i + 1, j, n, m)
    else:
        return dfs(i + 1, j, n, m) or dfs(i, j + 1, n, m)

n = len(arr)
m = len(arr[0])
for i in range(n):
    for j in range(m):
        if arr[i][j] == 0 and dfs(i, j, n, m):
            arr[i][j] = 3
print(arr)
```

**Note:** The above solution doesn't work correctly because I didn't consider the fact that we need to fill only those zeros which are connected to the border. So here is another approach where we first mark all the borders as visited and then do BFS starting from any zero cell until we don't encounter any unvisited zero cell. In this process whenever we visit a new cell we change its value to 3.

```
from collections import deque
def bfs(i, j):
    q = deque()
    q.append((i, j))
    while q:
        x, y = q.popleft()
        if arr[x][y]!= 0 and not vis[x][y]:
            vis[x][y] = True
            arr[x][y] = 3
            if x > 0:
                q.append((x - 1, y))
            if y > 0:
                q.append((x, y - 1))
            if x < n - 1:
                q.append((x + 1, y))
            if y < m - 1:
                q.append((x, y + 1))

n = len(arr)
m = len(arr[0])
vis = [[False]*m for _ in range(n)]
# Marking borders as visited
for i in range(n):
    vis[i][0] = True
    vis[-1][i] = True
for i in range(m):
    vis[0][i] = True
    vis[-1][i] = True
# Finding connected components
for i in range(n):
    for j in range(m):
        if arr[i][j] == 0 and not vis[i][j]:
            bfs(i, j)
print(arr)
```
