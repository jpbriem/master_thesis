You can use a recursive function to fill in the grid. The idea is that you start at each point with value 4 and then move upwards until you reach an edge or another value of 4. Then you do the same thing moving downwards from this point. You also need to keep track of which points have already been visited so as not to revisit them. def flood(grid):
    def _flood(x, y):
        if x < 0 or x >= len(grid) or y < 0 or y >= len(grid[0]):
            return
        if grid[x][y] == 0:
            grid[x][y] = 5
            _flood(x - 1, y)
            _flood(x + 1, y)
            _flood(x, y - 1)
            _flood(x, y + 1)

    for i in range(len(grid)):
        for j in range(len(grid[i])):
            if grid[i][j] == 4:
                _flood(i, j)
