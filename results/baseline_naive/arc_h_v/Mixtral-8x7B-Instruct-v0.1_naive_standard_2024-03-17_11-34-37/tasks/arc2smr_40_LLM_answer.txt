## Answer (1)

You can use a recursive function to fill in the grid. The idea is that you start from each point with value 4 and try to move upwards/downwards until you reach an edge or another 4-value cell. You also need to keep track of which cells have been visited so as not to visit them again. Here's how it could look like:

```
def flood(grid, i, j, visited):
    if i < 0 or i >= len(grid) or j < 0 or j >= len(grid[i]) or grid[i][j]!= 4 or visited[i][j]:
        return
    visited[i][j] = True
    grid[i][j] = 1
    flood(grid, i - 1, j, visited)
    flood(grid, i + 1, j, visited)
    flood(grid, i, j - 1, visited)
    flood(grid, i, j + 1, visited)

def solve(grid):
    nrows, ncols = len(grid), len(grid[0])
    visited = [[False]*ncols for _ in range(nrows)]
    for i in range(nrows):
        for j in range(ncols):
            if grid[i][j] == 4:
                flood(grid, i, j, visited)
    return grid
```
