If you reach at any point where value is not zero then add that number in your current path and call the same function with new grid which has one less row or column depending on direction of movement. You will have to keep track of maximum sum found so far as well. Here's how it looks like -

```
def maxPathSum(grid):
    return _maxPathSum(grid, len(grid)-1, len(grid[0])-1, 0)

def _maxPathSum(grid, i, j, curr_sum):
    if i < 0 or j < 0: # out of bounds
        return curr_sum

    if grid[i][j] == 0: # no path here
        return _maxPathSum(grid, i-1, j, curr_sum) or \
               _maxPathSum(grid, i, j-1, curr_sum)

    val = grid[i][j]
    grid[i][j] = 0 # mark this cell visited
    curr_sum += val

    ans = max(_maxPathSum(grid, i-1, j, curr_sum),
              _maxPathSum(grid, i, j-1, curr_sum))

    grid[i][j] = val # backtracking step
    return ans
```
