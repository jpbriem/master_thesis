## Answer (1)

You can use a recursive function to find all paths from top-left corner to bottom right. For each path you will have two options - either go down or go right. If there are no more moves possible then check if current cell has value of 6 and add it to result list.

Here's an example implementation in Python:

```
def getPaths(grid):
    def dfs(x, y):
        if x == len(grid) or y == len(grid[0]) or grid[x][y] == 0:
            return []
        elif grid[x][y] == 6:
            return [[grid[x][y]]]
        else:
            res = []
            if x + 1 < len(grid):
                res += [[grid[x][y]] + p for p in dfs(x+1, y)]
            if y + 1 < len(grid[0]):
                res += [[grid[x][y]] + p for p in dfs(x, y+1)]
            return res

    return [p for sublist in [dfs(i, j) for i in range(len(grid)) for j in range(len(grid[0]))] for p in sublist]
