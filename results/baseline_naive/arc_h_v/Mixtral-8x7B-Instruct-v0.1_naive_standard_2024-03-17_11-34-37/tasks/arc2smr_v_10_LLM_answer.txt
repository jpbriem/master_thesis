## Answer (1)

You can use a recursive function to find all possible paths from top-left corner to bottom right. For each path you will add its value to the current cell and then move on to next one using two pointers technique. You need to keep track of maximum sum found so far as well. Here's how it looks like in Python:

```
def maxPathSum(grid):
    def dfs(i, j, cur_sum):
        if i == len(grid) or j == len(grid[0]):
            return cur_sum

        grid[i][j] += cur_sum
        res = max(dfs(i + 1, j, grid[i][j]), dfs(i, j + 1, grid[i][j]))
        grid[i][j] -= cur_sum

        return res

    m = float('-inf')
    for i in range(len(grid)):
        for j in range(len(grid[0])):
            if not i and not j:
                m = max(m, dfs(i, j, 0))

    return m

**EDIT:** I just realized that this solution doesn't work correctly when there are multiple cells with non-zero values at the beginning. The problem is that we don't know which ones should be included into our final answer. So here's an updated version where we first calculate total number of ways to reach every cell and only after that we pick the best one:

```
def countPaths(grid, dp):
    def dfs(i, j):
        if i == len(grid) or j == len(grid[0]):
            return 1

        if dp[i][j]:
            return dp[i][j]

        cnt = 0
        if i > 0:
            cnt += dfs(i - 1, j)
        if j > 0:
            cnt += dfs(i, j - 1)

        dp[i][j] = cnt
        return cnt

    for i in range(len(grid)):
        for j in range(len(grid[0])):
            if not i and not j:
                continue
            dfs(i, j)

    def dfs(i, j, cur_sum):
        if i == len(grid) or j == len(grid[0]):
            return cur_sum

        grid[i][j] += cur_sum
        res = max(dfs(i + 1, j, grid[i][j]) * dp[i + 1][j], dfs(i, j + 1, grid[i][j]) * dp[i][j + 1])
        grid[i][j] -= cur_sum

        return res

    m = float('-inf')
    for i in range(len(grid)):
        for j in range(len(grid[0])):
            if not i and not j:
                continue
            m = max(m, dfs(i, j, 0))

    return m
