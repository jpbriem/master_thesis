```
## Answer (1)

You can use a recursive function to find all paths from top-left corner to bottom right. For each path you will have two options - either go down or go right. If there are no more moves possible in one direction then move only in another direction. When you reach the end of matrix add current path to result and return back.

Here's an example implementation using Python:

```
def getPaths(matrix):
    def dfs(x, y, curPath=[]):
        if x == len(matrix) or y == len(matrix[0]): # out of bounds
            return []
        elif matrix[x][y] == 0: # cannot step here
            return []
        elif x == len(matrix)-1 and y == len(matrix[0])-1: # reached destination
            return [curPath + [matrix[x][y]]]
        else:
            res = []
            if x < len(matrix)-1:
                res += dfs(x+1, y, curPath + [matrix[x][y]])
            if y < len(matrix[0])-1:
                res += dfs(x, y+1, curPath + [matrix[x][y]])
            return res

    return dfs(0, 0)
