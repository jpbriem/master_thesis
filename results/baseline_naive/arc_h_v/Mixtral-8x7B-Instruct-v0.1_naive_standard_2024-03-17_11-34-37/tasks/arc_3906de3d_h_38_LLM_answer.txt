## Answer (4)

You can use a recursive function to find all paths from top-left corner to bottom right. If you reach a cell with value of 7 then return False and stop traversing further in that path. For each cell check if it has been visited before or not. If it hasn't been visited before mark it as visited and call the function on next cells. If we are able to traverse through all cells without encountering any 7s then replace all non-visited cells with 0s. Here is an implementation using DFS approach -

```
def dfs(i, j, vis):
    n = len(grid)
    m = len(grid[0])
    # base cases
    if i == n or j == m:
        return True
    if grid[i][j] == 7:
        return False
    if vis[i][j]:
        return True
    vis[i][j] = True
    res = dfs(i + 1, j, vis) or dfs(i, j + 1, vis)
    if res:
        if grid[i][j]!= 1:
            grid[i][j] = 0
    else:
        grid[i][j] = 7
    return res

for row in range(len(grid)):
    for col in range(len(grid[0])):
        if grid[row][col] == 1:
            dfs(row, col, [[False]*m]*n)
```
