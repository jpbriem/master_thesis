## Answer (4)

You can use a recursive function to find all paths from top-left corner to bottom right. If you reach a cell with value 2 then return False because it's not possible to go further in this path. When you reach the last row or column and there are no more cells to visit then return True if current cell has value 0 otherwise return False. In other cases just check whether any of your neighbors have value 2 and if so return False. Otherwise continue visiting next cells by calling the same function again. Here is an example implementation using Python:

```
def validPath(grid):
    def dfs(r, c):
        if r == len(grid) or c == len(grid[0]): # out of bounds
            return grid[r][c] == 0
        if grid[r][c] == 2: # cannot move here
            return False
        nr = r + 1
        nc = c + 1
        if grid[nr][nc]!= 2: # down & right
            if dfs(nr, nc):
                return True
        if grid[nr][c]!= 2: # down
            if dfs(nr, c):
                return True
        if grid[r][nc]!= 2: # right
            if dfs(r, nc):
                return True
        return False
    return dfs(0, 0)

**Update:** To get the final matrix after removing invalid rows/columns you need to keep track of which columns were visited during DFS traversal. You can do that by adding another parameter to the helper function like this:

```
def validPath(grid):
    def dfs(r, c, cols):
        if r == len(grid) or c == len(grid[0]): # out of bounds
            return grid[r][c] == 0
        if grid[r][c] == 2: # cannot move here
            return False
        nr = r + 1
        nc = c + 1
        if grid[nr][nc]!= 2: # down & right
            if dfs(nr, nc, cols | (1 << c)):
                return True
        if grid[nr][c]!= 2: # down
            if dfs(nr, c, cols | (1 << c)):
                return True
        if grid[r][nc]!= 2: # right
            if dfs(r, nc, cols | (1 << c)):
                return True
        return False
    m = []
    for i, row in enumerate(grid):
        if dfs(i, 0, 0):
            m.append(row)
    return m
