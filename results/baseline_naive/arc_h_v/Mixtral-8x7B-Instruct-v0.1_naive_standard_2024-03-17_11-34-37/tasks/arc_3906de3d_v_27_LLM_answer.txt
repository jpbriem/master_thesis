## Answer (4)

You can use a recursive function to find all paths from top-left corner to bottom-right. If you reach a cell with value 9 then return False and stop traversing further in that path. For each cell if we have reached there by following any of its neighbours which has value 9 then also return False as it will not be possible to fill water in this cell now. Otherwise mark current cell as visited and check whether filling water in this cell makes the whole matrix filled or not using DFS. Return True if yes else False. At last if none of the cells are marked as visited then return False as no path exists between top-left and bottom-right corners.

**Code:**

```
def dfs(i, j, vis, mat):
    n = len(mat)
    m = len(mat[0])
    # Base Case
    if i == n - 1 and j == m - 1:
        return True

    # Check for invalid index
    if i < 0 or j < 0 or i >= n or j >= m:
        return False

    # If already visited or having value 9 then skip
    if vis[i][j] or mat[i][j] == 9:
        return False

    # Mark current cell as visited
    vis[i][j] = True

    # Traverse up, down, left and right
    ans = max(dfs(i + 1, j, vis, mat), dfs(i - 1, j, vis, mat),
              dfs(i, j + 1, vis, mat), dfs(i, j - 1, vis, mat))

    # Backtrack
    vis[i][j] = False

    return ans

def solve(mat):
    n = len(mat)
    m = len(mat[0])

    # Create a boolean grid to keep track of visited cells
    vis = [[False]*m for _ in range(n)]

    # Call DFS starting from first cell
    return dfs(0, 0, vis, mat)
