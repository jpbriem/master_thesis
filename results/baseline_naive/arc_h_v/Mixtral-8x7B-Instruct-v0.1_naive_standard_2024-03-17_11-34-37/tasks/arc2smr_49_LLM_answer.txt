## Answer (1)

You can use a recursive function to find all possible paths from top-left corner to bottom right. For each path you will add its value to the final result and then remove it by setting that cell's value to zero. After finding all paths you need to restore original values of cells so they could be used in next iteration. Here is an example implementation using DFS algorithm:

```
def dfs(grid, i, j):
    if not grid or i < 0 or j < 0 or i >= len(grid) or j >= len(grid[i]) or grid[i][j] == -1:
        return 0
    val = grid[i][j]
    grid[i][j] = -1
    res = max(dfs(grid, i + 1, j), dfs(grid, i - 1, j), dfs(grid, i, j + 1), dfs(grid, i, j - 1)) + val
    grid[i][j] = val
    return res

def solve(board):
    n = len(board)
    m = len(board[0])
    for i in range(n):
        for j in range(m):
            if board[i][j]!= 0:
                board[i][j] += dfs(board, i + 1, j)
                board[i][j] += dfs(board, i - 1, j)
                board[i][j] += dfs(board, i, j + 1)
                board[i][j] += dfs(board, i, j - 1)
