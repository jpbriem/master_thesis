If you reach at any point where value of matrix is not equal to 5 then return False and if you have reached at last cell i.e., bottom right cell then return True. You need to check that whether path exists or not before filling it with water. Here's my solution in python:

```
def fill(matrix):
    n = len(matrix)
    m = len(matrix[0])
    def dfs(i, j):
        if i == n - 1 and j == m - 1:
            return True
        elif i >= n or j >= m:
            return False
        elif matrix[i][j]!= 5:
            return False
        else:
            matrix[i][j] = '.'
            ans = dfs(i + 1, j) or dfs(i, j + 1)
            if ans:
                matrix[i][j] = 'W'
            return ans
    for i in range(n):
        for j in range(m):
            if matrix[i][j] == 5:
                dfs(i, j)
    return matrix
```
