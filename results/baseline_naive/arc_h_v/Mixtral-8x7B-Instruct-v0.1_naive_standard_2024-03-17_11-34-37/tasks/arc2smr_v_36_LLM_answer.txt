You can use a recursive function to find all possible paths from top-left corner to bottom right. If you reach at any point where value of matrix is not zero then increment count by one and move in four directions(up, down, left, right). You will get all possible path using this approach. Now just check if there are two consecutive zeros or more than two consecutive zeros in each path. If yes then return false otherwise true. Here is my code :

```
public static boolean validPath(int[][] grid){
    int m = grid.length;
    int n = grid[0].length;
    return helper(grid,m-1,n-1);
}
private static boolean helper(int[][] grid, int i, int j){
    if(i<0 || j<0 || grid[i][j]==-1)return false;
    if(i==0 && j==0)return true;
    if(grid[i][j]==0)return false;
    int temp = grid[i][j];
    grid[i][j]=-1;
    boolean up = helper(grid,i-1,j);
    boolean down = helper(grid,i+1,j);
    boolean left = helper(grid,i,j-1);
    boolean right = helper(grid,i,j+1);
    grid[i][j]=temp;
    if(!up &&!down &&!left &&!right)return false;
    if(temp>1 && (up||down||left||right))return false;
    return true;
}
```

Here I am returning false when there are two consecutive zeros or more than two consecutive zeros but you can modify it according to your requirement. Also note that here I have used -1 as invalid cell so make sure none of the cells contain -1 before passing it to this method.
