## Answer (4)

You can use a recursive function to find all paths from top-left corner to bottom right. For each path you will multiply the values of cells in that path and add it to your result. Here's how I would do this:

```
def getPaths(grid, i=0, j=0):
    if i == len(grid)-1 and j == len(grid[i])-1: # base case
        return [grid[i][j]]
    elif i >= len(grid) or j >= len(grid[i]): # out of bounds
        return []
    else:
        paths = []
        if grid[i][j]!= 0: # not zero
            paths += [grid[i][j] + x for x in getPaths(grid, i+1, j)]
            paths += [grid[i][j] + x for x in getPaths(grid, i, j+1)]
        else: # zero
            paths += getPaths(grid, i+1, j)
            paths += getPaths(grid, i, j+1)
        return paths

def maxPathSum(grid):
    return sum([max(getPaths(grid)) for _ in range(len(grid)*len(grid[0]))])
```

This works by finding all possible paths through the matrix using depth first search. It then multiplies the value of every cell on each path together and adds them up. This gives us the maximum product we could make with any number of paths.
