Here's code snippet in python -

```
def solve(matrix, i, j, n, m):
    if i == n or j == m:
        return 0
    if matrix[i][j] == 4:
        return 1 + solve(matrix, i+1, j, n, m) + solve(matrix, i, j+1, n, m)
    else:
        return solve(matrix, i+1, j, n, m) + solve(matrix, i, j+1, n, m)

n = len(matrix)
m = len(matrix[0])
print(solve(matrix, 0, 0, n, m))
```

Time complexity : O(2^N * M)
Space Complexity : O(N*M)

**Edit:**

If we want to print all possible paths as well then we need to modify above solution slightly. We will pass an array 'path' which stores current path taken so far. Whenever we encounter 4, we will append it to our final answer list. Here's modified code snippet -

```
def solve(matrix, i, j, n, m, path=[], ans=[]):
    if i == n or j == m:
        return []
    if matrix[i][j] == 4:
        ans.append(list(path)+[4])
    path += [matrix[i][j]]
    ans += solve(matrix, i+1, j, n, m, path, ans)
    ans += solve(matrix, i, j+1, n, m, path, ans)
    path.pop()
    return ans

n = len(matrix)
m = len(matrix[0])
ans = solve(matrix, 0, 0, n, m)
for x in ans:
    print(x)
```
