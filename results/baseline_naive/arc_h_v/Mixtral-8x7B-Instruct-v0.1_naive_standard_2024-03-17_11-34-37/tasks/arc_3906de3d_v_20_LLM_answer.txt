## Answer (1)

You can use a recursive function to find all possible paths from top-left corner to bottom right. If you reach at any point where value of cell is not equal to zero and it's not visited before then return false because this path will lead us outside grid or we have already been here so no need to go again in same direction. When you reach at last point which has value as zero then increment count by one. At end return true if count is greater than maxCount otherwise return false.

Here is code snippet that should work -

```
public static boolean solve(int[][] board, int i, int j, int m, int n, int maxCount, List<Integer> list){
    //base condition when out of bounds
    if(i < 0 || j < 0 || i >= m || j >= n)
        return false;

    //if current position contains zero then continue with next position
    if(board[i][j] == 0)
        return solve(board, i + 1, j, m, n, maxCount, list);

    //if current position is already visited then return false
    if(list.contains(i * n + j))
        return false;

    //marking current position as visited
    list.add(i * n + j);

    //reaching at last point with zero value means valid path found
    if((i == m - 1 && j == n - 1) && board[i][j] == 0) {
        maxCount = Math.max(maxCount, list.size());
        return true;
    }

    //checking four directions
    if(solve(board, i + 1, j, m, n, maxCount, list) || solve(board, i, j + 1, m, n, maxCount, list) || solve(board, i - 1, j, m, n, maxCount, list) || solve(board, i, j - 1, m, n, maxCount, list)){
        maxCount = Math.max(maxCount, list.size());
        return true;
    }

    //backtracking step
    list.remove(new Integer(i * n + j));
    return false;
}

public static int getMaxPaths(int[][] board){
    int m = board.length;
    int n = board[0].length;
    List<Integer> list = new ArrayList<>();
    return solve(board, 0, 0, m, n, 0, list)? list.size() : 0;
}
```

**Time Complexity:** O(4^mn)

**Space Complexity:** O(m*n)
