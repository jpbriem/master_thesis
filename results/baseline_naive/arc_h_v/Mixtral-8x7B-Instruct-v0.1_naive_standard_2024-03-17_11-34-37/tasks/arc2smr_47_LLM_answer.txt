You can use a recursive function to fill in the grid. The idea is that you start from the top-left corner and move right until you find a non-zero number or reach the end of the row. If you found a non-zero number then you go down one row and repeat this process. When you reach the bottom-right corner you are done. Here's how it looks like implemented in Python:

```
def flood(grid):
    def _flood(x, y):
        if x == len(grid[0]) or y == len(grid) or grid[y][x]!= 7:
            return
        grid[y][x] = 3
        _flood(x + 1, y)
        _flood(x, y + 1)

    _flood(0, 0)
