## Answer (1)

You can use a recursive function to find all paths from top-left corner to bottom right. For each path you will add its length to the matrix at the corresponding cells. The final result will be the sum of all these matrices. Here's an example implementation in Python:

```
def countPaths(m, i, j, n, mn):
    if i == n - 1 and j == mn - 1:
        return [[1]]
    elif i >= n or j >= mn:
        return []
    else:
        res = []
        left = countPaths(m, i, j + 1, n, mn)
        up = countPaths(m, i + 1, j, n, mn)
        for l in left:
            for u in up:
                r = [[sum(x)] for x in zip(l, u)]
                res.append(r[0])
        if m[i][j]!= 0:
            res.append([1])
        return res

def fillMatrix(m, n, mn):
    mat = [[0 for _ in range(mn)] for _ in range(n)]
    for i in range(n):
        for j in range(mn):
            p = countPaths(m, i, j, n, mn)
            for k in range(len(p)):
                mat[i][j] += len(p[k])
    return mat
```
