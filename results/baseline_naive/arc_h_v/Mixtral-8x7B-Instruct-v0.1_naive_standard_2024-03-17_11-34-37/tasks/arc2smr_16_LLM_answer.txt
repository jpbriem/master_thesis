You can use a recursive function to fill in the grid. The idea is that you start from each cell with value 3 and try to move up/down/left/right until you reach an edge or another cell with value 3. You keep track of all visited cells so as not to visit them again. Here's how it looks like:

```
def flood(grid, i, j, visited):
    if i < 0 or i >= len(grid) or j < 0 or j >= len(grid[i]) or grid[i][j]!= 3 or (i, j) in visited:
        return
    visited.add((i, j))
    grid[i][j] = 7
    flood(grid, i + 1, j, visited)
    flood(grid, i - 1, j, visited)
    flood(grid, i, j + 1, visited)
    flood(grid, i, j - 1, visited)

def solve(grid):
    visited = set()
    for i in range(len(grid)):
        for j in range(len(grid[i])):
            if grid[i][j] == 3:
                flood(grid, i, j, visited)
    return grid
```
