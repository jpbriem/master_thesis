If there are no more cells in that direction - return back with the same value. When you reach the last cell of the grid - save it's maximum value.

Here is an example implementation using DFS algorithm:

```
def maxPathSum(grid):
    def dfs(r, c, curVal):
        if r == len(grid) or c == len(grid[0]): # out of bounds
            return curVal
        val = grid[r][c] + max(dfs(r+1, c, curVal), dfs(r, c+1, curVal))
        grid[r][c] += val - curVal
        return val

    m = float('-inf')
    for i in range(len(grid)):
        for j in range(len(grid[i])):
            if not i and not j: # first cell
                m = max(m, dfs(i, j, 0))
    return m
