## Answer (1)

You can use a recursive function to find all possible paths from top-left corner to bottom right. For each path you will add its value to the result matrix at the current position and then move one step down or right depending on whether there are more columns or rows in that direction. If both directions have remaining cells you should choose randomly between them. Here's an example implementation of this idea using Python:

```
def dfs(matrix, i, j, nrows, ncols, res):
    if i == nrows - 1 and j == ncols - 1: # base case
        return matrix[i][j]
    elif i < nrows and j < ncols:
        val = 0
        if i + 1 < nrows:
            val += dfs(matrix, i+1, j, nrows, ncols, res)
        if j + 1 < ncols:
            val += dfs(matrix, i, j+1, nrows, ncols, res)
        res[i][j] += val
        return val
    else:
        raise Exception("Invalid input")

def maxPathSum(matrix):
    m = len(matrix)
    n = len(matrix[0])
    res = [[0]*n for _ in range(m)]
    dfs(matrix, 0, 0, m, n, res)
    return res
```

This solution has time complexity O(2^(min(m,n))), where m and n are number of rows and columns respectively. This is because it explores every possible path from top left to bottom right. Space complexity is also O(min(m,n)), which is used by the recursion stack.

**Update:** I just realized that your problem is actually simpler than what my original answer solves. You don't need to explore all paths but only those with maximum sum. So here's another version of the code above that does exactly that:

```
def dfs(matrix, i, j, nrows, ncols, res):
    if i == nrows - 1 and j == ncols - 1: # base case
        return matrix[i][j]
    elif i < nrows and j < ncols:
        val = 0
        if i + 1 < nrows:
            rval = dfs(matrix, i+1, j, nrows, ncols, res)
            if j + 1 < ncols:
                val = max(rval, dfs(matrix, i, j+1, nrows, ncols, res))
            else:
                val = rval
        else:
            if j + 1 < ncols:
                val = dfs(matrix, i, j+1, nrows, ncols, res)
            else:
                val = 0
        res[i][j] += val
        return val
    else:
        raise Exception("Invalid input")

def maxPathSum(matrix):
    m = len(matrix)
    n = len(matrix[0])
    res = [[0]*n for _ in range(m)]
    dfs(matrix, 0, 0, m, n, res)
    return res
```

The difference is that now we first try going down and only if there are no more rows do we go right. Also note that since we always take the maximum of two values, the order doesn't matter so we don't need to randomize anymore. The time complexity remains the same as before but space complexity is reduced to O(min(m,n)).
