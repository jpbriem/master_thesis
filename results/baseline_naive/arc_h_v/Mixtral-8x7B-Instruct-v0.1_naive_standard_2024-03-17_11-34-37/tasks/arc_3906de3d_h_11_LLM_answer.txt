## Answer (4)

You can use a recursive function to fill in all of the water. The idea is that you start at the bottom-right corner and work your way upwards and leftwards until you reach an edge or a wall. If there's no water below the current cell then it must be dry. Otherwise, if the current cell has already been filled with water then we skip over it since we don't want to flood the same area twice. Finally, if the current cell contains land but hasn't yet been flooded then we mark it as wet by filling it with water and then recursively call our function on its four neighbors. Here's what this looks like in code:

def fill(grid):
    def dfs(x, y):
        # Base cases: out of bounds or hit a wall
        if x < 0 or y < 0 or x >= len(grid[0]) or y >= len(grid) or grid[y][x] == "#":
            return

        # Skip cells that have already been filled with water
        if grid[y][x] == "~":
            return

        # Mark this cell as being filled with water
        grid[y][x] = "~"

        # Recursively explore each neighboring cell
        dfs(x - 1, y)
        dfs(x + 1, y)
        dfs(x, y - 1)
        dfs(x, y + 1)

    # Start from the lowest point that isn't a wall
    for row in reversed(grid):
        for i, cell in enumerate(row):
            if cell!= "#":
                dfs(i, len(row) - 1)
                break
