## Answer (4)

You can use a recursive function to find all paths from top-left corner to bottom-right. If you reach a cell with value 2 then return False and if you reach a cell with value 0 or out of bounds then return True. For each path check whether it's possible to go right or down by checking the corresponding cells in matrix. If both are possible then call the function twice once for going right and another time for going down. Return True only when both calls return True. Here is an implementation using python:

```
def solve(matrix):
    def dfs(i, j):
        if i == len(matrix) or j == len(matrix[0]):
            return True
        if matrix[i][j] == 2:
            return False
        if matrix[i][j] == 0:
            return True
        if i < len(matrix)-1 and dfs(i+1, j):
            matrix[i][j] = 0
            return True
        if j < len(matrix[0])-1 and dfs(i, j+1):
            matrix[i][j] = 0
            return True
        return False
    for i in range(len(matrix)):
        for j in range(len(matrix[0])):
            if dfs(i, j):
                break
        else:
            continue
        break
    return matrix
