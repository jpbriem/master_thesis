## Answer (4)

You can use a recursive function to find all possible paths from top-left corner to bottom right. If you reach an obstacle or out of bounds then return False else if you have reached the destination cell then return True. For each path check whether it has visited any cell twice and if yes then update that cell with 'X'. Finally print the matrix after updating all such cells.

Here's the code in Python -

```
def solve(i, j, m, n, vis):
    # base cases
    if i == m - 1 and j == n - 1:
        return True
    if i >= m or j >= n or vis[i][j] == 1:
        return False

    vis[i][j] = 1
    res = ((solve(i + 1, j, m, n, vis)) or
           (solve(i, j + 1, m, n, vis)))
    if not res:
        vis[i][j] = 0
    return res

def markPaths(m, n, mat):
    vis = [[0 for _ in range(n)] for _ in range(m)]
    for i in range(m):
        for j in range(n):
            if mat[i][j]!= 0 and solve(i, j, m, n, vis):
                for x in range(i, -1, -1):
                    for y in range(j, -1, -1):
                        if mat[x][y]!= 0 and vis[x][y]:
                            mat[x][y] = 'X'
    for row in mat:
        print(*row)
