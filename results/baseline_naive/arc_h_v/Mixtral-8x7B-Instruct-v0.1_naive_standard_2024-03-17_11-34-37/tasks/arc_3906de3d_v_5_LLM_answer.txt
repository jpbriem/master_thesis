## Answer (4)

You can use a recursive function to find all paths from top-left corner to bottom right. If you reach a cell with value 2 then return True and if any of the adjacent cells has already been visited or its value is less than 2 then don't visit that cell again. Here is my solution in Python:

```
def dfs(i, j, grid):
    n = len(grid)
    m = len(grid[0])
    if i == n - 1 and j == m - 1:
        return grid[i][j] == 2
    if i >= n or j >= m or grid[i][j] < 2:
        return False
    temp = grid[i][j]
    grid[i][j] = 0
    res = (dfs(i + 1, j, grid) or dfs(i, j + 1, grid))
    grid[i][j] = temp
    return res

def solve(grid):
    n = len(grid)
    m = len(grid[0])
    for i in range(n):
        for j in range(m):
            if grid[i][j] == 2:
                grid[i][j] = 0
                if dfs(i, j, grid):
                    return grid
    return None
```
