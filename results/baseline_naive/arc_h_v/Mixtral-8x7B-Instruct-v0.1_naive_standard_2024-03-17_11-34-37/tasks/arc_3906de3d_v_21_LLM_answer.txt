## Answer (4)

You can use a recursive function to find all possible paths from top-left corner to bottom right. If you reach at any point where value is not equal to 5 then return False and if you reach at bottom right with value as 5 then return True. You will have to check whether there exists some path or not by checking that after reaching last row/column we are able to get 5 or not.

Here's my solution in Python:

```
def solve(grid):
    def dfs(i, j):
        if i == len(grid)-1 and j == len(grid[0])-1:
            return grid[i][j]==5
        elif i >= len(grid) or j >= len(grid[0]):
            return False
        else:
            val = grid[i][j]
            if val!= 5:
                return False
            temp = []
            if i+1 < len(grid):
                temp.append(dfs(i+1, j))
            if j+1 < len(grid[0]):
                temp.append(dfs(i, j+1))
            return any(temp)

    n = len(grid)
    m = len(grid[0])
    for i in range(n):
        if dfs(i, 0):
            break
    else:
        return [-1]*n*m

    for j in range(m):
        if dfs(0, j):
            break
    else:
        return [-1]*n*m

    ans = [[-1]*m for _ in range(n)]
    stack = [(0, 0)]
    while stack:
        x, y = stack[-1]
        ans[x][y] = grid[x][y]
        stack.pop()
        if x+1<len(grid) and ans[x+1][y]==-1 and grid[x+1][y]!=5:
            stack.append((x+1, y))
        if y+1<len(grid[0]) and ans[x][y+1]==-1 and grid[x][y+1]!=5:
            stack.append((x, y+1))

    return ans
```
