## Answer (1)

You can use a recursive function to find all paths from top-left corner to bottom right. If you reach a cell with value 4 and there are no more cells in that row or column then return False because it means we cannot go further. Otherwise if current cell has value 0 then mark it as visited by changing its value to -1 and call the function on next cell. Return True only when both starting and ending points have been reached. Here's how you can do this using Python:

```
def exist(grid):
    def dfs(i, j):
        if i == len(grid) or j == len(grid[0]): # out of bounds
            return False
        if grid[i][j] == 4 and (not i or not j or i == len(grid)-1 or j == len(grid[0])-1):
            return False
        if grid[i][j] == 0:
            grid[i][j] = -1
            res = any(dfs(k, j) or dfs(i, l) for k in range(i+1, len(grid)) for l in range(j+1, len(grid[0])))
            grid[i][j] = 0
            return res
        return bool(i and j and grid[i-1][j]!= 4 and grid[i][j-1]!= 4)

    return any(dfs(i, j) for i in range(len(grid)) for j in range(len(grid[0])))

def exist(grid):
    def dfs(i, j, path=[]):
        if i == len(grid) or j == len(grid[0]): # out of bounds
            return None
        if grid[i][j] == 4 and (not i or not j or i == len(grid)-1 or j == len(grid[0])-1):
            return None
        if grid[i][j] == 0:
            grid[i][j] = -1
            res = dfs(i+1, j, path + [(i, j)]) or dfs(i, j+1, path + [(i, j)])
            grid[i][j] = 0
            return res
        return bool(i and j and grid[i-1][j]!= 4 and grid[i][j-1]!= 4)

    p = dfs(0, 0)
    return p and [(p[0]+x, p[1]+y) for x, y in p[1]]
