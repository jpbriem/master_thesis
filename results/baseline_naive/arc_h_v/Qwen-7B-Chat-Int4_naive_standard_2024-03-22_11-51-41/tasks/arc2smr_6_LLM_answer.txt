```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        if not strs:
            return []
        
        result = [[] for _ in range(len(strs))]
        
        hash_table = collections.defaultdict(list)
        
        for s in strs:
            key = ''.join(sorted(s))
            
            hash_table[key].append(s)
        
        for k, v in hash_table.items():
            result[hash_table[k] == v].append(k)

        return result
```

Explanation:

We can use a dictionary to store the sorted strings as keys and their original strings as values. Then we iterate through each string in `strs`, sort it, and add it to its corresponding list in the dictionary.

Finally, we convert the dictionary of lists into a list of lists where each inner list contains all the strings with the same sorted version.

The time complexity is O(n * m), where n is the number of strings and m is the maximum length of any string. The space complexity is also O(n * m) due to the dictionary used to store the sorted strings.
