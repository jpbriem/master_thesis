def longest_common_subsequence(A, B):
    m = len(A)
    n = len(B)

    # Initialize a 2D array to store the lengths of common subsequences
    dp = [[0 for _ in range(n+1)] for _ in range(m+1)]

    # Fill in the first row and column with zeros
    for i in range(1, m+1):
        dp[i][0] = 0
    for j in range(1, n+1):
        dp[0][j] = 0

    # Fill in the rest of the matrix using dynamic programming
    for i in range(1, m+1):
        for j in range(1, n+1):

            if A[i-1] == B[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    # Find the length of the longest common subsequence
    lcs_length = dp[m][n]
    
    # Build the LCS itself by backtracking through the dp matrix
    lcs = []
    i = m
    j = n
    while i > 0 and j > 0:
        if A[i-1] == B[j-1]:
            lcs.append(A[i-1])
            i -= 1
            j -= 1
        elif dp[i-1][j] >= dp[i][j-1]:
            i -= 1
        else:
            j -= 1
    
    # Reverse the LCS to get it in the correct order
    return "".join(reversed(lcs))

The final value of `dp[m][n]` is the length of the longest common subsequence. We then use backtracking to build the actual LCS by starting at the bottom-right corner of the `dp` matrix and working our way backwards.
