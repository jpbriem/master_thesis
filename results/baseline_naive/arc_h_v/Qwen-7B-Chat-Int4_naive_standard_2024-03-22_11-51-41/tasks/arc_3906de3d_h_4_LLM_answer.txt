  1. Initialize a matrix `dp` of size `(n+1) x (m+1)` where `n` and `m` are the lengths of the input arrays.
  2. Set all elements in the first row and first column of `dp` to 0, since there is only one possible solution for an empty grid.
  3. For each element `i` in the second row of `dp`, set its value to the maximum of three values: the value in the cell immediately above it (`dp[i-1][j]`), the value in the cell directly to the left of it (`dp[i][j-1]`), and the sum of the value in the cell immediately above it and the value in the cell directly to the left of it (`dp[i-1][j-1] + dp[i][j]`).
  4. Repeat step 3 for each remaining element in the array.

The final value in the bottom-right corner of `dp` will be the maximum number of ways to arrange the given letters such that no two adjacent letters are the same.

Here's some Python code that implements this algorithm:

```
def longest_palindrome(matrix):
    n = len(matrix)
    m = len(matrix[0])
    dp = [[0]*m for _ in range(n)]
    
    # base cases
    dp[0][0] = 1
    
    # fill in the rest of the matrix
    for i in range(1, n):
        for j in range(1, m):
            if matrix[i][j] != matrix[i-1][j]:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]) + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
                
    return dp[-1][-1]
```
