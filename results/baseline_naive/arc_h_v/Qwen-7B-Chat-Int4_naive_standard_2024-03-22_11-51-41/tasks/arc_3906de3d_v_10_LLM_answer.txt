To solve this problem, we can use a recursive approach. The basic idea is to group the characters in each row into pairs that are either equal or adjacent and then swap them. We will keep track of the number of swaps made at each level of recursion.

Here's the Python code to implement this algorithm:

```python
def countSwaps(matrix):
    # base case: if matrix has only one row, return 0 (no need to swap anything)
    if len(matrix) == 1:
        return 0

    # initialize variables
    num_swaps = 0
    current_row = 0

    # iterate over rows
    for i in range(len(matrix)):
        # get current row
        row = matrix[i]
        
        # iterate over columns
        for j in range(1, len(row)-1):
            # check if current character is different from previous two characters
            if row[j] != row[j-1] and row[j] != row[j+1]:
                # swap characters
                row[j], row[j-1] = row[j-1], row[j]
                
                # increment number of swaps
                num_swaps += 1
                
                # recursively process next column
                num_swaps += countSwaps(matrix[:i]+matrix[i+1:])
                
                # reset current row
                current_row = i + 1
    
    # add number of swaps made in last column
    num_swaps += countSwaps(matrix[current_row:])
    
    return num_swaps
```
