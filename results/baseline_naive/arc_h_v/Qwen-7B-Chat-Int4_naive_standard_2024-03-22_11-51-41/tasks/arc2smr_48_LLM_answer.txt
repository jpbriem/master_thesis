```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        if not strs:
            return []
        
        ans = [[] for _ in range(len(strs))]
        
        for i in range(len(strs)):
            key = ''.join(sorted(strs[i]))
            ans[key].append(strs[i])
            
        return ans
```

In the above code, we first initialize an empty list `ans` to store the result. Then, we loop through each string in the input list `strs`. For each string, we convert it into a sorted version of itself using the built-in Python function `sorted()`, which returns a new sorted list of characters from the original string. We use this sorted version as a unique identifier for each string by converting it into a string representation (i.e., `''.join(sorted(strs[i]))`). If the sorted version is already present in the `ans` list, we append the original string to that list; otherwise, we create a new list and append the original string to it. Finally, we return the `ans` list containing all the groups of anagrams.
