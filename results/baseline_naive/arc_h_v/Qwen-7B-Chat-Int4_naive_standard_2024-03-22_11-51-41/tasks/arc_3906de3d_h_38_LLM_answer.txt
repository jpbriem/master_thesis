Then, for each element in the input array, we will check if it is equal to any of the elements that come before it. If so, we will update `dp[i]` by adding `dp[j-1]` where `j` is the index of the previous occurrence of the current element.

If the current element is not equal to any of the elements that came before it, we will set `dp[i]` to 1.

Finally, we will return the value at the last index of the `dp` array, which represents the total number of occurrences of the first element in the input array.

Here's the Python code:

```python
def longest_common_subsequence(arr):
    n = len(arr)
    dp = [0] * (n+1)

    for i in range(1, n+1):
        for j in range(i):
            if arr[i] == arr[j]:
                dp[i] = max(dp[i], dp[j]+1)

    return dp[-1]
```
