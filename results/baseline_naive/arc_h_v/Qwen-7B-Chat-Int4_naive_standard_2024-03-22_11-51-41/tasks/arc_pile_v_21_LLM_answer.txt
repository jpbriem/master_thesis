```python
def group_rows(input):
    n = len(input)
    groups = [[] for _ in range(n)]
    
    for i, row in enumerate(input):
        for j, cell in enumerate(row):
            if j == 0 or cell != input[i-1][j]:
                groups[i].append(cell)
                
    return [row for row in groups if row]
```

For each cell in a given row, we check whether it is the same as the previous cell in the same row (i.e., the cell at index `0`). If it is not, then we append the current cell to the corresponding group in `groups`.

Finally, we return the outer list of all groups by flattening it using a list comprehension.

This solution has a time complexity of O(n*m), where n is the number of rows in the input array and m is the maximum length of any row.
