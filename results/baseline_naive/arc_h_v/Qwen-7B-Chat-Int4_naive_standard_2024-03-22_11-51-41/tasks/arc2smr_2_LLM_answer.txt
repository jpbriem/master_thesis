```python
def find_paths(matrix):
    rows = len(matrix)
    cols = len(matrix[0])
    
    # create a 2D array to store the path information
    paths = [[False] * cols for _ in range(rows)]
    
    def dfs(r, c):
        if r < 0 or r >= rows or c < 0 or c >= cols or matrix[r][c] == '#':
            return
        
        paths[r][c] = True
        
        # mark this cell as visited and recursively traverse its neighbors
        dfs(r-1, c) if r > 0 else None
        dfs(r+1, c) if r < rows - 1 else None
        dfs(r, c-1) if c > 0 else None
        dfs(r, c+1) if c < cols - 1 else None
    
    dfs(0, 0)
    
    # print all the paths
    for i in range(rows):
        for j in range(cols):
            if paths[i][j]:
                print(f'({i}, {j})')
```

This solution uses depth-first search (DFS) to traverse the matrix. The `dfs` function takes two arguments, the row and column of the current cell, and marks it as visited. Then, it recursively traverses the four neighboring cells if they are not marked as visited and have an appropriate value. Once all the cells have been processed, we can print out all the valid paths by iterating over the `paths` array.

The time complexity of this algorithm is O(n*m), where n is the number of rows and m is the number of columns in the matrix. This is because each cell is visited once during DFS traversal. The space complexity is also O(n*m), since we need to store the path information for each cell in the `paths` array.
