```python
def longest_common_subsequence(A, B):
    m = len(A)
    n = len(B)

    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if A[i - 1] == B[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1

            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]
```

We initialize a 2D array `dp` with all elements set to zero. The first row and column are initialized to zero since there is no common element between an empty string and any other string.

Then we iterate over each character in `A` and `B`, and check if they are equal. If they are, we add one to the value at the previous diagonal cell. Otherwise, we take the maximum of the values at the current cell and the cell above it or to the left of it.

Finally, we return the value at the bottom-right corner of the matrix, which represents the length of the longest common subsequence between `A` and `B`.

Note that this implementation assumes that the input strings have the same length. If the lengths differ, you can modify the code to handle that case by adding additional logic to adjust the dimensions of the `dp` matrix accordingly.
