The basic idea is to iterate through the input array and compare adjacent strings. If two adjacent strings are similar, we add them to the current group. Otherwise, if the current group has at least three elements, we append it to the output list and start a new group with the next string.

Here's the Python code:

```python
def find_groupings(strings):
    groups = []
    for i in range(len(strings) - 1):
        if strings[i] == strings[i+1]:
            group = [strings[i]]
            while i < len(strings) - 1 and strings[i] == strings[i+1]:
                group.append(strings[i])
                i += 1
            if len(group) >= 3:
                groups.append(group)
    return groups
```
