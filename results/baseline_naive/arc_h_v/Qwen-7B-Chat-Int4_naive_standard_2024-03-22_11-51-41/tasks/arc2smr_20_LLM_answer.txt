```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        anagram_map = collections.defaultdict(list)
        
        for s in strs:
            sorted_s = "".join(sorted(s))
            anagram_map[sorted_s].append(s)

        return [anagram_map[i] for i in sorted(anagram_map.keys())]
```

- For each string `s`, we sort its characters and use that sorted string as key to access the list of strings associated with it in the `anagram_map`.
- Finally, we return a list of lists containing all the anagrams.
