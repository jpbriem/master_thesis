```python
class Solution:
    def groupAnagrams(self, strs):
        """
        :type strs: List[str]
        :rtype: List[List[str]]
        """
        groups = [[] for _ in range(len(set(strs)))]
        
        for s in strs:
            key = "".join(sorted(s))
            groups[key].append(s)
            
        return groups
```

In this solution, we first create an empty list called `groups` that will hold the result. We then loop through each string in the input list `strs`. For each string, we sort its characters and convert them into a string key using the `sorted()` function and the `"".join()` method. We then append the sorted string to the corresponding group in the `groups` list based on the key.

After looping through all the strings, we have created a list of lists where each inner list contains all the strings with the same sorted character representation. This is what we wanted, so we can simply return the `groups` list as our output.

The time complexity of this solution is O(n * m), where n is the number of strings in the input list and m is the maximum length of any string. The space complexity is also O(m) since we need to store all the unique sorted strings.
