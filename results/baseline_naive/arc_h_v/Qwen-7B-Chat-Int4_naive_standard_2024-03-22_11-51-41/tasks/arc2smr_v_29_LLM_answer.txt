We initialize the first row and column of the `dp` array with 0, since there are no cells to be reached from the top or left corners. Then, for each cell `(i,j)` in the matrix, we compute the minimum cost by considering all possible paths that start from the previous cell `(i-1,j)` and end at the current cell `(i,j)`. We add the cost of the path to the corresponding entry in the `dp` array.

```python
def min_cost(matrix):
    n = len(matrix)
    m = len(matrix[0])
    
    dp = [[0] * (m + 1) for _ in range(n + 1)]
    
    # Initialize the first row and column
    for i in range(1, n + 1):
        dp[i][0] = dp[i - 1][0]
        dp[0][j] = dp[0][j - 1] for j in range(1, m + 1)

    # Compute the minimum cost for each cell
    for i in range(1, n + 1):
        for j in range(1, m + 1):
            dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + matrix[i - 1][j - 1]

    return dp[n][m]
```
