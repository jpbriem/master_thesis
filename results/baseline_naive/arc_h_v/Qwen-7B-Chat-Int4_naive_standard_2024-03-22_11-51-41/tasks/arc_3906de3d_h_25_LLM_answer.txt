```python
def longest_common_subsequence(A, B):
    m = len(A)
    n = len(B)

    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m+1):
        for j in range(1, n+1):
            if A[i-1] == B[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    lcs_length = dp[m][n]
    lcs = []

    i = m
    j = n

    while i > 0 and j > 0:
        if A[i-1] == B[j-1]:
            lcs.append(A[i-1])
            i -= 1
            j -= 1
        elif dp[i-1][j] >= dp[i][j-1]:
            i -= 1
        else:
            j -= 1

    return ''.join(reversed(lcs))
```

This function takes two lists `A` and `B` as input and returns the length of the longest common subsequence between them. The output is a string.

The algorithm uses dynamic programming to build up a table `dp` where `dp[i][j]` represents the length of the longest common subsequence between the first `i` elements of `A` and the first `j` elements of `B`. We start by initializing all entries in `dp` to zero except for the top-left entry which is one because there must be at least one character that appears in both sequences.

We then iterate over each pair of indices `(i, j)` and compare the corresponding characters in `A` and `B`. If they are equal, we add one to the value in `dp[i-1][j-1]`, since the current character extends the previous longest common subsequence. Otherwise, we take the maximum of the values in `dp[i-1][j]` and `dp[i][j-1]`, since either of these could potentially be part of the longest common subsequence.

Once we have filled out the entire `dp` table, we can backtrack through it to find the actual longest common subsequence. Starting from the bottom-right corner, we move left or down depending on whether the characters at the current position match or not. If they do match, we append the matching character to our result list and move diagonally left and down. If they don't match, we move horizontally or vertically accordingly until we reach the top-left corner.

Finally, we reverse the order of the characters in our result list and return it as the final output.
