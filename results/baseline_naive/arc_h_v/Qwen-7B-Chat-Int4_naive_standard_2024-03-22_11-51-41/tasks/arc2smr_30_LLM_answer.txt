```python
def longest_common_subsequence(input):
    m = len(input[0])
    n = len(input)

    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if input[i - 1][j - 1] == 'a':
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]) + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    lcs = []
    i = m
    j = n

    while i > 0 and j > 0:
        if input[i - 1][j - 1] == 'a':
            lcs.append('a')
            i -= 1
            j -= 1
        elif dp[i - 1][j] >= dp[i][j - 1]:
            i -= 1
        else:
            j -= 1

    return ''.join(reversed(lcs))
```

This function uses dynamic programming to build a table of length `(m+1)` x `(n+1)` where `dp[i][j]` represents the length of the longest common subsequence between the first `i` characters of the first string and the first `j` characters of the second string. The function then iterates over this table to construct the longest common subsequence.

The time complexity of this algorithm is O(mn), where m and n are the lengths of the two strings. This is because we need to fill out an m x n table with values that depend on the previous values in the table. The space complexity is also O(mn).
