```python
class Solution:
    def findDuplicate(self, nums: List[List[int]]) -> List[List[int]]:
        seen = set()
        
        for i in range(len(nums)):
            if nums[i] not in seen:
                seen.add(nums[i])
                
                for j in range(i+1, len(nums)):
                    if nums[j] == nums[i]:
                        return nums[:j]
        return nums
```

This solution works by iterating over the input list of lists and keeping track of which numbers have already been seen. If a number is encountered that has not been seen before, it adds it to the set of seen numbers. Then, it checks if any other numbers in the list are equal to the current number. If so, it returns all the elements up to the index of the duplicate number. This way, we can find all the duplicates in the list without having to sort or modify the original list.
