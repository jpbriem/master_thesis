To solve this problem, we can iterate through each sublist in the input array and check if it contains any duplicate characters. If a sublist contains duplicate characters, we can replace them with an empty string to create a new sublist without duplicates.

Here's one way to implement this algorithm:

```
def remove_duplicates(input):
    output = []
    for sublist in input:
        # Check if the sublist contains any duplicates
        unique_chars = set(sublist)
        # Create a new sublist without duplicates
        new_sublist = [char for char in sublist if char in unique_chars]
        # Add the new sublist to the output list
        output.append(new_sublist)
    return output
```
