```python
class Solution:
    def groupAnagrams(self, strs):
        """
        :type strs: List[str]
        :rtype: List[List[str]]
        """
        if not strs:
            return []
        
        anagram_map = {}
        for s in strs:
            sorted_s = "".join(sorted(s))
            if sorted_s in anagram_map:
                anagram_map[sorted_s].append(s)
            else:
                anagram_map[sorted_s] = [s]
                
        result = []
        for k, v in anagram_map.items():
            result.append(v)
            
        return result
```

Explanation:

- We first check if the input list is empty. If it is, we simply return an empty list.
- Next, we initialize an empty dictionary `anagram_map` to store the anagrams as key-value pairs, where the key is the sorted string and the value is a list of strings that are anagrams of the key.
- We then iterate through each string `s` in the input list. For each string, we sort its characters and use the sorted string as the key in the `anagram_map`. If the key already exists in the map, we append `s` to the corresponding list of anagrams. Otherwise, we add `s` to a new list with the key.
- Finally, we convert the contents of the `anagram_map` into a list and return it.

This solution has a time complexity of O(n * m), where n is the number of strings in the input list and m is the maximum length of any string in the list. This is because we need to sort each string before adding it to the `anagram_map`, which takes O(m) time, and then we need to iterate through all the strings in the input list, which takes O(n) time. The space complexity of this solution is also O(n * m), since we need to store all the anagrams in the `anagram_map`.
