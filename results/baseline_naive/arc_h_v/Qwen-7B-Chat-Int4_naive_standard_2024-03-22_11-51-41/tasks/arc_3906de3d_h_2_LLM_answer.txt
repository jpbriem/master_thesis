def longest_common_subsequence(A, B):
    m = len(A)
    n = len(B)

    dp = [[0 for _ in range(n+1)] for _ in range(m+1)]

    for i in range(1, m+1):
        for j in range(1, n+1):
            if A[i-1] == B[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1

            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    return dp[m][n]
```

This solution uses dynamic programming to solve the problem. The time complexity of this solution is O(m * n), where m and n are the lengths of the input arrays A and B respectively.

To see how it works:

1. Create a two-dimensional array dp with dimensions (m+1) x (n+1). dp[i][j] will store the length of the longest common subsequence between the first i characters of A and the first j characters of B.
2. Initialize all elements of dp to 0.
3. Iterate through each character of A and B from right to left. If the current characters match, set dp[i][j] to dp[i-1][j-1] + 1. Otherwise, set dp[i][j] to the maximum value between dp[i-1][j] and dp[i][j-1].
4. After iterating through all characters, dp[m][n] will contain the length of the longest common subsequence between A and B.

B = ["j", "j", "j", "j", "a", "a", "a", "a", "a", "a"]

```
