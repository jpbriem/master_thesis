    m = len(arr)
    n = len(arr[0])
    dp = [[0 for _ in range(n+1)] for _ in range(m+1)]
    
    for i in range(1, m+1):
        for j in range(1, n+1):
            if arr[i-1][j-1] == '*':
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]) + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

The LCS is computed using dynamic programming. We create a two-dimensional array called `dp`, where `dp[i][j]` represents the length of the LCS between the first `i` rows and the first `j` columns of the input arrays.
