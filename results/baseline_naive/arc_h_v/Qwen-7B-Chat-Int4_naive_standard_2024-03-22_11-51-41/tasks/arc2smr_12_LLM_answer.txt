To solve this problem, we can use a dynamic programming approach. We will create an array of size n+1 to store the result of each subproblem, where n is the length of the input string.
The base cases are when the input string has only one character or two characters. In these cases, the result is simply that character.
For larger strings, we can iterate over all possible pairs of adjacent characters and check if they match. If they do, we update the result for the current substring. Otherwise, we keep the previous result.
Here's the Python code to implement this algorithm:

```python
def longest_common_substring(input):
    n = len(input)
    dp = [[0] * (n + 1) for _ in range(n + 1)]
    max_len = 0

    for i in range(1, n + 1):
        for j in range(i, n + 1):
            if input[i - 1] == input[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
                max_len = max(max_len, dp[i][j])

    return max_len
```
