```python
def longest_common_subsequence(input):
    m = len(input)
    n = len(max(input, key=len))
    
    dp = [[0 for _ in range(n+1)] for _ in range(m+1)]
    
    for i in range(1, m+1):
        for j in range(1, n+1):
            if input[i-1] == input[j-1]:
                dp[i][j] = dp[i-1][j-1]+1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
                
    return dp[m][n]
```

Explanation:

The above code uses dynamic programming to solve the problem. We create a 2D array `dp` of size `(m+1) x (n+1)` where `m` and `n` are the lengths of the two input strings.

We initialize all elements of `dp` to 0 except for the first row and column which we set to -1.

Then we iterate over each character in both strings from left to right. If the characters match, we add 1 to the value at the previous diagonal cell (i.e., `dp[i-1][j-1]`). Otherwise, we take the maximum value between the previous diagonal cell (`dp[i-1][j]`) and the current cell (`dp[i][j-1]`).

Finally, the value at the bottom-right corner of the matrix is the length of the longest common subsequence.

This solution has a time complexity of O(m*n) as we only need to iterate over each character once. It also has a space complexity of O(m*n) as we use an additional array to store the intermediate results.
