For each cell (i, j), if its value is equal to "a", we will keep it as it is. Otherwise, we will replace it with the maximum value among all cells in the same row or column.

Here's the Python code for the solution:

```python
def longest_common_subsequence(matrix):
    # Create an empty matrix of size n x m to store the result
    result = [[0] * len(matrix[0]) for _ in range(len(matrix))]

    # Function to compute the length of LCS between two substrings
    def lcs(s1, s2):
        m, n = len(s1), len(s2)
        dp = [[0] * (n + 1) for _ in range(m + 1)]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if s1[i - 1] == s2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1] + 1
                else:
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

        return dp[m][n]

    # Compute LCS between rows and columns
    for i in range(len(matrix)):
        for j in range(len(matrix[0])):
            if matrix[i][j] != "a":
                continue

            # Find LCS between current row and previous row
            prev_row_lcs = lcs(result[i - 1], matrix[i])
            # Find LCS between current column and previous column
            prev_col_lcs = lcs(matrix[i], result[j - 1])

            # Update LCS between current row and column
            result[i][j] = max(prev_row_lcs, prev_col_lcs)

    # Return the longest common subsequence
    return result[-1][-1]
```

The time complexity of this algorithm is O(n*m), where n and m are the dimensions of the input matrix. The space complexity is also O(n*m).
