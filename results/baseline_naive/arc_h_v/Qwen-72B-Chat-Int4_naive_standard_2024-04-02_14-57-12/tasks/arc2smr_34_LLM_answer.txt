```
def connect_dots(grid):
    rows = len(grid)
    cols = len(grid[0])
    
    for i in range(rows):
        for j in range(cols):
            if grid[i][j] == 'd':
                count = 0
                directions = [(i+1, j), (i-1, j), (i, j+1), (i, j-1)]
                
                for di, dj in directions:
                    if 0 <= di < rows and 0 <= dj < cols and grid[di][dj] == 'd':
                        count += 1
                
                if count >= 4:
                    grid[i][j] = 'b'
                    
    return grid
```

This function takes as input a 2D list representing the grid of dots. It first gets the dimensions of the grid using `len()` and then iterates over each cell in the grid. If a cell contains a dot ('d'), it counts how many adjacent cells also contain a dot by checking each of the four cardinal directions (up, down, left, right) and incrementing a counter if the adjacent cell is within bounds and also contains a dot.

If the count is greater than or equal to 4, indicating that there are at least 5 connected dots, the original dot is replaced with an asterisk ('b') to indicate that it has been "blasted".

The function returns the modified grid.
