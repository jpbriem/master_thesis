You can use a recursive function that takes in an array of arrays as input and returns another 2D array with all `'d'` replaced by `'f'`. The idea is simple - if there are no more elements left then return empty list else check for each element whether it contains any `'d'`, replace them with `'f'` and call this same method again passing remaining part of matrix. Here is my implementation using Java8 streams API :

```
public static List<List<String>> change(List<List<String>> mat){
    //base condition when we reach end of matrix
    if(mat == null || mat.isEmpty()){
        return new ArrayList<>();
    }else {
        int size = mat.size();//getting number of rows
        boolean flag = false;//flag variable used later below
        //iterating over every row
        for(int i=0 ;i < size &&!flag; ++i){
            //checking if current row has d or not
            if(!mat.get(i).contains("d")) continue;
            //if yes then iterate through column values replacing d with f
            for(int j=0;j<mat.get(i).size()&&!flag;++j){
                String val = mat.get(i).set(j,"f");
                if(val!= "d")continue;
                flag = true;
            }
        }
        //recursively calling our own method after changing value from d to f
        if(flag){
           return Stream.of(change(new ArrayList<>(mat.subList(0,i))),
                             change(new ArrayList<>(mat.subList(i+1,mat.size())))).flatMap(Collection::stream).collect(Collectors.toList());
       }else{
          return Collections.singletonList(mat);
      }
   }
}
