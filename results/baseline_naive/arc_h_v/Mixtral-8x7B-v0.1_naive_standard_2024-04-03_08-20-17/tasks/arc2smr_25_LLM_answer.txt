You can use a recursive function that takes in an array of arrays and returns another one with all `'f'` replaced by `'g'`. The base cases are when there is no more elements or if we reach the end of any row. Otherwise, check whether it should be changed into g and then call itself for each element around this position :

```
def replace(arr):
    def helper(i, j):
        return arr[i][j] == "." \
            or ((i > 0 and i < len(arr)-1) and (helper(i+1, j)))\
                +((j > 0 and j < len(arr[0]) - 1) and (helper(i, j+1)))

    res = []
    for line in arr:
        newline = ""
        for char in line:
            if char!= ".":
                newline += ("g" if not helper(res.index(newline), newline.index(char)) else "f")
            else:
                newline += "."
        res.append(newline)
    return res
