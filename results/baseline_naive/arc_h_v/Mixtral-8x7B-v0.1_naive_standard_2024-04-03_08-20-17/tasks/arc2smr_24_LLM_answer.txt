## Answer (0)

You can use `itertools.groupby()`. The idea is that we group by consecutive elements in a row which are equal or not equal to ".". Then for each such sublist of groups we check if it contains any non-"." element - this means there was an intersection between two circles somewhere along these rows. If yes, replace all dots with "d". Otherwise leave them as they were. Finally join back into one list again using `chain.from_iterable()`.

Here's what I mean:

import itertools
from operator import itemgetter
from functools import reduce
from pprint import pp

def find_intersections(board):
    def has_non_dot(row):
        return any("."!= x[0] for x in row)

    # Grouping together same characters within each row
    board = map(lambda r: ["."] + sorted(r), board)  # Add dot before first character
    grouper = lambda lst: len([*filter(has_non_dot, lst)]) > 0
    result = []
    for k, v in itertools.groupby(map(itemgetter(1), zip(*board)), key=grouper):
        if k == True:
            result += ['d'] * sum(len(x) for _, x in v)
        else:
            result += chain.from_iterable((["."]*len(x)) for _, x in v)
    return result[:-1]   # Remove last added dot from every line
