### 4. 找出最小的数字

输入：[7, 6, 4, 8, 9, 10]
输出：4
解释：4是罗列中数字之和最小的一个。

```python
class Solution(object):
    def minNumber(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        if not nums or len(nums) == 0:
            return None
        
        res = nums[-1]
        for i in range(len(nums)-1,-1,-1):
            if nums[i] < res:
                res = nums[i]
                
        return res
```
