Here's how you could implement it:

```
function transform(arr1, arr2) {
    return arr1.map((val, i) => arr2[i]);
}
```

If you want to keep the order of the original array, you can use `reduce()` instead:

```
function transform(arr1, arr2) {
    return arr1.reduce((acc, val, i) => [...acc,...arr2[i]], []);
}
```
