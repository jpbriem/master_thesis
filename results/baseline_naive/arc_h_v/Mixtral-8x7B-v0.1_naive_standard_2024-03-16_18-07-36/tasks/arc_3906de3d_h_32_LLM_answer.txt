```
You can use `itertools.groupby()`, which groups consecutive elements of an iterable by a key function that returns some value for each element in the input sequence. In this example we group all values with same index together into tuples like `(index, list[index])`. Then it's just about checking if there is any nonzero number inside those lists or not. If yes then replace them with zeroes otherwise keep as they were before. Here I used numpy arrays instead of nested python lists because they provide more convenient ways to manipulate data but feel free to change back to regular lists if needed.

```
import numpy as np
from itertools import groupby

def fill_zeros(arr):
    arr = np.array(arr).astype('int') # convert from string representation to integer array
    indices = range(len(arr))        # get indices of rows
    result = []                      # initialize empty output matrix
    for k, g in groupby(zip(indices, arr), lambda x:x[0]):     # loop over pairs containing row numbers & their corresponding values
        vals = [v for _, v in g]     # extract only second part i.e., actual values themselves without pairing up again later
        if sum(vals)>0:              # check whether there was anything other than zeros present within current sublist
            result += zip([k]*len(vals), [0]*len(vals))         # add new tuple consisting entirely out of zeros having length equal to original one
        else:
            result += zip([k]*len(vals), vals)   # append existing ones unchanged since no need here either way around anyway...
    return np.array(result).T          # transpose resulting two dimensional structure back into desired format
