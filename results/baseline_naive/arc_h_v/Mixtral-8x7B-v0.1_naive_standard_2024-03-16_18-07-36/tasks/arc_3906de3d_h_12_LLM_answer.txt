## Answer (0)

You could use `itertools.groupby()`, which groups consecutive elements of an iterable by some key function or value. In this case we want to group all rows with equal values in column zero together into one list element each. Then it's just a matter of replacing those lists containing only zeros at index position zero with their corresponding nonzero counterparts from another row within the same sublist. The rest is formatting for display purposes.

```
from pprint import pprint as pp
import numpy as np
from itertools import groupby

def replace(arr):
    # Grouping indices where first col == 0 separately from others
    g = lambda x : arr[x][0]!= 0
    grps = []
    for k, v in groupby([i for i in range(len(arr)) if not g(i)], g):
        lst = list(v)
        idx = next((idx for idx in lst if g(idx)), None)
        if idx is not None:
            val = arr[idx].copy()
            del val[0]
            for jdx in lst:
                arr[jdx][1:] = val[:]
        else:
            pass
        grps += [lst]
    return grps
