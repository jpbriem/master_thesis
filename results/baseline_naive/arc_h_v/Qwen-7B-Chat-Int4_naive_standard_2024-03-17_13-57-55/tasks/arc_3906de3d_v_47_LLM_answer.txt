```python
class Solution:
    def groupAnagrams(self, strs):
        """
        :type strs: List[str]
        :rtype: List[List[str]]
        """
        result = [[] for _ in range(len(strs))]
        
        for i in range(len(strs)):
            if len(result[i]) == 0:
                result[i].append(strs[i])
            else:
                hash_table = {}
                for char in strs[i]:
                    if char not in hash_table:
                        hash_table[char] = []
                    hash_table[char].append(i)
                
                # check if the string is anagram of any other strings
                found = False
                for j in range(len(result)):
                    if len(result[j]) > 0 and set(hash_table.keys()) == set(result[j][0]):
                        result[j].extend(result[i])
                        found = True
                        break
                
                if not found:
                    result.append([strs[i]])
                    
        return result
```

This solution uses a dictionary to store the frequency of characters in each string. It then checks if the string is anagram of any other strings by comparing the frequency of characters in the two strings. If they are the same, it adds the current string to the list of anagrams. Otherwise, it creates a new list containing only the current string.

The time complexity of this solution is O(n*m), where n is the number of strings and m is the length of the longest string. The space complexity is also O(n*m) due to the use of a dictionary to store the frequency of characters.
