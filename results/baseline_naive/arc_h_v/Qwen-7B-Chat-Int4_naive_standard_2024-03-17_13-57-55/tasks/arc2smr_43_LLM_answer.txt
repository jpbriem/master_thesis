```python
class Solution:
    def largestRectangleArea(self, heights: List[int]) -> int:
        stack = []
        
        for i in range(len(heights)):
            while len(stack) > 0 and heights[stack[-1]] >= heights[i]:
                h = heights.pop()

                area = h * w
                self.max_area += area

        return self.max_area
```

Finally, we add the calculated area to our total `max_area`.

After iterating through all the height values, we return the final `max_area` as the result.
