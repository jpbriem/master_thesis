```python
class Solution:
    def maxArea(self, height):
        if not height or len(height) == 0:
            return 0
        
        # use two pointers to find the maximum area
        left = 0
        right = len(height) - 1
        
        max_area = 0
        
        while left < right:
            current_height = min(height[left], height[right])
            
            # calculate the current area and update the maximum area if necessary
            max_area = max(max_area, current_height * (right - left))
            
            # move the pointer towards the higher value
            if height[left] <= height[right]:
                left += 1
            else:
                right -= 1
                
        return max_area
```

Then, it moves one of the pointers closer to the higher value based on the comparison of the heights. Finally, it returns the maximum area found.
The time complexity of this solution is O(n), where n is the length of the input array. This is because we only iterate through the array once. The space complexity is also O(1) since we do not use any additional data structures that grow with the size of the input.
