```python
class Solution:
    def islands(self, grid: List[List[int]]) -> int:
        n = len(grid)
        m = len(grid[0])
        
        if n == 0 or m == 0:
            return 0
        
        visited = [[False] * (m + 2) for _ in range(n + 2)]
        num_islands = 0

        for i in range(1, n+1):
            for j in range(1, m+1):
                if grid[i-1][j-1] == 1 and not visited[i][j]:
                    self.dfs(i, j, visited)
                    num_islands += 1

    def dfs(self, i, j, visited):
        visited[i][j] = True
        x, y = j - 1, i - 1
        while x >= 0 and y >= 0 and x < m and y < n and grid[x][y] == 1 and not visited[x][y]:
            visited[x][y] = True

```

This solution uses depth-first search to traverse the grid. It starts at a cell that is land and has no adjacent cells that are also land. For each such cell, it recursively visits all of its adjacent cells that are also land and mark them as visited. If a cell has been visited before, then it means that there is already an island on top of it, so we increment the count of islands.
The time complexity of this solution is O(mn), where m and n are the number of rows and columns in the grid respectively. The space complexity is O(mn).
