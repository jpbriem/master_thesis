Here's a Python code that implements this algorithm:

```python
def fill_in_zeroes(input):
    rows = len(input)
    cols = len(input[0])
    
    for i in range(rows):
        for j in range(cols):
            if input[i][j] == 5:
                # Replace all neighbors with 0
                for di in [-1, 0, 1]:
                    for dj in [-1, 0, 1]:
                        if di != 0 or dj != 0:
                            if 0 <= i+di < rows and 0 <= j+dj < cols and input[i+di][j+dj] == 5:
                                input[i+di][j+dj] = 0
    return input
```
