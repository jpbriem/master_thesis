```python
class Solution:
    def findMaxChunks(self, arr):
        """
        :type arr: List[int]
        :rtype: int
        """
        if not arr:
            return 0
        
        result = []
        
        i = 0
        while i < len(arr) - 1:
            j = i + 1
            chunk_size = 1
            
            while j < len(arr) and arr[j] == arr[i]:
                j += 1
                
            chunk_size += j - i
            
            result.append(chunk_size)
            
            i = j
            
        return max(result)
```

We iterate through the array starting from `i`. At each step, we check if the element at index `i` is equal to the element at index `j-1`. If it is, then we increment both `i` and `j`. This ensures that we only consider contiguous chunks of elements with the same value.

At any point in time, we keep track of the size of the current chunk (`chunk_size`). When `i` reaches the end of the array, we add `chunk_size` to our result list.

This solution has a time complexity of O(n), where n is the length of the input array. The space complexity is also O(1).
