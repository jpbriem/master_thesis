```python
class Solution:
    def islandPerimeter(self, grid: List[List[int]]) -> int:
        directions = [(0, -1), (0, 1), (-1, 0), (1, 0)]
        
        def dfs(x, y):
            if x < 0 or x >= len(grid) or y < 0 or y >= len(grid[0]) or grid[x][y] == 0:
                return False
            grid[x][y] -= 1
            for dx, dy in directions:
                dfs(x + dx, y + dy)
            return True
        
        perimeter = sum(dfs(x, y) for x in range(len(grid)) for y in range(len(grid[0])))
        
        return perimeter
```

Explanation:

The solution uses depth-first search (DFS) to traverse the islands. The `dfs` function takes a starting point `(x, y)` and recursively visits all its neighbors that are not land (i.e., have value 0). If any of these neighbors is an island, it updates their value to 0 and calls itself on each neighbor. This process continues until no more neighbors can be visited.

At each step, we check if the current point is outside the bounds of the grid or has a value other than 0. If either of these conditions is true, we return False immediately, indicating that this point is not part of an island.

Finally, we add up the number of steps taken by DFS over all points in the grid to get the total length of the perimeter. We subtract 1 from each island's value at each step because we want to count the perimeter around the islands, not inside them.

This algorithm has time complexity O(mn), where m and n are the dimensions of the grid. It also has space complexity O(1) since we only use a constant amount of memory.
