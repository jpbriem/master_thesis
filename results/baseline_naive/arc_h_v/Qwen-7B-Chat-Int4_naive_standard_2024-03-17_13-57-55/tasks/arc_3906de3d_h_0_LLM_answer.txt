```python
class Solution:
    def islands(self, grid):
        """
        :type grid: List[List[int]]
        :rtype: int
        """
        def dfs(i, j):
            if i < 0 or j < 0 or i >= len(grid) or j >= len(grid[0]) or grid[i][j] == 0:
                return False

            if grid[i][j]:
                grid[i][j] = 0
                dfs(i-1, j)
                dfs(i+1, j)
                dfs(i, j-1)
                dfs(i, j+1)

            return True
        
        count = 0
        for i in range(len(grid)):
            for j in range(len(grid[0])):
                if grid[i][j]:
                    count += dfs(i,j)
        
        return count
```

Explanation:

We use a recursive function `dfs` to check whether there is an island of 1's in the given matrix. If there is no island of 1's, we simply return False.

If there is an island of 1's, we mark it as 0 and recursively call `dfs` on its four neighbors (i.e., top, bottom, left, right). This will remove all the 1's in the neighborhood, effectively marking them as 0.

After checking all the cells, we sum up the number of islands found and return it. Note that this solution has a time complexity of O(mn), where m and n are the dimensions of the matrix.
