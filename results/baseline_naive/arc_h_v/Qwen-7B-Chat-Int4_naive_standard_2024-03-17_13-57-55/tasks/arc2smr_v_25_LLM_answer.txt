The base case is when there are only two possible moves left, either going up or down. In that case, there is only one way to reach the destination and the value in the cell will be 1.

For all other cases, we will calculate the number of ways to reach any given cell by summing over all possible paths from the previous cells. To do this, we will iterate through the rows and columns of the matrix, and for each cell, we will check if it has been visited before (i.e., its value is not -1). If it has been visited, we will skip it since it's already been calculated. Otherwise, we will add the number of ways to get to that cell from the previous cells to the current cell.

Finally, we return the value in the bottom-right corner of the matrix as the answer.

```python
def max_path_sum(grid):
    m = len(grid)
    n = len(grid[0])
    dp = [[-1] * n for _ in range(m)]
    dp[m-1][n-1] = grid[m-1][n-1]
    
    # Base case: only two possible moves left
    for i in range(m-2, -1, -1):
        for j in range(n-2, -1, -1):
            if grid[i][j] == 0:
                continue
            if i > 0 and grid[i-1][j] != -1:
                dp[i][j] += dp[i-1][j]
            if j > 0 and grid[i][j-1] != -1:
                dp[i][j] += dp[i][j-1]
                
    return dp[0][0]
```
