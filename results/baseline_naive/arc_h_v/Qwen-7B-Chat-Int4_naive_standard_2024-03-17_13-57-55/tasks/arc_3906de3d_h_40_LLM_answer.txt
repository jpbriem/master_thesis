```python
class Solution:
    def matrixBlockSum(self, matrix: List[List[int]]) -> List[List[int]]:
        n = len(matrix)
        m = len(matrix[0])
        
        result = [[0 for _ in range(m)] for _ in range(n)]
        
        def dfs(i, j):
            if i < 0 or j < 0 or i >= n or j >= m:
                return
            
            result[i][j] += matrix[i][j]
            
            dfs(i-1, j)
            dfs(i+1, j)
            dfs(i, j-1)
            dfs(i, j+1)
            
        dfs(0, 0)
        
        return result
```

The time complexity of this solution is O(n*m), where n is the number of rows and m is the number of columns in the matrix. The space complexity is also O(n*m) because we need to store the result in a two-dimensional array.
