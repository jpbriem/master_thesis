```python
def matrixRotation(matrix):
    n = len(matrix)
    if n == 1:
        return matrix

    for i in range(n):
        for j in range(i + 1, n):
            temp = matrix[i][j]
            matrix[i][j] = matrix[j][i]
            matrix[j][i] = temp

    return matrix
```

This function takes a 2D list `matrix` as input and returns the rotated version of it. The algorithm works by iterating over each element of the matrix and swapping its position with the corresponding element at the opposite corner of the matrix (top-right, top-left, bottom-right, bottom-left).
