I think you can use a simple loop to solve this problem.

First of all, we need to find out which row has the maximum value in each column. Then we can replace that value with `2`. After that, we can do the same thing for the columns.

```
def floodFill(self, image: List[List[int]], sr: int, sc: int, newColor: int) -> List[List[int]]:
    m = len(image)
    n = len(image[0])
    visited = set()
    def dfs(i, j):
        if i < 0 or i >= m or j < 0 or j >= n or (i,j) in visited: return
        visited.add((i,j))
        image[i][j] = newColor
        dfs(i+1, j)
        dfs(i-1, j)
        dfs(i, j+1)
        dfs(i, j-1)
    maxes = []
    for i in range(m):
        maxes.append([max(image[i][j] for j in range(n)) for j in range(n)])
    for i in range(m):
        for j in range(n):
            if image[i][j] == maxes[i][j]:
                image[i][j] = 2
                dfs(i, j)
    for j in range(n):
        maxes = [max(image[i][j] for i in range(m)) for i in range(m)]
        for i in range(m):
            if image[i][j] == maxes[i]:
                image[i][j] = 2
                dfs(i, j)
    return image
```
