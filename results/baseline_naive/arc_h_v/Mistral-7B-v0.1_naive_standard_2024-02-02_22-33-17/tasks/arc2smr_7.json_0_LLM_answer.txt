我们可以先对输入的字符串数组进行遍历，然后根据当前元素是否为 `'g'` 来决定下一步的操作：

- 若当前元素不为 `'g'`，则直接将其添加到结果中；
- 若当前元素为 `'g'`，则需要判断之前的元素是否也为 `'g'`，若是则将之前的元素替换为 `'d'`，并且将当前元素添加到结果中。

### 代码

```python
class Solution:
    def transform(self, s):
        res = []
        for i in range(len(s)):
            if s[i] == 'g':
                if i > 0 and s[i - 1] == 'g':
                    s[i - 1] = 'd'
                res.append('g')
            else:
                res.append(s[i])
        return res
```
