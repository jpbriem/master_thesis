```python
def longest_palindrome_substring(s):
    if not s:
        return []
    
    start = 0
    max_len = 0
    
    for i in range(len(s)):
        # Check for odd length palindromes starting from current character
        left, right = i, i + 1
        while left >= 0 and right < len(s) and s[left] == s[right]:
            if right - left > max_len:
                max_len = right - left
                start = left + 1
            left -= 1
            right += 1
        
        # Check for even length palindromes with center at current character
        center = i
        left, right = i, i + 1
        while left >= 0 and right < len(s) and s[left] == s[right]:
            if right - left > max_len:
                max_len = right - left
                start = left + 1
            left -= 1
            right += 1
            center += 1
        
        # Check for even length palindromes with center at current character but excluding current character
        center = i
        left, right = i, i + 1
        while left >= 0 and right < len(s) and s[left] == s[right]:
            if right - left > max_len:
                max_len = right - left
                start = left + 1
            left -= 1
            right += 1
            center -= 1
        
    return s[start:start+max_len]
```

This function takes a string `s` as input and returns a list of all substrings of maximum length that are palindromes. The algorithm uses three nested loops to check for different types of palindromes: odd-length palindromes that start at each character, even-length palindromes that have their centers at each character, and even-length palindromes that exclude the center character. The algorithm keeps track of the maximum length seen so far and returns only those substrings whose length is equal to or greater than this maximum length.
