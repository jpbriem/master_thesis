```python
class Solution:
    def connect(self, grid):
        """
        :type grid: List[List[str]]
        :rtype: NoneType
        """
        for i in range(len(grid)):
            if len(grid[i]) == 0:
                continue

            start = i * len(grid[0])
            end = (i + 1) * len(grid[0])

            for j in range(start, end):
                if grid[j][0] != '.':
                    break

            for k in range(end, start - 1, -1):
                if grid[k][len(grid[0])-1] != '.':
                    break

            for j in range(start, end):
                for k in range(end, start - 1, -1):
                    if grid[j][k] != '.' and grid[j-1][k+1] == '.':
                        grid[j-1][k+1] = grid[j][k]
                        grid[j][k] = '.'
```

We initialize two pointers `start` and `end` to keep track of the starting and ending indices of the sub-matrix we want to process. 

We then iterate over each character in the sub-matrix. If any character is not "." or the character before it is not "." but the character after it is ".", we update the character before it to be "." and set the character after it to be "." as well.

Finally, we return the updated matrix.
