The given examples show how to convert a string of characters into a 2D array where each character represents a cell in the grid. The example also shows how to fill the cells with '.' if they are empty, and 'a' or 'h' if they contain those characters.

To solve this problem, we can follow these steps:

1. Initialize an empty 2D array of size (n+1) x (m+1), where n is the number of rows and m is the number of columns.
2. Iterate over each character in the input string.
3. If the character is '.' or ' ', add it to the corresponding cell in the 2D array.
4. If the character is 'a' or 'h', replace all the empty cells around that character with that character.
5. Return the resulting 2D array.

Here's the Python code that implements the above algorithm:

```python
def convert_string_to_grid(input_str):
    # Step 1: Initialize an empty 2D array
    grid = [[None] * (len(input_str)+1) for _ in range(len(input_str[0])+1)]

    # Step 2: Iterate over each character in the input string
    for i in range(1, len(input_str)+1):
        for j in range(1, len(input_str[0])+1):
            # Step 3: Add '.' or ' ' to the corresponding cell
            if input_str[i-1][j-1] == '.' or input_str[i-1][j-1] == ' ': 
                grid[i][j] = None
            elif input_str[i-1][j-1] == 'a':
                grid[i][j] = 'a'
                grid[i-1][j] = 'a'
                grid[i][j-1] = 'a'
                grid[i-1][j-1] = 'a'
            elif input_str[i-1][j-1] == 'h':
                grid[i][j] = 'h'
                grid[i-1][j] = 'h'
                grid[i][j-1] = 'h'
                grid[i-1][j-1] = 'h'

    return grid
