```python
def longest_palindrome_substring(s):
    n = len(s)
    if n == 0 or n == 1:
        return s

    start, max_len = 0, 1
    for i in range(n):
        if s[i] == s[start]:
            start += 1
            max_len += 1
        else:
            start = i + 1

    return s[start - max_len : start + max_len + 1]
```

Explanation:

This function takes a string `s` as input and returns the longest palindrome substring of `s`. The algorithm works by iterating through each character of the string and checking if it matches the previous character. If it does, we increment the length of the current palindrome and move on to the next character. If it doesn't match, we reset the start index to the当前位置和当前长度，并继续检查下一个字符。最后，我们返回最长的回文子串。

For example, given the input "babad", the output is "bab" because "aba" and "baa" are both palindromes with the same length (3).
