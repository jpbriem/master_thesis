def longest_palindrome_substring(s):
    n = len(s)
    if n < 2:
        return s

    dp = [[False] * (n + 1) for _ in range(n + 1)]
    ans = ""

        dp[i][i] = True

    start, end = 0, 0
    max_len = 1
    for i in range(n):
        for j in range(i + 1, n + 1):
            if s[i] == s[j]:
                if j - i > max_len:

We then iterate through each character of the input string and update the corresponding value in the `dp` matrix. If we encounter two characters that are equal, we check whether their indices form a palindrome substring or not. If it does, we update the maximum palindrome substring length found so far and keep track of its starting and ending indices.

Finally, we return the longest palindrome substring by extracting it from the original string using the indices stored in `start` and `end`.
