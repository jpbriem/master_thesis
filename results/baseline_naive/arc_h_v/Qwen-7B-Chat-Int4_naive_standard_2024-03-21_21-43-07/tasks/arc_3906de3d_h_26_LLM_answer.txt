```python
class Solution:
    def findLargestRectangle(self, heights: List[List[int]]) -> int:
        n = len(heights)
        max_area = 0
        
        for i in range(n):
            h = heights[i]
            
            if not h:
                continue
            
            left = 0
            right = i + 1
            
            while right < n and heights[right] == h:
                right += 1
                
            areas = []
            for j in range(left, right):
                area = (j - left) * h
                areas.append(area)
                
            areas.sort(reverse=True)
            
            current_area = areas[0]
            for area in areas[1:]:
                current_area += area
                
            max_area = max(max_area, current_area)
        
        return max_area
```

This solution uses dynamic programming to keep track of the maximum possible area that can be enclosed by a rectangle with height `h` starting from index `i`. We initialize `left` as `0` and `right` as `i+1`, which means we are only considering rectangles that start at index `i`. We then iterate through the remaining indices from `i+1` to `n`, checking if each index has the same height as the current one. If it does, we increment `right` until there is no more indices with the same height.
For each index `j` between `left` and `right`, we calculate the area of the rectangle using the formula `(j-left)*h`. We store these areas in a list called `areas`.
Next, we sort the `areas` list in descending order, since we want to add the largest areas first. We then initialize a variable `current_area` to the first area in the sorted list, and loop through the rest of the areas, adding them to `current_area` in reverse order.
Finally, we update `max_area` to be the maximum value of `current_area` and the previous `max_area`. This gives us the final result.
