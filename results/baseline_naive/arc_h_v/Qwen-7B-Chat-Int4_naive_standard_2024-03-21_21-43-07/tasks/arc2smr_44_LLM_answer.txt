To solve this problem, we can use a recursive approach. We will define a function `bubble_sort` that takes in an array of characters and returns the sorted array.

The basic idea behind bubble sort is to repeatedly swap adjacent elements if they are in the wrong order until the entire array is sorted. This process is repeated multiple times until no more swaps are needed.

Here's how the `bubble_sort` function works:

1. Start by comparing the first two elements of the array.
2. If the first element is greater than the second element, swap them.
3. Move on to the next pair of elements (i.e., the third and fourth elements), compare them, and swap if necessary.
4. Repeat step 3 for all pairs of adjacent elements in the array.
5. After each pass through the array, the largest element will have "bubbled up" to the end of the array. Therefore, at the start of the next pass, we only need to consider the first n-1 elements of the array.
6. Repeat steps 1-5 until no more swaps are needed.

Here's the Python code for the `bubble_sort` function:
```python
def bubble_sort(arr):
    n = len(arr)
    
    # Base case: if there are only one or zero elements, return the array as it is already sorted
    if n <= 1:
        return arr
    
    # Do a single pass over the array to get its size
    for i in range(n):
        # Compare adjacent elements and swap if necessary
        if arr[i] > arr[i+1]:
            arr[i], arr[i+1] = arr[i+1], arr[i]
    
    # Recursively call the function with the remaining unsorted part of the array
    return bubble_sort(arr[:n-1])
```
