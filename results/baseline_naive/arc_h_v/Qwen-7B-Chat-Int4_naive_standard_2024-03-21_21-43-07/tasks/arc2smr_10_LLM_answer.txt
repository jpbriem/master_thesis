```python
class Solution:
    def groupAnagrams(self, words: List[str]) -> List[List[str]]:
        if not words or len(words) == 0:
            return []
        
        word_map = collections.defaultdict(list)
        for word in words:
            key = ''.join(sorted(word))
            word_map[key].append(word)
            
        groups = []
        for key, value in word_map.items():
            groups.append(value)
            
        return groups
```

Explanation:

The given problem is to group anagrams together. An anagram is a word formed by rearranging the letters of another word.

To solve this problem, we can use a dictionary to store each unique word as its key and its list of occurrences as its value. We iterate over the input list of words, sort them using `sorted()`, and join them into a string using `''.join()` method. Then, we add this string to the dictionary with its corresponding list of words.

After all the words have been processed, we iterate over the dictionary keys and append their values (i.e., lists of anagramic words) to a final result list.

Note that we use `collections.defaultdict` instead of regular dictionary to ensure that every key has at least one empty list associated with it, even if no word occurs more than once. This allows us to handle cases where some words do not occur in the input list.
