1. Initialize an empty list to store the output matrix.
2. Iterate through each row of the input matrix.
3. For each row, iterate through each column.
4. Check if the current cell is a '.' or not.
5. If it's a '.' then get the corresponding uppercase letter using the `upper()` method.
6. Replace the '.' with the uppercase letter in the current cell of the output matrix.
7. Repeat steps 4-6 for all cells in the current row.
8. Append the current row to the output matrix.
9. Return the output matrix.

Here's the Python code for implementing this solution:

```python
def convert_to_uppercase(input_matrix):
    output_matrix = []
    for i in range(len(input_matrix)):
        row = []
        for j in range(len(input_matrix[i])):
            if input_matrix[i][j] == '.':
                row.append('C')
            else:
                row.append(input_matrix[i][j].upper())
        output_matrix.append(row)
    return output_matrix
```

This implementation has a time complexity of O(n*m), where n is the number of rows in the input matrix and m is the number of columns in the input matrix. This is because we need to iterate through each element in the input matrix once to generate the output matrix.
