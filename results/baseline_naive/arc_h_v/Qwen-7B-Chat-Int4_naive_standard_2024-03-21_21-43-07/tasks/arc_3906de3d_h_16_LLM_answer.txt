```python
def flood_fill(matrix):
    rows = len(matrix)
    cols = len(matrix[0])
    
    # initialize the queue with the starting cell
    q = [(0, 0)]
    
    while q:
        row, col = q.pop(0)
        
        # check if the current cell is a wall
        if matrix[row][col] == '#':
            continue
        
        # mark the current cell as visited
        matrix[row][col] = '*'
        
        # add the four adjacent cells to the queue
        if row > 0 and matrix[row-1][col] != '#':
            q.append((row-1, col))
        if row < rows-1 and matrix[row+1][col] != '#':
            q.append((row+1, col))
        if col > 0 and matrix[row][col-1] != '#':
            q.append((row, col-1))
        if col < cols-1 and matrix[row][col+1] != '#':
            q.append((row, col+1))
```

The algorithm works by starting at the top-left corner of the matrix and visiting all cells that are not marked as walls yet. For each unvisited cell, it marks it as visited and adds its neighbors to the queue for further processing. This process continues until the entire matrix has been visited.

Note that we are using two pointers `row` and `col` to keep track of the current position in the matrix. When we visit a cell, we update both these pointers to move to the next cell. Also, we are using recursion to handle nested walls.
