class Solution:
    def permute(self, nums):
        """
        :type nums: List[int]
        :rtype: List[List[int]]
        """
        if len(nums) == 0:
            return []
        
        res = []
        self.permuteHelper(nums, 0, [], res)
        return res
    
    def permuteHelper(self, nums, start, path, res):
        if start == len(nums):
            res.append(path[:])
            return
        
        for i in range(start, len(nums)):
            path.append(nums[i])
            self.permuteHelper(nums, i + 1, path, res)
            path.pop()

In the helper function, we first check if `start` has reached the end of the `nums` list. If it has, then we append the current `path` (which represents a permutation of `nums`) to the `res` list and return.

If `start` has not reached the end of the `nums` list, we loop through the remaining elements of `nums` starting from `start`. For each element, we add it to the `path` and call the helper function again with `i+1` as the new value of `start`, `path` unchanged, and `res` updated with the current permutation. After calling the helper function recursively, we remove the last element added to `path` using the `pop()` method before moving on to the next iteration of the outer loop.
