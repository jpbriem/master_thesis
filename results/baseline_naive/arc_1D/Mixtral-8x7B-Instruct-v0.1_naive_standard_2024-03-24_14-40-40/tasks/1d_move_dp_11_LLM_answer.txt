You can use a sliding window approach. The idea is to keep track of the indices where you have seen an element different from `'a'`. When you see another such character, you expand your window by removing its leftmost boundary and adding the new index. If you encounter an `'a'`, you shrink the window by removing its rightmost boundary. You then return all elements in between the boundaries of this final window.

Here is how it could look like:

```
def longestSubarray(nums):
    start = 0
    end = -1
    for i, num in enumerate(nums):
        if num!= 'a':
            if start == -1:
                start = i
            else:
                end = i
    if start == -1:
        return nums
    while end + 1 < len(nums) and nums[end+1] == 'a':
        end += 1
    return nums[start:end+1]
```

This solution has time complexity O(n), as we iterate over each element once. It also uses constant space complexity O(1).
