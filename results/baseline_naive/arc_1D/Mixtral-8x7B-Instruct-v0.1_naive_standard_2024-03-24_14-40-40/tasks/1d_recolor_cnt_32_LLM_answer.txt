You can use a dictionary to keep track of the last time each letter was seen, then iterate through the list and replace letters based on that information. Here is an example implementation in Python 3:

```
def convert(lst):
    # Initialize dictionary with all values set to -1
    last = {letter: -1 for letter in "ag"}

    output = []
    for letter in lst:
        if letter == "g":
            index = max([last[other] for other in "ag"]) + 1
            last["g"] = index
            output.append("g" if index % 3 else "c")
        elif letter == "a":
            index = max([last[other] for other in "ag"]) + 1
            last["a"] = index
            output.append("a" if index % 3 else "i")

    return output
```
