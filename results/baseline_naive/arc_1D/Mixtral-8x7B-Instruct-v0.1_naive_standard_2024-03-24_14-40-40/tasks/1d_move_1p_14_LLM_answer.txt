You can use `itertools.groupby()` to group the consecutive elements and then replace the first element of each group with a new value if it is not equal to that value already, like this:

```
from itertools import groupby

def fix(s):
    return "".join("ag"[k!= g[0]] + k for k, g in groupby(s))

print(fix('agggggggggggggggggggggggaaa'))  # -> aggggggggggggggggggggggggaaaaa
print(fix('adddddddddddddddddddddddaaa'))  # -> adddddddddddddddddddddddaaa
print(fix('afffffffffffffffffaaaaaaaa'))  # -> afffffffffffffffffaAAAA
print(fix('aaaaaiiiiiiiiiiiiiaaaaa'))      # -> aaaaaiiiiiiiiiiiiiaaaaa
```
