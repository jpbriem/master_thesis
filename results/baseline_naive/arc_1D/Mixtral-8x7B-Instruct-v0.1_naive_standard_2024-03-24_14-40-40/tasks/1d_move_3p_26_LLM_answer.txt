If it's not, add all elements between the current position and the previous one to the result list. Then update the last seen index with the current one.

Here's some sample code in Python:

```
def remove(arr):
    last = -1
    res = []
    i = 0
    while i < len(arr):
        if arr[i]!= arr[last]:
            # Add everything from last+1 to i-1 to the result list
            res += arr[last + 1 : i]
            last = i
        i += 1
    return res
