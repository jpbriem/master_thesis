You can use a sliding window approach. The idea is to keep track of the last index where you found an "a" and then iterate over your input array, checking if there are enough "g"s between the current position and the last known "a". If so, replace all those "g"s with "a"s. Here's how that would look like in code:

```
def fix_array(arr):
    # Keep track of the last index we saw an "a" at
    last_a = -1

    # Iterate through the array
    for i in range(len(arr)):
        # Check if this element is a "g", and if it is, check if there were enough "g"s since the last "a"
        if arr[i] == "g":
            if i - last_a >= 4:
                # Replace all these "g"s with "a"s
                j = i
                while j < len(arr) and arr[j] == "g":
                    arr[j] = "a"
                    j += 1
                # Update our last seen "a" index
                last_a = j

    return arr
```
