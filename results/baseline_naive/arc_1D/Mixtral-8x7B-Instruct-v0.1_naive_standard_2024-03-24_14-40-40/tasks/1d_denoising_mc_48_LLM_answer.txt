You can use a `for` loop to iterate over the list, and two variables to keep track of the current group and its length. If an element is different from the previous one, reset these variables. Otherwise, append it to the result list if the group has reached the desired size.

Here is how this could look like in Python:

```
def expand(lst, n):
    res = []
    i = 0
    while i < len(lst):
        j = i + 1
        while j < len(lst) and lst[j] == lst[i]:
            j += 1
        grp = lst[i : j]
        grp_len = j - i
        if grp_len >= n or grp[-1]!= lst[j]:
            # add all elements of the group
            res.extend(grp)
        else:
            # only add enough elements to reach the desired size
            res.extend(grp[:n])
        i = j
    return res
```
