If we find an "f", then we set both variables to `False`. Otherwise, if we find another "a", we reset them to `True`, since this means we found a new group of consecutive elements starting with an "a":

```
def replace(arr):
    i = 0
    lastWasA = True
    hasF = False

    while i < len(arr):
        if arr[i] == 'a':
            lastWasA = True
            hasF = False
        elif arr[i] == 'f':
            lastWasA = False
            hasF = True

        if lastWasA and hasF:
            arr[i] = 'i'

        i += 1

    return arr
```
