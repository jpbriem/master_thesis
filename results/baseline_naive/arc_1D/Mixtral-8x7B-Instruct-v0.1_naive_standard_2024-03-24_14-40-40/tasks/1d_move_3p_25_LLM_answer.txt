You can use a simple for loop to iterate through the array, keeping track of the current group size with `groupSize`, resetting it when you encounter an element that is different from the previous one. You also need another variable `maxGroupSize` to keep track of the largest group found so far. Here's how I would do it in Python:

```
def longestGroup(arr):
    maxGroupSize = 0
    groupSize = 1
    i = 0
    while i < len(arr)-1:
        if arr[i] == arr[i+1]:
            groupSize += 1
        else:
            maxGroupSize = max(maxGroupSize, groupSize)
            groupSize = 1
        i += 1
    return max(maxGroupSize, groupSize)
```

This function will work correctly even if there are multiple groups of equal length or if all elements belong to the same group.
