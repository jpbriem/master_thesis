Here's some Python code that implements this approach:
```python
def process_list(lst):
    # Initialize variables for counting consecutive 'a's and keeping track of indices
    count = 0
    indices = []

    # Iterate through the list and keep track of indices where 'a' appears
    for i, char in enumerate(lst):
        if char == 'a':
            count += 1
            indices.append(i)
        else:
            count = 0

    # Replace consecutive 'a's with 'f' in the original list and its reversed version
    for i in range(len(indices) - 1, -1, -1):
        if count >= 5:
            lst[indices[i]:indices[i] + count] = ['f'] * count
            lst[:indices[i]+count] = lst[:indices[i]] + ['f']*(count-1)

    return lst
```
