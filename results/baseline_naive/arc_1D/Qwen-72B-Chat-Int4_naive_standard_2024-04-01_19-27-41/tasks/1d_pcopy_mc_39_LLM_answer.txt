```python
def replace_chars(s):
    # Create a dictionary to map characters to their counts
    char_counts = {}
    for c in s:
        if c.isalpha():
            char_counts[c] = char_counts.get(c, 0) + 1
    
    # Iterate through the string and replace characters with their counts
    result = []
    prev_char = None
    for i, c in enumerate(s):
        if c == '.' or not c.isalpha():
            result.append(c)
        elif c != prev_char:
            count = char_counts.pop(c, 0)
            if count >= 3:
                result.extend([c]*count)
            else:
                result.append(c)
            prev_char = c
    return ''.join(result)
```

This function works by first creating a dictionary `char_counts` that maps each distinct alphabetical character in the input string to its frequency of occurrence. Then it iterates through the string again, replacing each non-dot character with either its corresponding count (if it appears at least three times), or itself (if it appears fewer than three times). The resulting list is then joined back into a single string.
