To solve this problem, we can iterate through the input list and check if a character is surrounded by at least 5 identical characters on both sides. If it is not, we replace that character with a period.

Here's the Python code to implement this solution:

```python
def clean_string(input_list):
    output_list = []
    
    for i in range(len(input_list)):
        left_count = 0
        right_count = 0
        
        # Check left side
        for j in range(i-1, -1, -1):
            if input_list[j] == input_list[i]:
                left_count += 1
            else:
                break
        
        # Check right side
        for j in range(i+1, len(input_list)):
            if input_list[j] == input_list[i]:
                right_count += 1
            else:
                break
        
        # Replace character if it doesn't meet the condition
        if left_count < 5 or right_count < 5:
            output_list.append('.')
        else:
            output_list.append(input_list[i])
    
    return output_list
```
