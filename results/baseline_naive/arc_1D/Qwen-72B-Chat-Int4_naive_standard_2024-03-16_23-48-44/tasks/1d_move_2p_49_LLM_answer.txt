The idea is to move the right pointer until we find a non-zero element, then keep moving the left pointer until we find a zero element. While doing so, update the maximum length of the consecutive sequence of non-zero elements seen so far.

Here's the Python code that implements this solution:

```python
def longest_consecutive_subarray(arr):
    n = len(arr)
    max_len = 0
    left = 0

    for right in range(n):
        while arr[right] == 0:
            right += 1

        if right - left > max_len:
            max_len = right - left

        while arr[left] != 0:
            left += 1

    return max_len + 1
```
