Here's the Python code to implement this solution:

```python
def cumulative_sum_diff(arr):
    cum_sum = 0
    result = []
    
    for i in range(len(arr)):
        if arr[i] != 0:
            cum_sum += arr[i]
        
        if arr[i] == 0:
            result.append(cum_sum - (cum_sum // 5 * 5))
        else:
            result.append(cum_sum)
            
    return result
```
