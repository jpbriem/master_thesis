To solve this problem, you can iterate over the input list and keep track of the last non-zero element seen. When a new non-zero element is encountered, replace all the elements between the current position and the last non-zero element with that value.

Here's an implementation in Python:

```python
def merge_zeros(input_list):
    last_non_zero = None
    for i, num in enumerate(input_list):
        if num != 0:
            if last_non_zero is not None:
                input_list[last_non_zero:i] = [num] * (i - last_non_zero)
            last_non_zero = i

    return input_list
```
