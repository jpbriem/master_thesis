To solve this problem, we can iterate over the input list and keep track of the current count of consecutive 2s. Whenever we encounter a 0 or the end of the list, we append the current count to the output list and reset the count to zero.
Here's the Python code that implements this algorithm:
```python
def compress_consecutive_twos(input_list):
    output = []
    count = 0
    
    for num in input_list:
        if num == 2:
            count += 1
        else:
            if count > 0:
                output.append(count)
                count = 0
            output.append(num)
    
    # Append the last count (if any) to the output list
    if count > 0:
        output.append(count)
        
    return output
```
