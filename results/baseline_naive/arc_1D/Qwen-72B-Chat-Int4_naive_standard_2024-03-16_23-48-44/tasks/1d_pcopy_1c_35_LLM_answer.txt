To solve this problem, we can iterate through the input list and check if a non-zero element is followed by one or more zeros. If so, we replace those zeros with the same number as the previous non-zero element.
Here's an implementation of this approach in Python:
```python
def expand_zeros(input_list):
    output_list = []
    prev_non_zero = None
    
    for num in input_list:
        if num != 0:
            prev_non_zero = num
            output_list.append(num)
        elif prev_non_zero is not None:
            output_list.extend([prev_non_zero] * (len(output_list) + 1))
    
    return output_list
```
