```python
class Solution:
    def groupAnagrams(self, strs):
        """
        :type strs: List[str]
        :rtype: List[List[str]]
        """
        d = {}
        for s in strs:
            key = ''.join(sorted(s))
            if key not in d:
                d[key] = []
            d[key].append(s)
        
        return [d[i] for i in sorted(d)]
```

Explanation:

We use a dictionary to store the anagram groups. We iterate through each string and sort its characters. Then we take the sorted string as the key of the dictionary and append it to the corresponding list of strings.

Finally, we return a list containing all the lists of anagrams.

Time complexity: O(n*mlogm), where n is the number of strings and m is the maximum length of a string.

Space complexity: O(n*m), where n is the number of strings and m is the maximum length of a string.
