If a new character is encountered, we update its last occurrence to be at the current index in the string. Finally, we return all characters from the input string that have not been updated.
Here's an implementation in Python:

```python
def last_occurrence(input_string):
    last_occurrences = {}
    for i, char in enumerate(input_string):
        if char in last_occurrences:
            last_occurrences[char] = max(last_occurrences[char], i)
        else:
            last_occurrences[char] = i

    output_string = []
    for char in input_string:
        if char not in last_occurrences:
            output_string.append(char)

    return ''.join(output_string)
```
