Then, we will use that information to create an output string with the characters in the same order as they appear in the input string, but with their frequency replaced by a period followed by their count.
Here is some Python code that implements this approach:

```python
def replace_chars(input_string):
    # Count the number of occurrences of each character
    char_counts = {}
    for char in input_string:
        if char not in char_counts:
            char_counts[char] = 0
        char_counts[char] += 1

    # Create an output string with periods and counts
    output_string = ''
    for char, count in sorted(char_counts.items()):
        output_string += '.' * count + char

    return output_string
```
