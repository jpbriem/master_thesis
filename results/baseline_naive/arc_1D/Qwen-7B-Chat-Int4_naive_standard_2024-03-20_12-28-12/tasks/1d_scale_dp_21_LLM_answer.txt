If it's not a dot, we check if it's equal to the last character in the output string (which should be a dot). If they are equal, we add another dot to the output string. Otherwise, we add the current character as the new last character of the output string.
Here's the Python code for this solution:

```python
def replace_dot(input_string):
    output_string = []
    last_char = None

    for char in input_string:
        if char == '.':
            if last_char != '.':
                output_string.append('.')
        else:
            output_string.append(char)
            last_char = char

    return ''.join(output_string)
```
