```python
class Solution:
    def removeDuplicates(self, s: str) -> str:
        if not s:
            return s
        
        # Convert the string to a set of characters
        unique_chars = set(s)
        
        # Initialize an empty list to store the result
        result = []
        
        # Iterate over the unique characters and add them to the result
        for char in unique_chars:
            result.append(char)
            
        # Join the characters in the result into a single string
        return ''.join(result)
```

Explanation:

The first step is to check if the input string `s` is empty. If it is, then we simply return it since there are no duplicates.

Next, we convert the input string to a set of characters using the built-in `set()` function. This will automatically remove any duplicate characters from the string.

We initialize an empty list called `result` to store the final output. We will iterate over the unique characters in the set and add each character to the `result` list.

Finally, we join the characters in the `result` list into a single string using the `join()` method with an empty separator string.

This solution has a time complexity of O(n), where n is the length of the input string. The space complexity is also O(n), as we need to store all the unique characters in the set.
