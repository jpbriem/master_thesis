If they are not equal, we will move the right pointer one position to the right and update the output string accordingly. If they are equal, we will move the left pointer one position to the right.
Here's an implementation of this algorithm in Python:

```python
def removeDuplicates(s):
    n = len(s)
    output = [s[0]]
    for i in range(1, n):
        while output[-1] != s[i]:
            output.pop()
        output.append(s[i])
    return ''.join(output)
```
