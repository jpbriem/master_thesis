```python
class Solution:
    def groupAnagrams(self, s: str) -> List[List[str]]:
        d = {}
        res = []
        
        for i in range(len(s)):
            if s[i] not in d:
                d[s[i]] = [s[i]]
            else:
                d[s[i]].append(s[i])
                
        for k in d.keys():
            res.append(d[k])
            
        return res
```

In this solution, we use a dictionary to store the anagram groups. We iterate through each character in the input string `s` and check if it is already in the dictionary. If it is not, we add it as a new key with value equal to a list containing only that character. If it is already in the dictionary, we append the character to its existing value.

After iterating through all characters in the input string, we convert the values of the dictionary into a list and return it as the output. This list contains one element per anagram group.
