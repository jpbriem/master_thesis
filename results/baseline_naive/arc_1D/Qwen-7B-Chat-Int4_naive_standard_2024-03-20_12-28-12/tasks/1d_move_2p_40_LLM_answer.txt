To solve this problem, we can use a simple approach. We iterate over the input string and check if each character is equal to '.' or not. If it is equal to '.', we append it to the output string. If it is not equal to '.', we increment a counter variable and skip it.
Here's the Python code for the solution:

```python
def find_unique_chars(input_string):
    output_string = []
    count = 0

    for char in input_string:
        if char == '.':
            output_string.append('.')
        else:
            count += 1

    while count > 0:
        output_string.append('.')

    return output_string
```
