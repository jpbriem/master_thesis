To solve this problem, we can iterate through the input string and keep track of the last character that was printed. If a new character is encountered that is different from the previous one, we print it and reset the last printed character to be the current one. We also need to check if the current character is '.' and skip printing it.
Here's the Python code to implement this solution:

```python
def longest_common_subsequence(input):
    # initialize an empty list to store the result
    output = []
    
    # initialize variables to keep track of the last printed character
    prev_char = None
    curr_char = None
    
    # iterate through the input string
    for char in input:
        # check if the current character is different from the previous one
        if char != prev_char:
            # print the current character
            output.append(char)
            
            # update the last printed character
            prev_char = curr_char
            
            # update the current character
            curr_char = char
        
    # check if the last printed character is different from the first one
    if prev_char != curr_char:
        # print the last character
        output.append(curr_char)
        
    return ''.join(output)
```
