```python
class Solution:
    def longestPalindrome(self, s: str) -> int:
        if len(s) == 0:
            return 0

        # Base cases
        if len(s) <= 1:
            return len(s)

        start = 0
        max_len = 1
        for i in range(len(s)):
            # Check odd length palindromes
            if i - max_len >= 0 and s[i-max_len-1:i+1] == s[i-max_len-1:i+1][::-1]:
                start = i - max_len - 1
                max_len += 2

            # Check even length palindromes
            if i - max_len >= 0 and i + max_len < len(s) and s[i-max_len:i+max_len+1] == s[i-max_len:i+max_len+1][::-1]:
                start = i - max_len
                max_len += 2

        return max_len
```

This solution uses dynamic programming to find the longest palindrome in a string. It iterates over each character in the string and checks whether it is part of an odd or even length palindrome centered at that position.

The `start` variable keeps track of the starting index of the current longest palindrome, and the `max_len` variable keeps track of its length. If a new longer palindrome is found, the `start` and `max_len` variables are updated accordingly.

At the end of the loop, the function returns the length of the longest palindrome found.
