```python
def remove_periods(input_string):
    # Convert the input string to lowercase for easier processing
    input_string = input_string.lower()
    
    # Initialize an empty list to store the repeating characters
    repeating_chars = []
    
    # Iterate through each character in the input string
    for char in input_string:
        # If the current character is not a period, add it to the repeating_chars list
        if char != '.':
            repeating_chars.append(char)
            
    # Return the resulting string with no periods
    return ''.join(repeating_chars)

# Test the function with the given examples

```

Finally, it joins the elements of the repeating_chars list into a single string using the `join()` method and returns the result.
