```python
class Solution:
    def removeDuplicateLetters(self, s: str) -> str:
        # Create a set to keep track of characters we've seen so far.
        unique_chars = set()
        
        # Initialize an empty string to store the result.
        result = ''
        
        # Iterate through each character in the input string.
        for char in s:
            if char not in unique_chars:
                # If this is a new character, add it to the result string and the set of unique characters.
                result += char
                unique_chars.add(char)
            
        return result
```

This solution works by iterating through each character in the input string. For each character, it checks whether it has already been seen before by checking if it's in the `unique_chars` set. If the character hasn't been seen before, it adds it to both the `result` string and the `unique_chars` set. This ensures that only one copy of each character is included in the output string.

The time complexity of this solution is O(n), where n is the length of the input string. The space complexity is also O(n), since we need to store all the characters we've seen so far in the `unique_chars` set.
