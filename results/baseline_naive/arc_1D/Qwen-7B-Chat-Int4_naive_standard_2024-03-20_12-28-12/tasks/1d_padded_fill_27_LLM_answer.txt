To solve this problem, we can use a sliding window approach. We will maintain two pointers, one at the beginning of the array and one at the end of the array. The pointer at the end will move forward while the pointer at the beginning will move backward until they meet in the middle.
At each step, we will check if the character at the current position is equal to the character that was previously encountered. If it is not equal, then we will add the character to the output string and remove all characters from the input string that were encountered before the current position. This way, we can keep track of only the most recent occurrences of each character.
Here's the Python code to implement this algorithm:
```python
def longest_substring(input_string):
    n = len(input_string)
    i, j = 0, n - 1

    # Initialize an empty string for storing the result
    output_string = ""

    # Iterate over the string
    while i < j:
        # Check if the characters are equal
        if input_string[i] == input_string[j]:
            # Add the character to the output string
            output_string += input_string[i]
            # Move both pointers forward
            i += 1
            j -= 1
        else:
            # Remove all characters from the input string that were encountered before
            # the current position
            k = max(i - 1, 0)
            l = min(j + 1, n - 1)

            for m in range(k, l):
                input_string[m] = ""
            # Move the pointer at the beginning to the next occurrence of the character
            i = k + 1

    return output_string
```
