```python
class Solution:
    def longestPalindrome(self, s: str) -> int:
        if len(s) < 2:
            return 0

        start = max(0, 1 - self.helper(s, 0))
        end = min(len(s), self.helper(s, len(s) - 1))

        return end - start + 1

    def helper(self, s: str, i: int) -> int:
        while i < len(s):
            j = i + 1
            while j < len(s) and s[j] == s[i]:
                j += 1
            
            if j - i >= 3:
                return j - i

            i += 1
```

Then it iterates over the string starting from index `i`, and for each character, it checks if there is another character at distance `j-i+1` that is equal to it. If so, it updates `j` and continues iterating until it reaches the end of the string or finds a pair of characters that are not equal. If `j-i>=3`, then we have found a palindrome of length `j-i`, which is greater than any previously found palindromes, so we update `end` accordingly. Finally, the function returns the difference between `end` and `start`, plus one, since the last palindrome may be only one character long.
