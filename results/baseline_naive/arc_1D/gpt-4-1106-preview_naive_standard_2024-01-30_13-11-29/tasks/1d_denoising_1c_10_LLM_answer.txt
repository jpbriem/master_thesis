Here's the code to achieve this in Python:

```python
from collections import Counter

def group_elements(input_list):
    # Count the occurrences of each element
    element_counts = Counter(input_list)
    
    # Identify the most common element
    most_common_element, _ = element_counts.most_common(1)[0]
    
    # Split the input list into two lists: one for the most common element and one for the rest
    most_common_list = [elem for elem in input_list if elem == most_common_element]
    rest_list = [elem for elem in input_list if elem != most_common_element]
    
    # Concatenate the lists to get the desired output
    output_list = most_common_list + rest_list
    
    return output_list
```
