To solve this problem, you can iterate over the input list and keep track of the last seen element and its count. When a new element is encountered, append it to the output list along with its count if it's different from the last seen element. Update the last seen element and its count accordingly.
Here's the code that implements this approach:
```python
def group_elements(lst):
    last_seen = None
    last_count = 0
    
    result = []
    
    for elem in lst:
        if elem != last_seen:
            if last_count > 0:
                result.append((last_seen, last_count))
            last_seen = elem
            last_count = 1
        else:
            last_count += 1
            
    if last_count > 0:
        result.append((last_seen, last_count))
        
    return result
```
