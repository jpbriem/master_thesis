To solve this problem, we can iterate through the input list and keep track of the current group of characters. Whenever a different character is encountered, we can append the previous group to the output list and start a new group with the current character.
Here's the code that implements this approach:
```python
def compress(input_list):
    if not input_list:
        return []
    
    output = [input_list[0]]
    prev_char = input_list[0]
    count = 1
    
    for i in range(1, len(input_list)):
        curr_char = input_list[i]
        
        if curr_char == prev_char:
            count += 1
        else:
            output.append(count)
            output.append(prev_char)
            count = 1
        
        prev_char = curr_char
    
    # Append the last group to the output list
    output.append(count)
    output.append(prev_char)
    
    return output
```
