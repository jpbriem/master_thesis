Here's an example implementation in Python:

```python
def group_chars(lst):
    result = []
    prev_char = None
    count = 0

    for char in lst:
        if char != prev_char:
            if prev_char is not None:
                result.append((prev_char, count))
            prev_char = char
            count = 1
        else:
            count += 1

    result.append((prev_char, count))

    return [char * num for char, num in result]
```
