```python
def replace_adjacent_chars(input_list):
    output_list = []
    prev_char = None
    
    for char in input_list:
        if char == prev_char and prev_char is not None:
            continue
        
        output_list.append(char)
        prev_char = char
    
    return output_list
```

This function works by iterating over each character in the input list, and only appending it to the output list if it's different from the previous character. This effectively replaces adjacent occurrences of the same character with a single occurrence.
