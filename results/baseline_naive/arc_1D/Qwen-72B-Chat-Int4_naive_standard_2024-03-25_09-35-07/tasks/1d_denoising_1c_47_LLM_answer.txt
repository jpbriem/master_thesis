To solve this problem, we can iterate over the input list and keep track of the current character being processed. We'll also maintain a count of consecutive occurrences of that character.

When the character changes or we reach the end of the list, we can append the most recent character and its count to the output list.

Here's the Python code to implement this solution:

```python
def compress_chars(input_list):
    result = []
    prev_char = None
    count = 0

    for char in input_list:
        if char != prev_char:
            if prev_char is not None:
                result.append(prev_char)
                if count > 1:
                    result.append(str(count))
            prev_char = char
            count = 1
        else:
            count += 1
    
    # Append the last character and its count (if necessary)
    if prev_char is not None:
        result.append(prev_char)
        if count > 1:
            result.append(str(count))

    return result
```
