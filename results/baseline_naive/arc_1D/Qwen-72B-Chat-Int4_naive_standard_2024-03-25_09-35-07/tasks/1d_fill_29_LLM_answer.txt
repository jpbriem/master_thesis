To solve this problem, you can iterate through the input list and check if the current character is different from the previous one. If it is different, append the new character to the output list and update the count of consecutive characters. If it is the same as the previous one, increment the count.
Here's a Python function that implements this algorithm:
```python
def compress_chars(lst):
    out = [lst[0]]
    count = 1
    
    for i in range(1, len(lst)):
        if lst[i] == lst[i-1]:
            count += 1
        else:
            out.append(count)
            out.append(lst[i])
            count = 1
            
    out.append(count)
    
    return out
```
