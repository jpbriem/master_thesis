To solve this problem, you can iterate through the input list and keep track of the current character being processed. If the current character is different from the previous one, append it to the output list along with its count. If the current character is the same as the previous one, increment its count.
Here's a Python function that implements this approach:
```python
def compress_string(lst):
    if not lst:
        return []
    
    prev_char = lst[0]
    count = 1
    
    result = [prev_char]
    
    for char in lst[1:]:
        if char == prev_char:
            count += 1
        else:
            result.append(str(count))
            result.append(char)
            prev_char = char
            count = 1
            
    result.append(str(count))
    
    return result
```
