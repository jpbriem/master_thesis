To solve this problem, we can iterate through the input list and keep track of the last seen character. Whenever a new character is encountered that is different from the last seen character, we append it to the output list along with the count of consecutive occurrences of the previous character. We update the count for the current character as well.
Here's the Python code that implements this approach:
```python
def compress_chars(lst):
    if not lst:
        return []
    
    last_char = lst[0]
    count = 1
    
    result = [last_char]
    
    for i in range(1, len(lst)):
        if lst[i] != last_char:
            result.append(str(count))
            result.append(last_char)
            count = 1
        else:
            count += 1
        
        last_char = lst[i]
    
    result.append(str(count))
    result.append(last_char)
    
    return result
```
