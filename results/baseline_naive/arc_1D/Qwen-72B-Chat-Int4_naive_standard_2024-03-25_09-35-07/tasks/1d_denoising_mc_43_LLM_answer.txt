To solve this problem, we can iterate through the input list and keep track of the current character being processed. If the current character is different from the previous one, we append it to the output list along with its count. We also update the previous character variable to the current character.
Here's the Python code that implements this approach:
```python
def compress(input_list):
    if not input_list:
        return []
    
    prev_char = input_list[0]
    count = 1
    
    output = [prev_char]
    
    for i in range(1, len(input_list)):
        curr_char = input_list[i]
        
        if curr_char != prev_char:
            output.append(str(count))
            output.append(curr_char)
            count = 1
        else:
            count += 1
            
        prev_char = curr_char
    
    # Append last character and its count
    output.append(str(count))
    
    return output

# Test cases
```
