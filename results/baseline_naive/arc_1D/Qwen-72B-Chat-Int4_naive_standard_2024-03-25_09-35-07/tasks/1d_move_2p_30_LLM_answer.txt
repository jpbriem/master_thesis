To solve this problem, we can iterate through the input list and keep track of the current character being processed. If a new character is encountered, we add it to the output list along with the count of occurrences so far.

Here's the code that implements this approach:

```python
def compress_characters(input_list):
    result = []
    prev_char = ''
    count = 0
    
    for char in input_list:
        if char != prev_char:
            if count > 0:
                result.append(prev_char + str(count))
            prev_char = char
            count = 1
        else:
            count += 1
            
    result.append(prev_char + str(count))
    
    return result
```
