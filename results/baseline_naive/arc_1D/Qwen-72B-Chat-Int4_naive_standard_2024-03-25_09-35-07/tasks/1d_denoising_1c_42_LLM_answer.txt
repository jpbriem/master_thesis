To solve this problem, we can iterate through the input list and keep track of the current character being processed. We will also maintain a count of consecutive occurrences of that character.

When we encounter a new character or reach the end of the input list, we can append the current character to the output list along with its maximum consecutive occurrence count.

Here's the Python code to implement this approach:

```python
def compress(input_list):
    if not input_list:
        return []
    
    result = [input_list[0], 1]
    max_count = 1
    
    for i in range(1, len(input_list)):
        if input_list[i] == input_list[i - 1]:
            max_count += 1
        else:
            result.append(max_count)
            result.append(input_list[i])
            max_count = 1
            
    result.append(max_count)
    
    return result
```
