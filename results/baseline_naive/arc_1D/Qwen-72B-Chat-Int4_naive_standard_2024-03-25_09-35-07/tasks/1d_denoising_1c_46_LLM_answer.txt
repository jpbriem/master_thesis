To solve this problem, we can iterate through the input list and keep track of the current character being processed. If we encounter a new character that is different from the previous one, we append it to the output list along with its count so far. We also update the current character to the new one.
Here's an implementation in Python:
```python
def compress(input):
    if not input:
        return []
    
    result = [input[0]]
    count = 1
    
    for i in range(1, len(input)):
        if input[i] == input[i-1]:
            count += 1
        else:
            result.append(count)
            result.append(input[i])
            count = 1
            
    result.append(count)
    return result
```
