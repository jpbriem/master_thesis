To solve this problem, we can iterate through the input list and keep track of the current character being processed. We will also maintain a count of consecutive occurrences of that character. If the count becomes greater than or equal to 5, we add the character to the output list only once. Otherwise, we add it every time we encounter it.

Here's the Python code that implements this approach:

```python
def process_list(input_list):
    result = []
    prev_char = None
    count = 0

    for char in input_list:
        if char == prev_char:
            count += 1
        else:
            if count >= 5:
                result.append(prev_char)
            count = 1
        
        prev_char = char
    
    if count >= 5:
        result.append(prev_char)

    return result
```
