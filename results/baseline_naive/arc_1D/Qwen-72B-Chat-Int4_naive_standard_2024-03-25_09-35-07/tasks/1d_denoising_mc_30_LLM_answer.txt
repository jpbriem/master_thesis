To solve this problem, we can iterate through the input list and keep track of the current character being processed. We'll also maintain a count of how many times that character has appeared in a row. If the count becomes greater than 4, we replace all occurrences of that character with a single occurrence.

Here's a Python function that implements this approach:

```python
def compress_string(input_list):
    result = []
    count = 1

    for i in range(1, len(input_list)):
        if input_list[i] == input_list[i - 1]:
            count += 1
        else:
            if count <= 4:
                result.extend([input_list[i - 1]] * count)
            else:
                result.append(input_list[i - 1])
            count = 1
    
    # Add the last character to the result
    if count <= 4:
        result.extend([input_list[-1]] * count)

    return result
```
