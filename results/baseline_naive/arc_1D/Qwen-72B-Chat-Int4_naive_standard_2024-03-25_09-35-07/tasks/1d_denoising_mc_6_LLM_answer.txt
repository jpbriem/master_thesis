To solve this problem, we can iterate over the input list and keep track of the current character being processed. Whenever a new character is encountered or the count of the current character exceeds 4, we add it to the output list along with its count.

Here's an implementation in Python:

```python
def compress_chars(lst):
    result = []
    count = 1

    for i in range(1, len(lst)):
        if lst[i] == lst[i - 1]:
            count += 1
        else:
            result.append(lst[i - 1])
            result.extend([str(count)] * (count > 1))
            count = 1
    
    result.append(lst[-1])
    result.extend([str(count)] * (count > 1))

    return result
```
