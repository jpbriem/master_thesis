```python
def remove_vowels_and_duplicates(input_list):
    vowels = set(['a', 'e', 'i', 'o', 'u'])
    output_list = []
    last_char = None
    
    for char in input_list:
        if char.lower() not in vowels and char != last_char:
            output_list.append(char)
            last_char = char
            
    return output_list
```

This function starts by initializing an empty list `output_list` and a variable `last_char` that will keep track of the last character added to the output list. It then loops through each character in the input list, checking whether it is a vowel (case-insensitively) or equal to the last character added to the output list.
If the character is neither a vowel nor equal to the last character added to the output list, it is appended to `output_list` and `last_char` is updated to its value.
Finally, the function returns `output_list`.
