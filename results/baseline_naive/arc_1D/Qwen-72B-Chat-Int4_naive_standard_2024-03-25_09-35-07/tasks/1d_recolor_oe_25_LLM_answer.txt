To solve this problem, we can iterate through the input list and keep track of the current streak of consecutive characters. If the character is different from the previous one or if it's the first character in the streak, we reset the counter to 0. Otherwise, we increment the counter.
When the counter reaches a multiple of 5, we append the index of the next character to be replaced (starting at 'i' for the first group) to a result list. Finally, we return the output list with the specified characters replaced by their corresponding indices.

Here's the code that implements this approach:

```python
def replace_chars(input_list):
    output = []
    prev_char = None
    streak_counter = 0
    
    for char in input_list:
        if char != prev_char or not prev_char:
            streak_counter = 0
        
        streak_counter += 1
        prev_char = char
        
        if streak_counter % 5 == 0:
            output.append('i' + str(int(streak_counter / 5)))
        else:
            output.append(char)
    
    return output
```
