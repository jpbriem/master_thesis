To solve this problem, we can iterate over the input list and keep track of the current character being processed. If the current character is different from the previous one or if it has reached its maximum limit, we add it to the output list. We also need to update the count of characters seen so far.

Here's a Python function that implements this approach:

```python
def process_list(input_list):
    max_limit = 5
    output_list = []
    prev_char = None
    char_count = 0

    for char in input_list:
        if char != prev_char or char_count == max_limit:
            output_list.append(char)
            char_count = 1
            prev_char = char
        else:
            char_count += 1
    
    return output_list
```
