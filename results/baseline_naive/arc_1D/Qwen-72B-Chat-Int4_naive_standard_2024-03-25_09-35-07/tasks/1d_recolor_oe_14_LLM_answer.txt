To solve this problem, we can iterate over the input list and keep track of the current character being processed. If the current character is different from the previous one, we append a new letter to the output list with an incremented value. Otherwise, we continue appending the same letter to the output list.

```python
def convert_to_unique_chars(input_list):
    output_list = []
    prev_char = None
    count = 0
    
    for char in input_list:
        if char != prev_char:
            count += 1
            prev_char = char
        
        output_list.append(chr(ord('a') + count))
    
    return output_list
```
