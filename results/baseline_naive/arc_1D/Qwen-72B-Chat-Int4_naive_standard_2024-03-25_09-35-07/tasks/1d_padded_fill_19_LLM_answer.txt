To solve this problem, you can iterate through the input list and keep track of the current character being processed. If the current character is different from the previous one, append it to the output list along with its count so far. If it's the same as the previous one, increment its count instead.
Here's some Python code that implements this algorithm:
```python
def group_chars(lst):
    prev_char = None
    count = 0
    
    result = []
    
    for char in lst:
        if char != prev_char:
            if prev_char is not None:
                result.append((prev_char, count))
            prev_char = char
            count = 1
        else:
            count += 1
            
    result.append((prev_char, count))
    
    return [char * num for char, num in result]
```
