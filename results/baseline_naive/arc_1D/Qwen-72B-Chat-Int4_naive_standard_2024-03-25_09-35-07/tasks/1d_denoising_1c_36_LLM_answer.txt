To solve this problem, we can iterate over the input list and keep track of the current character being processed. If the current character is different from the previous one or if it's the first character in the list, we add it to the output list. Otherwise, we continue processing the next characters until we find a different character.

Here's the Python code that implements this approach:

```python
def compress_chars(lst):
    if not lst:
        return []

    result = [lst[0]]
    prev_char = lst[0]
    
    for char in lst[1:]:
        if char != prev_char:
            result.append(char)
            prev_char = char
    
    return result
```
