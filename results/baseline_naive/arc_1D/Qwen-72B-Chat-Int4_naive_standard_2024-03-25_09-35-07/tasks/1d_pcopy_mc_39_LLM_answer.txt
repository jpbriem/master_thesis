To solve this problem, we can iterate through the input list and keep track of the current character being processed. If a new character is encountered, we add it to the output list along with its count so far. If the same character as the previous one is encountered, we increment its count in the output list.
Here's the Python code that implements this algorithm:
```python
def process_list(lst):
    if not lst:
        return []
    
    prev_char = lst[0]
    count = 1
    
    result = [prev_char]
    
    for i in range(1, len(lst)):
        curr_char = lst[i]
        
        if curr_char == prev_char:
            count += 1
        else:
            result.append(curr_char)
            result.append(str(count))
            count = 1
            
        prev_char = curr_char
        
    # Add the last character and its count to the result list
    result.append(prev_char)
    result.append(str(count))
    
    return result
```
