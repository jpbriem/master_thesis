## Answer (4)

You can use a sliding window approach to solve this problem. You need to keep track of the sum of elements in the current window and also the sum of elements in the previous window. Whenever you encounter an element that is greater than or equal to the sum of the previous window, you add it to the output.

Here's the Python code for the above approach:

```python
def maxSum(arr):
    n = len(arr)
    result = []
    curr_sum = 0
    prev_sum = 0
    for i in range(n):
        if curr_sum >= prev_sum:
            result.append(curr_sum)
        curr_sum += arr[i]
        prev_sum = curr_sum
    return result
```

The time complexity of this solution is O(n).
