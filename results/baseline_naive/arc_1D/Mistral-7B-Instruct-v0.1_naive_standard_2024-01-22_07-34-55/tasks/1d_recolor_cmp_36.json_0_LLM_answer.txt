You can use a `for` loop to iterate over each element in the array and check if it is equal to zero or not. If it is equal to zero then you can add one to that element otherwise you can leave it as it is.

Here's an example of how you could do this:

```
function transform(arr) {
    let result = [];
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === 0) {
            result.push(result[i] + 1);
        } else {
            result.push(arr[i]);
        }
    }
    return result;
}
