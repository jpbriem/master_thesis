You can use a sliding window approach to solve this problem. You need to keep track of the current sum and the previous sum. If the current sum is greater than or equal to the target sum, you can return true. Otherwise, you need to move the window one element to the right.

Here's an implementation in Python:

```
def hasSum(numbers: List[int], target: int) -> bool:
    cur_sum = 0
    prev_sum = 0
    for num in numbers:
        cur_sum += num
        if cur_sum >= target:
            return True
        prev_sum = cur_sum
        cur_sum -= prev_sum
    return False
```

The time complexity of this solution is O(n), where n is the length of the input list.
