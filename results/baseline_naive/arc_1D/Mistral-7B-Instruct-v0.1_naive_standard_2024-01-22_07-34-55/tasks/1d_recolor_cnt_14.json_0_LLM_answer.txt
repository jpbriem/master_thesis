Whenever you encounter an element that is greater than zero, you add it to the sum and increment the count. If the count becomes equal to the size of the array, you remove the first element from the sum and decrement the count. Finally, you return the sum.

```python
def max_sum(arr):
    n = len(arr)
    curr_sum = 0
    count = 0
    result = 0
    for i in range(n):
        if arr[i] > 0:
            curr_sum += arr[i]
            count += 1
        else:
            if count == n:
                curr_sum -= arr[i - n % n]
            count -= 1
        result = max(result, curr_sum)
    return result
```
