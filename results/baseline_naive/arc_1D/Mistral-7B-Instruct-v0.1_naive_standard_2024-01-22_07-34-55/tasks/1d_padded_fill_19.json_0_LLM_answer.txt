You can use a sliding window approach to solve this problem. You need to keep track of the current sum and the previous sum. If the current sum is greater than or equal to the previous sum, you add the current element to the output list. Otherwise, you remove the first element from the output list.

Here's an implementation in Python:

```
def maximumSubarraySum(numbers):
    if not numbers:
        return []
    
    # Initialize variables
    curr_sum = nums[0]
    prev_sum = nums[0]
    output = [curr_sum]
    
    for i in range(1, len(numbers)):
        # Update current sum
        curr_sum = max(prev_sum + numbers[i], curr_sum)
        
        # Check if current sum is greater than or equal to previous sum
        if curr_sum >= prev_sum:
            # Add current element to output list
            output.append(curr_sum)
            
        # Update previous sum
        prev_sum = curr_sum
    
    return output
```

This solution has a time complexity of O(n), where n is the length of the input array.
