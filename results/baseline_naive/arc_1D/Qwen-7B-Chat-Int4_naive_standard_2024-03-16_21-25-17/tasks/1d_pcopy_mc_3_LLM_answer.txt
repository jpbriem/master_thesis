```python
    def majorityElement(self, nums):
        count = {}
        for num in nums:
            if num not in count:
                count[num] = 1
            else:
                count[num] += 1
        
        max_count = max(count.values())
        
        result = []

        return result
```

Explanation:

After counting all elements, we find the maximum count and create an empty list to store the majority element(s).

Then, we iterate through the input array again and check if the current element's count is equal to the maximum count. If so, we append it to the result list.

Finally, we return the result list containing all majority elements. Note that if there are multiple majority elements, they will all be included in the result.
