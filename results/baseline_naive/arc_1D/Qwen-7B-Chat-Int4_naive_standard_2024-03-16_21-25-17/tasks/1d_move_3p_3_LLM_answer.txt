To solve this problem, we can use a hash table to count the frequency of each element in the input array. We iterate through the array and update the counts for each element. Then, we iterate through the hash table again and return an array where the index i corresponds to the value at that index in the original array.

Here's the Python code:

```python
def solution(arr):
    if len(arr) == 0:
        return arr

    freq = {}
    for num in arr:
        if num not in freq:
            freq[num] = 0
        freq[num] += 1

    result = []
    for num in arr:
        result.append(freq[num])

    return result
```
