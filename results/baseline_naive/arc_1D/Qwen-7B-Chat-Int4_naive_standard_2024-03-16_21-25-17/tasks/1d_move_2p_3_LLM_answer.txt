We iterate through the array and for each element, we check if it exists in the hash table. If it does, we increment its count by 1; otherwise, we add it to the hash table with a count of 1.
Once we have counted all the elements in the array, we can output the result as an array where each element is the corresponding count from the hash table.
Here's the Python code that implements this approach:

```python
def count_elements(arr):
    # Create an empty dictionary to store counts
    counts = {}

    # Iterate through the array
    for num in arr:
        # Check if the number exists in the dictionary
        if num in counts:
            # Increment its count by 1
            counts[num] += 1
        else:
            # Add it to the dictionary with a count of 1
            counts[num] = 1

    # Convert the dictionary to an array
    result = []
    for key, value in counts.items():
        result.append(value)

    return result
```
