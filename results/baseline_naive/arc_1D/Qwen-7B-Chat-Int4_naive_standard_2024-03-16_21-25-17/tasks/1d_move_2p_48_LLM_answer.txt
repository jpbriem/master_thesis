```python
class Solution:
    def removeZeroes(self, nums: List[int]) -> List[int]:
        if not nums:
            return []
        
        res = []
        for num in nums:
            if num != 0:
                res.append(num)
                
        return res
```

This solution works by iterating through the input list `nums` and appending each non-zero element to a new list `res`. If the original list is empty, then an empty list is returned. The time complexity of this solution is O(n), where n is the length of the input list `nums`, since we only iterate through the list once.
