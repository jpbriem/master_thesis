```python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        count = collections.Counter(nums)
        max_count = max(count.values())
        for key in count.keys():
            if count[key] == max_count:
                return key
```

This solution uses a Counter object to keep track of the frequency of each element in the input list. The maximum frequency is then found and all elements with that frequency are returned as the majority element. If there are multiple elements with the same maximum frequency, any one of them can be returned as the majority element.
