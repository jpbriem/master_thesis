The first pointer will move forward while the second pointer moves backward. Whenever the pointers meet in the middle, we'll swap the values at those indices.
Here's the Python code to implement this solution:

```python
def findMaxProduct(nums):
    if not nums or len(nums) == 1:
        return nums

    left = right = nums[0]
    max_product = nums[0]

    for i in range(1, len(nums)):
        product = nums[i] * (left if nums[left] < nums[i] else right)
        max_product = max(max_product, product)

        # Move pointers towards each other
        left = nums[i] if nums[left] < nums[i] else left - 1
        right = nums[right] if nums[right] > nums[i] else right + 1

    return max_product
```
