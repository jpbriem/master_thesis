```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        if len(strs) == 0:
            return []
        
        ans = [[] for _ in range(len(set(strs)))]
        
        for s in strs:
            hash_s = [ord(c)-97 for c in s]
            
            # find the index of this string in the list
            i = 0
            while i < len(ans) and hash_s != ans[i]:
                i += 1
            
            # append to the current group
            if i == len(ans):
                ans.append([s])
            else:
                ans[i].append(s)
                
        return ans
```

It then finds the index of this string in the `ans` list by iterating over `ans`, starting from 0, until it finds a match or reaches the end of the list.

If the string is not already in a group, it appends itself to the beginning of that group's sublist. If it is already in a group, it appends itself to the end of that group's sublist.

Finally, the function returns the `ans` list containing all groups of anagrams.
