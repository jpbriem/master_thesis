```python
class Solution:
    def majorityElement(self, nums):
        count = collections.Counter(nums)
        
        max_count = max(count.values())
        
        for num in count:
            if count[num] == max_count:
                return num
```

Explanation:

- We use a dictionary to keep track of the frequency of each number.
- Then we find the maximum frequency and loop through all the numbers again. For each number, if its frequency is equal to the maximum frequency, we return it as the majority element.

Note that this solution assumes that there is at least one majority element in the input array. If there are multiple majority elements with the same highest frequency, this solution will only return one of them.
