To solve this problem, we can use a two-pointer approach. We will have one pointer that starts at the beginning of the array and another pointer that starts at the end of the array. At each step, we compare the values at the two pointers. If they are different, we swap them and move both pointers towards the center of the array. This way, we ensure that all elements in the array are unique.

Here's the Python code to implement this approach:

```python
def findUniqueElements(nums):
    left = 0
    right = len(nums) - 1
    
    while left < right:
        if nums[left] != nums[right]:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
        else:
            left += 1
    
    return nums
```
