We will initialize the array with all zeros and then iterate through the input array. For each element, if it is equal to its current frequency at that position, we increment its frequency; otherwise, we set its frequency to one.
Once we have iterated through the entire input array, we return the resulting two-dimensional array where each element represents the frequency of its corresponding element in the original array.
Here's the Python code for the solution:
```python
def count_elements(input_array):
    # Initialize an empty two-dimensional array with all zeros
    freq_count = [[0] * len(input_array) for _ in range(len(input_array[0]))]
    
    # Iterate through the input array
    for i in range(len(input_array)):
        for j in range(len(input_array[i])):
            # If the current element is equal to its current frequency,
            # increment its frequency; otherwise, set its frequency to one
            if input_array[i][j] == freq_count[i][j]:
                freq_count[i][j] += 1
            else:
                freq_count[i][j] = 1
    
    # Return the resulting two-dimensional array
    return freq_count
```
