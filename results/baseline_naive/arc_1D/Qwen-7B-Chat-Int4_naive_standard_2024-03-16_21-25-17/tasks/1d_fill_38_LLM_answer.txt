To solve this problem, we can use a sliding window approach. We will maintain two pointers, one at the beginning of the array and one at the end. The pointer at the beginning will move forward while the pointer at the end will move backward. Whenever the value at the end pointer becomes 0, we will add all the values to the output array from the start pointer until the end pointer.

Here is the Python code for this solution:

```python
def maxSum(nums):
    if len(nums) == 0:
        return []
    
    n = len(nums)
    i = j = k = 0
    
    result = [0] * (n + 1)
    result[0] = nums[0]
    
    while True:
        # Move left pointer
        while i < n and nums[i] != 0:
            i += 1
        
        # Move right pointer
        while j >= 0 and nums[j] != 0:
            j -= 1
            
        # Add current sum to result
        total_sum = 0
        if i > 0:
            total_sum += result[i - 1]
        if j > 0:
            total_sum += result[j + 1]
        
        # Update result
        if total_sum > result[k]:
            result[k] = total_sum
            k += 1
        
        # If either pointer reaches the end, break loop
        if i == n or j == -1:
            break
    
    return result[:k]
```
