```python
class Solution:
    def permute(self, nums):
        """
        :type nums: List[int]
        :rtype: List[List[int]]
        """
        if len(nums) == 0 or len(nums) == 1:
            return [[nums]]
        
        res = []
        for i in range(len(nums)):
            temp = nums[:i] + nums[i+1:]
            self.permute(temp)
            
            # swap the element at index i with itself
            nums[i], nums[i] = nums[i], nums[i]
            
            # append it to result
            res.append([nums])
            
            # restore original order
            nums[i], nums[i] = nums[i], nums[i+1]
            
        return res
```

Finally, we append the resulting permutation to the list of results and restore the original order of the array before moving on to the next level of recursion.
