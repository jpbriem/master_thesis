```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        if not strs:
            return []
        
        d = {}
        for s in strs:
            key = "".join(sorted(s))
            if key in d:
                d[key].append(s)
            else:
                d[key] = [s]
                
        res = [[] for _ in range(len(d))]
        for k, v in d.items():
            res[v[0]] += v[1:]
            
        return res
```

This solution uses a dictionary to store the anagram groups. The keys are sorted versions of the strings and the values are lists of the anagrams. Then it iterates over the dictionary and appends each string to its corresponding list of anagrams. Finally, it returns a list of all the lists of anagrams. This approach has a time complexity of O(n * m), where n is the number of strings and m is the maximum length of any string.
