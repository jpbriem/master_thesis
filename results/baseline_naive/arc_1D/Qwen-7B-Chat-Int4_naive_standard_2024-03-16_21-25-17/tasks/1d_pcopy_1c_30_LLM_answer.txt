We'll swap the elements pointed by these pointers if they are different, and move the pointers towards each other until they meet.
Here's the Python code to implement this solution:

```python
def permute(nums):
    def swap(nums, i, j):
        nums[i], nums[j] = nums[j], nums[i]
    
    n = len(nums)
    left, right = 0, n - 1
    
    while left < right:
        # Swap elements if necessary
        if nums[left] != nums[right]:
            swap(nums, left, right)
        
        # Move left pointer towards right
        if nums[left] == 0:
            left += 1
        else:
            left += 1
        
        # Move right pointer towards left
        if nums[right] == 0:
            right -= 1
        else:
            right -= 1
            
    return nums
```
