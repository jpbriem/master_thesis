```python
class Solution:
    def majorityElement(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        count = collections.Counter(nums)
        
        max_count = max(count.values())
        for num in count.keys():
            if count[num] == max_count:
                return num
```

This solution uses a dictionary to keep track of the frequency of each element in the input list. It then iterates through the dictionary and returns the element with the highest frequency. If there are multiple elements with the same highest frequency, it will return any one of them. This approach has a time complexity of O(n) because it only requires iterating through the input list once.
