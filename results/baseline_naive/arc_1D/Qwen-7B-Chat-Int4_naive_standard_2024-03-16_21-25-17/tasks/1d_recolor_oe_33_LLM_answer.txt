```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        d = {}
        for s in strs:
            chars = list(s)
            sorted_chars = ''.join(sorted(chars))
            if sorted_chars not in d:
                d[sorted_chars] = []
            d[sorted_chars].append(s)
        
        result = [[] for _ in range(len(d))]
        for key, value in d.items():
            result[key.index('3')].extend(value)
            
        return result
```

Explanation:

We first create a dictionary `d` to store the frequency of each sorted character. We iterate through each string in `strs`, and for each string we convert it into a list of characters, sort them, and then join them back together. This gives us a unique sorted representation of the string.

Next, we check if this sorted representation is already in the dictionary. If it's not, we add it as an empty list to the dictionary. If it is, we append the original string to that list.

After all strings have been processed, we create a new list called `result` which will contain the groups of anagrams. For each sorted representation in `d`, we find its index in the sorted list of characters (which represents the position where '3' appears) and extend the corresponding element in `result` with the values from `d`.

Finally, we return the `result` list, which contains one list per group of anagrams.
