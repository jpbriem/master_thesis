```python
class Solution:
    def maxChunksToSorted(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        if not nums or len(nums) == 0:
            return 0
        
        res = []
        
        for i in range(len(nums)):
            # add the current number to the result and remove it from the list
            res.append(nums[i])
            
            # find the last element that is less than or equal to the current number
            j = i - 1
            while j >= 0 and nums[j] > nums[i]:
                res.pop()
                j -= 1
                
        return len(res)
```

Explanation:

This solution uses a dynamic programming approach to solve the problem. We iterate through the input array `nums` and for each number we add it to the result list `res`. Then we check if there are any numbers before this one that are greater than it. If so, we pop them off the end of the list until we find the first number that is less than or equal to it.

The time complexity of this solution is O(n^2), where n is the length of the input array. This is because for each number in the array, we need to search through all previous numbers to find the last one that is less than or equal to it. The space complexity is also O(n^2) since we need to store all intermediate results in the list `res`.

Note that this solution assumes that the input array contains only non-negative integers. If the input can contain negative integers or floating-point numbers, the solution would need to be modified accordingly.
