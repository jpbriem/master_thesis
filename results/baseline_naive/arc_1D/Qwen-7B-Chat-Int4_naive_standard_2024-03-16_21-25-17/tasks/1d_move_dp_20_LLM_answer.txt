The first pointer will move forward until it finds an element that is not equal to 0, while the second pointer will move backward until it finds an element that is not equal to 0.

Once we find an element at each pointer position that is not equal to 0, we swap them and continue moving the pointers towards the middle of the array. This way, all elements in between the two non-zero elements will be swapped with their corresponding elements in the other half of the array.

Here's the Python code for this solution:

```python
def merge_sorted_arrays(input):
    left = 0
    right = len(input) - 1

    while left < right:
        if input[left] == 0:
            left += 1
        elif input[right] == 0:
            right -= 1
        else:
            input[left], input[right] = input[right], input[left]
            left += 1
            right -= 1
    
    return input
```
