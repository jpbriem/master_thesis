```python
class Solution:
    def groupAnagrams(self, strs):
        """
        :type strs: List[str]
        :rtype: List[List[str]]
        """
        d = {}
        for s in strs:
            key = ''.join(sorted(s))
            if key not in d:
                d[key] = []
            d[key].append(s)
        
        result = []
        for k in d:
            result.append(d[k])
            
        return result
```

This solution uses a dictionary to store the sorted versions of each string as keys. It then iterates over the input strings and adds them to the appropriate key in the dictionary. Finally, it creates a list of lists containing all the groups of anagrams.
