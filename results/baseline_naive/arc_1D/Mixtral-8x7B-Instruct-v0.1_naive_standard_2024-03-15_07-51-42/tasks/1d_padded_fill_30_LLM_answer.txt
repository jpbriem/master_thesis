You can use a sliding window approach. The idea is to keep track of the last index where you saw a non-zero value and then iterate over your array from that point onwards until you see another non-zero value or reach the end of the array. In this way, you will be able to find all consecutive groups of zeroes in between two non-zero values. Then, if the current group has more than one element, replace it with the desired number. def fill(arr, n):
    i = len(arr) - 1
    while i >= 0 and arr[i] == 0:
        i -= 1
    j = i
    while j < len(arr):
        if arr[j]!= 0:
            k = j + 1
            while k < len(arr) and arr[k] == 0:
                k += 1
            if k - j > 1:
                arr[j+1:k] = [n] * (k - j - 1)
            j = k
        else:
            j += 1
