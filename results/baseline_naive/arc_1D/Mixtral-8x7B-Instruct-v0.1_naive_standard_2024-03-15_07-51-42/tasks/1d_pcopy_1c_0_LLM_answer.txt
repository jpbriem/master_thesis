You can use a stack to keep track of the current island indexes as well as the previous one. If the next element is zero then we check if it has an island on its left or right side by checking the top two elements in our stack. We also pop them off since they are no longer relevant after this step.

```
def expandIslands(arr):
    stack = []
    for i, x in enumerate(arr):
        if x == 1:
            stack.append((i, -1)) # add new island with previous value set to -1
        elif len(stack) > 0:
            prev, curr = stack[-1], stack[-2]
            if arr[prev[0]] == 1:
                arr[prev[0]] = max(arr[prev[0]], curr + 2)
            if arr[curr[0]] == 1:
                arr[curr[0]] = max(arr[curr[0]], prev + 2)
            stack.pop()
            stack.pop()
    return arr
