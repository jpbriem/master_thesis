You can use a sliding window approach. The idea is to keep track of the last seen non-zero element and its value in two variables say `last_seen` and `val`. Then iterate over the array using a variable `i` as start index of the current window. If you find a zero at position `i`, then update the end index of the previous window by setting it equal to `i - 1`. Now if there was some non-zero value present before this zero, copy that value from `val` into the array starting from the beginning of the current window till the end of the previous one. Increment `i` so that we move on to the next window. Continue this process until the end of the array. Here's how you can implement it:

```
def fill(arr):
    n = len(arr)
    val, i = arr[0], 0
    last_seen = 0
    while i < n:
        if arr[i]:
            last_seen = i
            val = arr[i]
        else:
            j = max(0, last_seen - 1)
            k = min(i, last_seen + val - 1)
            while j <= k:
                arr[j] = val
                j += 1
            last_seen = i
        i += 1
    return arr
