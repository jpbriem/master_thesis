You can use a simple for loop to iterate over the array and check if there is any non-zero value in the next `n` elements where `n` is the number of consecutive zeroes before the current element. If yes then replace all those zeros with that non-zero value. Here is an example implementation -

```
def fillZeros(arr):
    n = len(arr)
    i = 0
    while i < n:
        # Find how many consecutive zeroes are present at index 'i'.
        count = 0
        while i + count < n and arr[i+count] == 0:
            count += 1

        # Check if there is any non-zero value in the next 'count' elements.
        j = min(i + count, n - 1)
        foundNonZero = False
        while j > i:
            if arr[j]!= 0:
                foundNonZero = True
                break
            j -= 1

        # Replace all those zeroes with the first non-zero value encountered.
        if foundNonZero:
            val = arr[j]
            while i <= j:
                arr[i] = val
                i += 1
        else:
            i += count
