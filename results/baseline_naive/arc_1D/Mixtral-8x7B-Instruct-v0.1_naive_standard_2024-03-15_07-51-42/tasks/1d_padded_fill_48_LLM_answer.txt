## Answer (4)

You can use a sliding window approach. The idea is to keep track of the last index where you saw a non-zero value and then iterate over the array from that point onwards until you see another non-zero value or reach the end of the array. In each iteration, if the current element is zero, copy it from the previous position in the array; otherwise, update the last seen index and continue with the next iteration. Here's an implementation using Python:

```
def fillZeros(arr):
    n = len(arr)
    i = -1 # initialize last non-zero index as -1
    j = 0  # start iterating from first element
    while j < n:
        if arr[j]!= 0:
            i = j # update last non-zero index
        else:
            if i == -1: # no non-zero elements found yet
                j += 1
                continue
            arr[j] = arr[i] # replace zero by the last non-zero value
        j += 1
    return arr
```

## Answer (1)

Here is a simple way to do it in python:

```
def fillZeros(arr):
    prev = None
    for i, x in enumerate(arr):
        if x!= 0:
            prev = i
        elif prev is not None:
            arr[i] = arr[prev]
    return arr
```

This function uses a variable `prev` to store the most recent index at which a non-zero number was encountered. If the current number is zero and there has been a non-zero number before, then the current number is replaced by the number stored in `prev`. Otherwise, nothing happens.

Note that this function does not modify the original list, so you need to assign its result back to your list if you want to change it in place. For example:

```

l[:] = fillZeros(l)
