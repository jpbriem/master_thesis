```
## Answer (1)

You can use a simple loop to do this. You just need to keep track of whether you're in an island or not and if so what its value was. If you find a zero while in an island then it ends there. Otherwise, when you leave an island you replace all zeros with the last island value until you reach another island. Here's how that looks as code:

```
def fillIslands(nums):
    # Keep track of whether we are currently in an island
    inIsland = False
    # And if so, what its value was
    lastValue = None

    i = 0
    while i < len(nums):
        if nums[i] == 0:
            # We found a zero - end the current island
            inIsland = False
            lastValue = None
        else:
            if not inIsland:
                # Start a new island
                inIsland = True
                lastValue = nums[i]
            elif nums[i]!= lastValue:
                # This isn't part of the same island
                inIsland = False
                lastValue = None
            else:
                # It is still part of the same island
                pass

        if inIsland:
            # Replace any zeros with the island value
            j = i + 1
            while j < len(nums) and nums[j] == 0:
                nums[j] = lastValue
                j += 1

        i += 1
