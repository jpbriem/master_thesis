## Answer (1)

You can use a sliding window approach. The idea is to keep track of the number of consecutive zeros and ones in two variables `zeros` and `ones`. If you encounter a zero while `ones > 0`, then replace all the ones with zeros until there are no more ones left. Then reset both counters. Here's how it looks like:

```
def fix(arr):
    n = len(arr)
    i = j = zeros = ones = 0
    ans = [-1]*n
    while j < n:
        if arr[j] == 0:
            zeros += 1
        else:
            ones += 1
        # slide the window when we have at least one group of consecutive ones
        if ones > 0:
            ans[i] = 0
            i += 1
        elif zeros > 0:
            ans[i] = 0
            i += 1
        j += 1
    return ans
```

This solution has time complexity O(N), where N is the length of the array. It also uses O(N) extra space for the answer array.
