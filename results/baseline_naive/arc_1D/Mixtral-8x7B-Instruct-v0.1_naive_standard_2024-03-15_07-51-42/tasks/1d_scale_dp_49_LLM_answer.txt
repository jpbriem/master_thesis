## Answer (1)

You can use a simple loop and check if you are at the start of the sequence or not by checking if the previous element was zero or not. If yes then update the current index with the first non-zero value found in the sequence else just copy that value as it is.

Here's how you can do this using python -

```
def fillSequence(numbers):
    prev = None # initialize previous number as none
    i = 0
    while i < len(numbers):
        if numbers[i] == 0:
            i += 1
        elif prev is None:
            prev = numbers[i]
            i += 1
        else:
            numbers[i] = prev
            i += 1
    return numbers
```
