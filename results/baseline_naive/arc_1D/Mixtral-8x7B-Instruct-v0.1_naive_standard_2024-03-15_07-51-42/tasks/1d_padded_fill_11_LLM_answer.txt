## Answer (1)

You can use a simple for loop to iterate through the array and check if there is an island on either side of the current element. If there isn't then set it to zero. Here is some sample code that should work:

```
def removeIslands(arr):
    n = len(arr)
    m = len(arr[0])
    # Iterate over each cell in the grid
    for i in range(n):
        for j in range(m):
            # Check if this cell is part of an island
            if arr[i][j] == 1 and areNeighborsZero(arr, i, j):
                # Set all cells in the connected component to zero
                floodFill(arr, i, j, 0)

# Function to check if both neighbors of a given cell are zero
def areNeighborsZero(grid, x, y):
    return (x > 0 and grid[x - 1][y] == 0) or \
           (x < len(grid) - 1 and grid[x + 1][y] == 0) or \
           (y > 0 and grid[x][y - 1] == 0) or \
           (y < len(grid[0]) - 1 and grid[x][y + 1] == 0)

# Flood fill algorithm to mark all cells in a connected component as visited
def floodFill(grid, x, y, newVal):
    if x < 0 or x >= len(grid) or y < 0 or y >= len(grid[0]) or grid[x][y]!= 1:
        return
    grid[x][y] = newVal
    floodFill(grid, x - 1, y, newVal)
    floodFill(grid, x + 1, y, newVal)
    floodFill(grid, x, y - 1, newVal)
    floodFill(grid, x, y + 1, newVal)
```

This will modify the input array in place. You can call `removeIslands()` with your example arrays to see how it works.
