## Answer (4)

You can use a sliding window approach. The idea is to keep track of the last index where you found a non-zero number and then iterate over your array from that point onwards until you find another non-zero number or reach the end of the array. If you don't find any other non-zero numbers in this interval, replace all zeros with the value at the first position of the interval. Here is some sample code:

```
def fill(arr):
    i = len(arr) - 1
    while arr[i] == 0:
        i -= 1

    j = i
    while j < len(arr) and arr[j]!= 0:
        j += 1

    if j > i + 1:
        val = arr[i]
        for k in range(i+1, j):
            arr[k] = val

# Test cases
