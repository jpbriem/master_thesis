## Answer (1)

You can use a sliding window approach. The idea is to keep track of the last index where you found an element that should be replaced and then replace all elements between this index and the current one if they are zeroes. Here's how it looks like in code:

```
def fill(arr):
    i = 0
    while i < len(arr):
        # find next non-zero element or end of array
        j = arr[i:].index(6) + i if 6 in arr[i:] else len(arr)
        # replace zeros with sixes
        arr[i:j] = [6] * (j - i) if j > i and arr[i] == 0 else []
        i = j
    return arr
```
