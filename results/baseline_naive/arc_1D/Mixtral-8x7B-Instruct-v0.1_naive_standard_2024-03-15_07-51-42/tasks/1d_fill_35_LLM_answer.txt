You can use a simple for loop to iterate through the array and check if there is any non-zero value in the current window of size `k`. If yes then replace all values with that number else continue. Here is an example implementation -

```
def fill(arr, k):
    n = len(arr)
    i = j = 0
    while j < n:
        count = sum(arr[i : min(j + k, n)])
        if count > 0:
            arr[i : min(j + k, n)] = [count] * min(k, n - i)
        i += 1
        j += 1
