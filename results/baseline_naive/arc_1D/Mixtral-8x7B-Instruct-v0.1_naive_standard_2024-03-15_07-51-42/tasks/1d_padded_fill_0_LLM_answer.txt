## Answer (1)

You can use a sliding window approach. The idea is to keep track of the last index where you found a non-zero number and then iterate over your array from that point onwards until you find another non-zero number or reach the end of the array. If you don't find any other non-zero numbers in this interval, it means that all the zeros between them should be replaced with the value at the first position. Here's how you could implement this using Python:

```
def replace_zeros(arr):
    i = 0
    while i < len(arr):
        if arr[i]: # Found a non-zero number
            j = i + 1
            while j < len(arr) and not arr[j]: # Find next non-zero number
                j += 1
            if j == len(arr): # No more non-zero numbers were found
                k = i
                while k >= 0 and not arr[k]: # Replace all zeroes up to the previous non-zero number
                    arr[k] = arr[i]
                    k -= 1
            else:
                i = j # Move to the next non-zero number
        else:
            i += 1 # Skip zeroes
    return arr
```
