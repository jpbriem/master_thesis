## Answer (4)

You can use a sliding window approach. The idea is to keep track of the last index where you found a number different than zero and then iterate over your array with a step equal to that value. In this way you will always check if there are other non-zero values in the same group. If so, update the end position of the current group. When the next iteration starts it will be able to detect the new group because the start position has been updated as well.

Here's an example implementation using Python:

```
def expand(arr):
    i = 0
    while i < len(arr):
        # Find first non-zero element
        j = i
        while arr[j] == 0:
            j += 1
        i = j

        # Expand groups
        k = j + 1
        while k < len(arr) and arr[k]!= 0:
            k += 1
        j = k - 1

        # Update elements between i and j
        while i <= j:
            arr[i] = arr[j]
            i += 1

**EDIT:** I just realized that my solution doesn't work when there are multiple consecutive zeros at the beginning or at the end of the input list. Here's a modified version which should fix this issue:

```
def expand(arr):
    i = 0
    while i < len(arr):
        # Find first non-zero element
        j = i
        while j < len(arr) and arr[j] == 0:
            j += 1
        i = j

        # Expand groups
        k = j + 1
        while k < len(arr) and arr[k]!= 0:
            k += 1
        j = k - 1

        # Update elements between i and j
        while i <= j:
            arr[i] = arr[j]
            i += 1

# Remove leading/trailing zeros
while arr[0] == 0:
    del arr[0]
while arr[-1] == 0:
    del arr[-1]

# Add back missing zeros
if arr:
    arr.insert(0, 0)
    arr.append(0)
```
