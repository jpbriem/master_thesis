You can use a sliding window approach. The idea is to keep track of the first index where you have an element that should be replaced and then replace all elements from this index until the end of the array with the new character, if such an index exists. If no such index exists, it means that there are only dots in the input string so we just return the same string as the result. Here's how you could implement it in Python:

```
def removeDupes(arr):
    i = 0
    while i < len(arr) - 1:
        # Find the next non-dot character after the current one
        j = i + 1
        while arr[j] == '.':
            j += 1

        # Replace all characters between the current position and the found one
        # with the new character
        if j < len(arr):
            arr[i+1 : j] = [arr[i]] * (j - i - 1)

        i = j

    return arr
```
