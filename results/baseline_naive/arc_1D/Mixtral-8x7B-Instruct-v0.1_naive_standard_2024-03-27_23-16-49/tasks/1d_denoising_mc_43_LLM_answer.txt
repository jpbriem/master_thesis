You can use a sliding window approach. The idea is to keep track of all elements in the current window and check if it contains any invalid element or not. If yes, then remove the leftmost element from the window and add the next element on its right side into the window. Keep doing this until you find that the current window does not contain any invalid element. Once found such a valid window, update your answer accordingly.

Here's how we can implement this using Python:

```
def longestValidSubstring(str):
    n = len(str)
    maxLen = 0
    i = 0
    while i < n:
        j = i
        count = 0
        # Count number of invalid characters in the current window
        while j < n and str[j]!= '.':
            if str[j] == 'a':
                count += 1
            j += 1

        # Check if there are more than one invalid character in the current window
        if count > 1:
            # Remove the leftmost invalid character from the window
            while i < j and str[i]!= '.':
                if str[i] == 'a':
                    count -= 1
                i += 1
            i += 1
        else:
            # Update the maximum length substring so far
            maxLen = max(maxLen, j - i)
            i = j + 1

    return maxLen
```
