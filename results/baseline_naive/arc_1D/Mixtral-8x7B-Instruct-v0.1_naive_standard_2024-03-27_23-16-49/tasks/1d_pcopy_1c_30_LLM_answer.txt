You can use a sliding window approach. The idea is to keep track of the last position where you found an "h" and then check if there are any other "h"s in the next positions until you find another non-"h". If so, replace all those "h"s with "."s. Here's how it would look like:

```
def fix(arr):
    i = 0 # index of last h
    while i < len(arr):
        if arr[i] == 'h':
            j = i + 1
            while j < len(arr) and arr[j]!= '.':
                j += 1
            if j > i + 1:
                arr[i+1:j] = ['.' for _ in range(j - i - 1)]
            i = j
        else:
            i += 1
    return arr
```
