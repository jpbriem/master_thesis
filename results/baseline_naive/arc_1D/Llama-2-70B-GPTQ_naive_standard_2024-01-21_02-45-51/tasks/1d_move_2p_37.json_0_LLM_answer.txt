```python
class Solution:
    def rotate(self, nums):
        """
        :type nums: List[int]
        :rtype: void Do not return anything, modify nums in-place instead.
        """
        
        for i in range(len(nums)):
            if nums[i] == 0:
                continue
            
            j = len(nums)-1 - i
            while j > 0:
                temp = nums[j]
                nums[j] = nums[i]
                nums[i] = temp
                
                i += 1
                j -= 1
```
