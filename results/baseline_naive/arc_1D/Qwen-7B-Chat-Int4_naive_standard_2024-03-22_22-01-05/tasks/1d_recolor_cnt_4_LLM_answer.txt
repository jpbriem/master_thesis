To solve this problem, we can use a hash set to keep track of the characters that have appeared so far. We iterate through the input string and add each character to the set if it has not already been seen. If a character appears multiple times in a row, we replace it with its last appearance in the set.
Here's the Python code to implement this approach:

```
def longest_consecutive_subsequence(s):
    # Create an empty set to store unique characters
    unique_chars = set()
    
    # Initialize variables to keep track of the maximum length and current sequence
    max_length = 0
    current_sequence = []
    
    # Iterate through the input string
    for char in s:
        # Add the character to the set if it is not already there
        if char not in unique_chars:
            unique_chars.add(char)
        
        # Check if the character is the same as the previous one
        elif len(current_sequence) > 0 and current_sequence[-1] == char:
            # If it is, update the current sequence
            current_sequence.append(char)
            
        else:
            # Otherwise, reset the current sequence and update the maximum length
            current_sequence = [char]
            max_length = max(max_length, len(current_sequence))
    
    # Return the maximum length found
    return max_length
```

This solution has a time complexity of O(n), where n is the length of the input string, since we only need to iterate through the string once. The space complexity is also O(n), since we need to store all unique characters in the set.
