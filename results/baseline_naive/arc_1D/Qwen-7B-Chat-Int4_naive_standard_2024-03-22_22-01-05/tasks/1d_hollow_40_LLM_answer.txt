To solve this problem, we can use a sliding window approach. We will keep track of the current window and update it whenever we encounter an "a". When we encounter a "b", we will remove all elements in the current window that are not "a" and add "b" to the end of the window.
Here's the Python code for this solution:

```python
def find_unique_elements(nums):
    result = []
    i = 0
    while i < len(nums):
        if nums[i] == 'a':
            j = i + 1
            while j < len(nums) and nums[j] != 'b':
                j += 1
            if j >= len(nums):
                break
            result.append(''.join(nums[i:j]))
            i = j
        else:
            i += 1
    return result
```
