```python
class Solution:
    def groupAnagrams(self, strs):
        """
        :type strs: List[str]
        :rtype: List[List[str]]
        """
        anagrams = {}
        
        for s in strs:
            key = "".join(sorted(s))
            if key not in anagrams:
                anagrams[key] = []
            anagrams[key].append(s)
            
        return [anagrams[i] for i in sorted(anagrams)]
```

In this solution, we use a dictionary `anagrams` to store the anagrams. We iterate over each string in the input list and sort its characters to get a unique key. Then we check whether the key is already in the dictionary or not. If it's not, we create a new list containing the current string as an element and add it to the dictionary with the key as the value. If it's already in the dictionary, we append the current string to the corresponding list.

Finally, we convert the dictionary into a list of lists by sorting the keys of the dictionary and using them as indices to access the corresponding lists. This gives us the desired output.
