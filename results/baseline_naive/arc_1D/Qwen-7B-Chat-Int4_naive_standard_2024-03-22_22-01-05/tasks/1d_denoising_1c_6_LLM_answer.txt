To solve this problem, we can use a simple approach. We iterate through the input array and keep track of the frequency of each character using a dictionary. Then, we create an output array that contains only the characters with their corresponding frequencies. Finally, we sort the output array in descending order based on the frequency of each character.

Here's the Python code to implement this solution:

```python
def most_common_chars(input_array):
    freq_dict = {}
    for char in input_array:
        if char in freq_dict:
            freq_dict[char] += 1
        else:
            freq_dict[char] = 1

    output_array = []
    for key, value in freq_dict.items():
        output_array.append((key, value))

    output_array.sort(key=lambda x: x[1], reverse=True)

    return [char for char, _ in output_array]
```
