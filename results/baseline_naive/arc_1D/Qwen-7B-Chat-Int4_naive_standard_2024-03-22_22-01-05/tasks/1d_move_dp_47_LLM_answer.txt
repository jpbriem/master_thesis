To solve this problem, we can iterate through the input string and keep track of two pointers: one to move forward in the string (pointer1) and one to move backward in the string (pointer2). At each step, we compare the characters at pointer1 and pointer2. If they are equal, we increment both pointers and continue comparing. If they are different, we swap the characters at pointer1 and pointer2.

Here's the Python code for this approach:

```python
def replace_chars(input_string):
    pointer1 = 0
    pointer2 = len(input_string) - 1
    
    while pointer1 < pointer2:
        if input_string[pointer1] == input_string[pointer2]:
            # Increment both pointers
            pointer1 += 1
            pointer2 -= 1
            
        else:
            # Swap characters at pointer1 and pointer2
            temp_char = input_string[pointer1]
            input_string[pointer1] = input_string[pointer2]
            input_string[pointer2] = temp_char
            
            # Move pointer1 and pointer2 closer together
            if pointer1 > pointer2:
                pointer1 -= 1
            else:
                pointer2 += 1
                
    return input_string
```
