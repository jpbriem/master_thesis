To solve this problem, we can use a hash table to keep track of the frequency of each character in the input string. We iterate over the input string and for each character, we update its frequency in the hash table. If the frequency is greater than or equal to 2, we remove all characters with that frequency from the input string. Finally, we return the updated string.

Here's the Python code to implement this solution:

```python
def removeDuplicates(input_string):
    freq = {}
    output_string = []
    
    for char in input_string:
        if char not in freq:
            freq[char] = 0
        freq[char] += 1
        
    for char, count in freq.items():
        if count >= 2:
            while count > 0:
                output_string.append(char)
                count -= 1
                
    return ''.join(output_string)
```
