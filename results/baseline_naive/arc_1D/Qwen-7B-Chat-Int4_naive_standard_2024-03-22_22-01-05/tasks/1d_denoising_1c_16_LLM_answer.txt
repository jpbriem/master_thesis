```python
def longest_common_subsequence(input):
    m = len(input)
    n = len(set(input))
    
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if input[i - 1] == input[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
                
    result = []
    i, j = m, n
    
    while i > 0 and j > 0:
        if input[i - 1] == input[j - 1]:
            result.append(input[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] >= dp[i][j - 1]:
            i -= 1
        else:
            j -= 1
            
    return ''.join(result[::-1])
```

It uses dynamic programming to build up a table `dp`, where `dp[i][j]` represents the length of the longest common subsequence between the first `i` characters of the input and the first `j` unique characters of the set.

Once the table is built, we can iterate over it to construct the actual longest common subsequence by backtracking through the rows and columns of the table, starting from the bottom right corner and working our way up and left.

Finally, we reverse the resulting sequence and join its elements together into a string before returning it.
