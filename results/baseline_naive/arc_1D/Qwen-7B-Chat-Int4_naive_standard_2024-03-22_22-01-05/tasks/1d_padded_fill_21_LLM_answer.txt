To solve this problem, we can use a hash set to keep track of the characters that have already been seen in each input string. We iterate over each character in the input string and check if it is already in the hash set for the current string. If it is not, we add it to the hash set and append it to the output string. If it is already in the hash set, we skip it.

Here's the Python code:

```python
def group_chars(input):
    # Initialize an empty list to store the output strings
    output = []

    # Create a hash set to keep track of characters that have already been seen
    seen = set()

    # Iterate over each input string
    for i in range(len(input)):
        # Get the current string
        s = input[i]

        # Check if the current string has any characters that have not yet been seen
        new_chars = [char for char in s if char not in seen]

        # Add the new characters to the output string
        output.append(''.join(new_chars))

        # Update the hash set with the characters in the current string
        seen.update(s)

    return output
```
