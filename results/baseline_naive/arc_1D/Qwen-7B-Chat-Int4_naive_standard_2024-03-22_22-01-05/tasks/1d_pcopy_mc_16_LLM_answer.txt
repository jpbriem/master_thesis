To solve this problem, we can use a dictionary to keep track of the frequency of each character in the input string. We iterate through the string and update the frequency count for each character encountered. Once all characters have been processed, we return the list of unique characters sorted in descending order based on their frequency count.
Here is an example implementation in Python:

```python
def process_string(input_str):
    # Initialize empty dictionary to store character frequencies
    freq_dict = {}

    # Iterate through the string and update frequency counts
    for char in input_str:
        if char in freq_dict:
            freq_dict[char] += 1
        else:
            freq_dict[char] = 1

    # Convert dictionary to list of tuples with (character, frequency)
    freq_list = [(char, freq) for char, freq in freq_dict.items()]

    # Sort list by frequency in descending order
    freq_list.sort(key=lambda x: x[1], reverse=True)

    # Return unique characters from sorted list
    unique_chars = [char for char, _ in freq_list]

    return unique_chars
```
