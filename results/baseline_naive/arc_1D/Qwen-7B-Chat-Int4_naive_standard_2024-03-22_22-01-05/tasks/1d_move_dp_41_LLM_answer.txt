To solve this problem, we can use a hash set to keep track of the characters that have already been seen in the input string. We iterate over each character in the input string and check if it has already been seen. If not, we add it to the output list. Finally, we return the output list.
Here's an implementation in Python:

```python
def group_chars(input):
    seen = set()
    output = []
    for char in input:
        if char not in seen:
            seen.add(char)
            output.append(char)
    return output
```
