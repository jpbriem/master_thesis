To solve this problem, we can use a set to keep track of the characters that have been seen so far. We iterate over the input string and add each character to the set if it has not already been seen. If a character is seen for the first time in the string, we append it to the output list.
Here's the Python code that implements this approach:

```
def remove_duplicates(input):
    seen = set()
    output = []
    for char in input:
        if char not in seen:
            seen.add(char)
            output.append(char)
    return output
```
