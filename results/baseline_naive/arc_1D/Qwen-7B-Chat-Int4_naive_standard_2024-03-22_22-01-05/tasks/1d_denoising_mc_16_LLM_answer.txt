To solve this problem, we can iterate through the input array and keep track of two pointers, one at the beginning and one at the end. We will move the pointer to the right until it reaches an element that is not equal to the current character being processed in the left pointer's position. At that point, we will swap the characters between the two pointers and continue moving the right pointer forward. This process will repeat until all characters have been processed.
Here is a Python implementation of this approach:

```python
def removeDuplicates(input):
    # initialize pointers
    i = 0
    j = len(input) - 1
    
    while i < j:
        if input[i] != input[j]:
            # swap characters
            input[i], input[j] = input[j], input[i]
            
            # move right pointer
            i += 1
            
        else:
            # move left pointer
            j -= 1
            
    return input
```
