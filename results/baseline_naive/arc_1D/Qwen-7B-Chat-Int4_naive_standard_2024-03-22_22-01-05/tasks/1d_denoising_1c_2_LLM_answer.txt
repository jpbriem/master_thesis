To solve this problem, we can use a hash set to keep track of the unique characters in each string. We iterate through each input string and add its characters to the corresponding hash set. If any character is already present in the hash set for that string, it means that the string contains duplicates and we return an error message.

Here's the Python code to implement this solution:

```python
def remove_duplicates(strings):
    result = []
    seen_strings = {}
    
    for string in strings:
        if string not in seen_strings:
            seen_strings[string] = True
            result.append(string)
        else:
            print("Error: String {} contains duplicates".format(string))
            
    return result
```
