To solve this problem, we can use a hash set to keep track of the characters that have already been seen in the input string. We iterate over each character in the input string and check if it has already been seen. If not, we add it to the output string. Finally, we return the output string.

Here is the Python code for this solution:

```
def remove_duplicates(input_string):
    # Create an empty hash set to store the characters that have been seen
    seen = set()

    # Initialize an empty list to store the output string
    output = []

    # Iterate over each character in the input string
    for char in input_string:
        # Check if the character has already been seen
        if char not in seen:
            # Add the character to the output string and the seen set
            output.append(char)
            seen.add(char)

    # Return the output string
    return ''.join(output)
```
