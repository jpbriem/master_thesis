```python
def longest_common_subsequence(input):
    m = len(input)
    n = len(max(input, key=len))
    
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if input[i - 1] == max(input, key=len)[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
                
    result = []
    i, j = m, n
    
    while i > 0 and j > 0:
        if input[i - 1] == max(input, key=len)[j - 1]:
            result.append(input[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] >= dp[i][j - 1]:
            i -= 1
        else:
            j -= 1
            
    return ''.join(result[::-1])
```

This solution uses dynamic programming to build a matrix `dp` where `dp[i][j]` represents the length of the longest common subsequence between the first `i` characters of the input and the first `j` characters of the maximum string in the input. We then backtrack through this matrix to construct the actual sequence.

The time complexity of this solution is O(nm), where n and m are the lengths of the two inputs. The space complexity is also O(nm).
