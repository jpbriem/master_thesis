Otherwise, if the current character is equal to the previous character, we check if it has appeared more than once in the input string. If it has, we skip it; otherwise, we add it to the output string.
Here's the Python code that implements this solution:

```python
def group_chars(input_str):
    output_str = []
    prev_char = None

    for char in input_str:
        if char != prev_char:
            output_str.append(char)
        else:
            count = 0
            for c in input_str:
                if c == char:
                    count += 1
                    if count > 1:
                        break
            if count == 1:
                output_str.append(char)

        prev_char = char

    return ''.join(output_str)
```
