```python
def longest_common_subsequence(input):
    n = len(input)
    dp = [[0] * (n + 1) for _ in range(n + 1)]
    
    # Build dynamic programming table
    for i in range(1, n+1):
        for j in range(1, n+1):
            if input[i-1] == input[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
                
    # Backtrack to find the LCS
    lcs = []
    i = n
    j = n
    
    while i > 0 and j > 0:
        if input[i-1] == input[j-1]:
            lcs.append(input[i-1])
            i -= 1
            j -= 1
        elif dp[i-1][j] >= dp[i][j-1]:
            i -= 1
        else:
            j -= 1
            
    return ''.join(reversed(lcs))
```

If the characters match, the length of the LCS is increased by 1; otherwise, it is taken from the maximum of the lengths of the LCS obtained by ignoring the current character or the prefix without it.

Finally, the function backtracks through the `dp` table to find the LCS itself, starting from the end of the input and working backwards towards the beginning. It keeps track of the characters that make up the LCS as it goes along, and returns them in reverse order.
