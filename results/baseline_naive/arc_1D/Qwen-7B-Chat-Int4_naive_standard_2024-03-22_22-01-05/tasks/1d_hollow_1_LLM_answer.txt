```python
def group_chars(input):
    # Create a set of characters that have been seen
    seen = set()

    # Initialize an empty list to store the output
    output = []

    for char in input:
        if char not in seen:
            # If we haven't seen this character before, add it to the set and append it to the output
            seen.add(char)
            output.append(char)
        else:
            # If we have seen this character before, just append it to the output
            output.append(char)

    return output
```

This function takes in a list of characters `input` and returns a new list `output` containing only the unique characters. It uses a set called `seen` to keep track of which characters have already been seen. For each character in `input`, the function checks whether it has been seen before by checking if it is in the `seen` set. If the character hasn't been seen before, it adds it to both the `seen` set and the `output` list. If the character has been seen before, it simply appends it to the `output` list.
