To solve this problem, we can use a hash table to keep track of the frequency of each character in the input string. We iterate through the string and update the frequency count for each character encountered. Then, we iterate through the hash table and return all characters that have a frequency greater than or equal to 2.

Here's an implementation in Python:

```python
def find_duplicates(input_string):
    freq = {}
    duplicates = []
    
    # Count the frequency of each character in the input string
    for char in input_string:
        if char in freq:
            freq[char] += 1
        else:
            freq[char] = 1
    
    # Iterate through the hash table and return all characters with a frequency >= 2
    for char, freq_count in freq.items():
        if freq_count >= 2:
            duplicates.append(char)
    
    return duplicates
```
