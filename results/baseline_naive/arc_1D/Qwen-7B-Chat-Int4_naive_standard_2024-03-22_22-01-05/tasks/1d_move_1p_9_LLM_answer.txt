To solve this problem, we can use a set to keep track of the characters that have been seen so far. We iterate over the input string and add each character to the set if it has not already been encountered. If a character is encountered multiple times, we only need to keep one copy in the output.
Here's the Python code to implement this solution:

```python
def unique_chars(input_string):
    # Initialize an empty set to store unique characters
    unique_set = set()

    # Iterate over the input string
    for char in input_string:
        # Add the character to the set if it has not already been encountered
        if char not in unique_set:
            unique_set.add(char)

    # Convert the set to a list and return it as the output
    return list(unique_set)
```
