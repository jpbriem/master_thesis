To solve this problem, we can use a hash table to count the frequency of each character in the input string. We can then iterate through the hash table and remove any characters that have a frequency greater than 1.

Here's an implementation in Python:

```python
def remove_duplicates(input_string):
    # Create a hash table to count the frequency of each character
    char_count = {}
    for char in input_string:
        if char not in char_count:
            char_count[char] = 0
        char_count[char] += 1
    
    # Remove characters with a frequency greater than 1
    output_string = []
    for char in input_string:
        if char_count[char] == 1:
            output_string.append(char)
    
    return ''.join(output_string)
```

This function takes an input string as input and returns a new string with all duplicates removed. The time complexity is O(n) where n is the length of the input string, since we only need to iterate through the input once. The space complexity is also O(n), since we need to store the frequency of each character in the hash table.
