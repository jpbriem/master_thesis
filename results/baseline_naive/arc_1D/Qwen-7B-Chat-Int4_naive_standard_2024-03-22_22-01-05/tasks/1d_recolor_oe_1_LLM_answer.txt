To solve this problem, we can use a dictionary to keep track of the frequency of each character in the input string. We will iterate over the characters and update the frequency count in the dictionary. Then, we will iterate over the dictionary again and replace any character that appears more than once with its corresponding index in the dictionary.

Here is the Python code for this solution:

```python
def group_chars(input_str):
    # Initialize an empty dictionary to store the frequency of each character
    freq_dict = {}

    # Iterate over the characters in the input string
    for char in input_str:
        if char in freq_dict:
            freq_dict[char] += 1
        else:
            freq_dict[char] = 1

    # Create a new output string by replacing each character with its corresponding index in the dictionary
    output_str = ''
    for i, char in enumerate(input_str):
        if char in freq_dict:
            output_str += str(freq_dict[char])
        else:
            output_str += char

    return output_str
```
