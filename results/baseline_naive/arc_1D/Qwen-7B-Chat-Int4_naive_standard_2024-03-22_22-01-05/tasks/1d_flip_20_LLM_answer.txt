Whenever we encounter a different character in the window, we will append it to the output array and remove all other characters from the window.
Here is the Python code for the solution:

```
def find_unique_chars(input):
    output = []
    current_char = None
    for char in input:
        if char != current_char:
            current_char = char
            output.append(char)
        else:
            output.pop()
    return output
```
