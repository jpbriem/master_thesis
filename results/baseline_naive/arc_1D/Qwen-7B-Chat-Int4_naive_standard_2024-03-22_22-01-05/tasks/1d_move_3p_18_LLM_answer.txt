```python
def group_chars(input):
    # Create an empty list to store the groups of characters
    output = []
    
    # Initialize a variable to keep track of the current group
    current_group = []
    
    # Iterate over each character in the input string
    for char in input:
        # If the character is already in the current group, append it to the output and start a new group
        if char in current_group:
            output.append(current_group)
            current_group = [char]
        else:
            # Otherwise, add the character to the current group
            current_group.append(char)
    
    # Add the last group to the output
    output.append(current_group)
    
    return output
```

This function takes in a string `input` as its argument and returns a list of lists containing the groups of characters. The algorithm works by iterating over each character in the input string, and keeping track of the current group using a list called `current_group`. If the character is already in the current group, it appends it to the output and starts a new group. Otherwise, it adds the character to the current group. Finally, after all characters have been processed, it adds the last group to the output.
