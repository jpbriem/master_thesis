```python
def longest_common_subsequence(input):
    n = len(input)
    dp = [[0] * (n + 1) for _ in range(n + 1)]

    # Build the dp table
    for i in range(1, n + 1):
        for j in range(1, n + 1):
            if input[i - 1] == input[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    # Find the maximum length of common subsequence
    result = []
    index = n
    while index > 0:
        result.append(input[index - 1])
        index -= dp[index][index]

    return ''.join(result[::-1])
```

This function takes an input string as input and returns the longest common subsequence. It uses dynamic programming to build a dp table where `dp[i][j]` represents the length of the longest common subsequence between the first `i` characters of the input and the first `j` characters of the input. The function then iterates through the dp table starting from the bottom-right corner and works its way up to find the maximum length of common subsequence.

The time complexity of this algorithm is O(n^2), where n is the length of the input string. This is because we need to iterate through each character of both strings once to build the dp table. However, since we only store one row and one column of the dp table at any given time, the space complexity of this algorithm is O(n).
