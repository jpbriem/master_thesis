To solve this problem, we can use a sliding window approach. We maintain two pointers, one at the beginning of the string and another at the end. At each step, we move the left pointer to the right until it reaches the end of the string or encounters an element that is not equal to the current element at the right pointer. If we encounter such an element, we swap it with the current element at the left pointer and move both pointers to the right. This way, we ensure that all elements in the string are either 'a' or 'e'. The final output will be the modified string.
Here's the Python code for the solution:
```python
def remove_a_e(s):
    n = len(s)
    i = 0
    while i < n:
        if s[i] != 'a':
            j = i + 1
            while j < n and s[j] == 'e':
                j += 1
            if j == n:
                break
            else:
                s[i], s[j] = s[j], s[i]
        i += 1
    return ''.join(s)
```
