To solve this problem, we can use a simple approach of iterating over the input string and keeping track of the last occurrence of each character. We will initialize an empty dictionary to store the last occurrences of each character.
Then, for each character in the input string, we check if it is already present in the dictionary. If it is not, we add it to the dictionary with its index as the value. Otherwise, we update the value at that index to be one more than the previous value.
Finally, we iterate over the dictionary again and return all characters whose values are equal to their indices.
Here's the Python code to implement this solution:

```python
def longest_consecutive(nums):
    if not nums:
        return []
    
    last_dict = {}
    max_length = 0
    
    for num in nums:
        if num in last_dict:
            last_dict[num] += 1
        else:
            last_dict[num] = 1
        
        if last_dict[num] == num:
            max_length = max(max_length, last_dict[num])
        
    result = [num for num, count in last_dict.items() if count == max_length]
    return result
```
