Once we have counted all the characters, we iterate over the hash table again and for each key-value pair (character-frequency), we output the character followed by its frequency.

Here is the Python code that implements this solution:

def count_chars(s):
    freq = {}
    for char in s:
        if char in freq:
            freq[char] += 1
        else:
            freq[char] = 1

    result = []
    for char, freq in freq.items():
        result.append(char)
        result.append(str(freq))

    return " ".join(result)
