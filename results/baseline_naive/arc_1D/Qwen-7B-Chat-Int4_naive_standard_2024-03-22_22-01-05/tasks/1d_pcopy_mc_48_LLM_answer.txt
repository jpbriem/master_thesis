To solve this problem, we can use a hash set to keep track of the characters that have already been encountered in the input string. We iterate over each character in the input string and add it to the hash set if it has not been encountered before. If the character is already in the hash set, we skip it. Finally, we return an array containing all the unique characters in the input string.

Here's the Python code for this solution:

```python
def unique_chars(input_string):
    char_set = set()
    output_array = []
    for char in input_string:
        if char not in char_set:
            output_array.append(char)
            char_set.add(char)
    return output_array
```
