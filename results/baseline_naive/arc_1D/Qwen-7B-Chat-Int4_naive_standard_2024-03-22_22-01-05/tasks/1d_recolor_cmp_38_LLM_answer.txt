```python
def longest_common_subsequence(input):
    n = len(input)
    dp = [[0] * (n + 1) for _ in range(n + 1)]

    for i in range(1, n+1):
        for j in range(1, n+1):
            if input[i-1] == input[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    result = []
    i, j = n, 0

    while i > 0 and j > 0:
        if input[i-1] == input[j]:
            result.append(input[i-1])
            i -= 1
            j -= 1
        elif dp[i-1][j] > dp[i][j-1]:
            i -= 1
        else:
            j -= 1

    return ''.join(result[::-1])
```

Otherwise, we take the maximum value from either extending the longest common subsequence with the previous character from the other string or taking the maximum value from the two non-overlapping substrings.

Finally, we iterate over the table `dp` starting from the bottom right corner and building up the longest common subsequence by backtracking through the table.
