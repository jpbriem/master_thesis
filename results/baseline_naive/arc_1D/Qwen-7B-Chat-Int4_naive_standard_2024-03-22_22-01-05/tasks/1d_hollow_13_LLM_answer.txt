To solve this problem, we can use a hash table to keep track of the frequency of each character in the input string. We iterate through the string and update the count for each character in the hash table.
After counting all the characters, we iterate through the hash table again and replace any occurrences of a character with its corresponding frequency if it appears more than once.
Finally, we return the modified string.
Here's the Python code that implements this solution:

```
def remove_duplicates(input_string):
    # Create an empty hash table
    freq_table = {}

    # Iterate through the input string and count the frequency of each character
    for char in input_string:
        if char in freq_table:
            freq_table[char] += 1
        else:
            freq_table[char] = 1

    # Iterate through the hash table and replace any occurrences of a character with its corresponding frequency
    output_string = ''
    for char, freq in freq_table.items():
        while freq > 0:
            output_string += char
            freq -= 1

    return output_string
```
