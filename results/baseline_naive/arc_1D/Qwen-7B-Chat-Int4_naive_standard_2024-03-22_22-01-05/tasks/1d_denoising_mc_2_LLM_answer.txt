To solve this problem, we can use a dictionary to count the frequency of each character in the input string. We iterate over the characters and update the counts in the dictionary accordingly. Then, we create an output string by iterating over the characters again and appending them only if their count is greater than 0.

Here's the Python code for the solution:

```python
def group_chars(input_string):
    # Create an empty dictionary to store character frequencies
    char_counts = {}

    # Iterate over the characters and update the counts in the dictionary
    for char in input_string:
        if char in char_counts:
            char_counts[char] += 1
        else:
            char_counts[char] = 1

    # Create an empty list to store the output characters
    output_chars = []

    # Iterate over the characters again and append them only if their count is greater than 0
    for char in input_string:
        if char_counts[char] > 0:
            output_chars.append(char)

    return ''.join(output_chars)
```
