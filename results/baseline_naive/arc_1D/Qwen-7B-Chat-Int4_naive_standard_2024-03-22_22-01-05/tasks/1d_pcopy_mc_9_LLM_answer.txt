To solve this problem, we can use a dictionary to keep track of the frequency of each character in the input string. We iterate over the input string and update the count for each character in the dictionary. Then, we iterate over the dictionary again and return only the characters that have occurred at least twice.

Here is the Python code to implement this approach:

```
def remove_duplicates(input_string):
    char_count = {}
    output_string = []
    
    # Count the frequency of each character in the input string
    for char in input_string:
        if char not in char_count:
            char_count[char] = 0
        char_count[char] += 1
    
    # Iterate over the dictionary and add characters with frequency >= 2 to the output string
    for char, count in char_count.items():
        if count >= 2:
            output_string.append(char)
    
    return ''.join(output_string)
```
