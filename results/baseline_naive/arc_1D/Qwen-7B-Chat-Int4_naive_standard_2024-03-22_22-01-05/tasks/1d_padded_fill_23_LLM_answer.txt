Here's an example implementation in Python:

```python
def frequency_count(input_string):
    # Create an empty hash table
    freq_table = {}

    # Iterate through the input string
    for char in input_string:
        # Update the frequency count in the hash table
        if char in freq_table:
            freq_table[char] += 1
        else:
            freq_table[char] = 1

    # Return the updated frequency count dictionary
    return freq_table
```

We can then use this function to find all characters that appear exactly twice in the input string:

```python
def find_duplicates(input_string):
    # Get the frequency count dictionary from the previous step
    freq_table = frequency_count(input_string)

    # Initialize an empty list to store the duplicates
    duplicates = []

    # Iterate through the frequency count dictionary
    for char, count in freq_table.items():
        # If the count is greater than or equal to 2, add the character to the list of duplicates
        if count >= 2:
            duplicates.append(char)

    # Return the list of duplicates
    return duplicates
```
