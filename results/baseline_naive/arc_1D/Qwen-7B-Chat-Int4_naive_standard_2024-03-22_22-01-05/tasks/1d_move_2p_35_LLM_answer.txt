To solve this problem, we can use a hash table to keep track of the frequency of each character in the input string. We can iterate over the string and for each character, update its frequency count in the hash table. If the frequency count is equal to 1, we add it to the output string. Finally, we return the output string.

Here's the Python code that implements this approach:

```python
def unique_chars(s):
    char_freq = {}
    for char in s:
        if char not in char_freq:
            char_freq[char] = 0
        char_freq[char] += 1

    result = []
    for char, freq in char_freq.items():
        if freq == 1:
            result.append(char)

    return ''.join(result)
```
