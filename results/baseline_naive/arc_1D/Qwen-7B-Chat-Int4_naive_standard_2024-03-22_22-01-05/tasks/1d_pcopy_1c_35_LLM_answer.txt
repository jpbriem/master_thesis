If we encounter a new character that is different from the last seen character, we append it to the output array. If we encounter a character that matches the last seen character, we ignore it.

```python
def longest_common_subsequence(arr):
    if not arr:
        return []

    n = len(arr)
    dp = [[0] * (n + 1) for _ in range(n + 1)]

    # Build the dynamic programming table
    for i in range(1, n + 1):
        for j in range(1, n + 1):
            if arr[i - 1] == arr[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    # Backtrack to find the LCS
    lcs = []
    i, j = n, n
    while i > 0 and j > 0:
        if arr[i - 1] == arr[j - 1]:
            lcs.append(arr[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] >= dp[i][j - 1]:
            i -= 1
        else:
            j -= 1

    # Reverse the LCS to get the final result
    lcs.reverse()

    return lcs
```

This function has a time complexity of O(n^2), where n is the length of the input array. The space complexity is also O(n^2).
