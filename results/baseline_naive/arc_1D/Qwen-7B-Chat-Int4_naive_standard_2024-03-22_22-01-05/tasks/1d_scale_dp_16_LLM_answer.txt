To solve this problem, we can use a hash table to count the frequency of each character in the input string. We iterate through the string and for each character, we update its count in the hash table if it already exists or add it with a count of 1 if it doesn't exist.

Here's the Python code that implements this solution:

```python
def most_common_char(input_string):
    # Create an empty dictionary to store the character counts
    char_counts = {}

    # Iterate through the input string
    for char in input_string:
        # If the character is already in the dictionary, increment its count
        if char in char_counts:
            char_counts[char] += 1
        else:
            # Otherwise, add the character with a count of 1
            char_counts[char] = 1

    # Find the character with the highest count
    max_count = max(char_counts.values())
    most_common_chars = [char for char, count in char_counts.items() if count == max_count]

    return most_common_chars
```
