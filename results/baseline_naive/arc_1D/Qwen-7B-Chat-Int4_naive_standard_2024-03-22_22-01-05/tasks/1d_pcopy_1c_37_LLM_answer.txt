To solve this problem, we can use a dictionary to keep track of the frequency of each character in the input string. We iterate through the input string and update the frequency count for each character using the dictionary. Finally, we return the updated frequency count as the output.

Here is the Python code that implements this approach:

```python
def count_chars(input_string):
    # Create an empty dictionary to store frequency counts
    freq_counts = {}

    # Iterate through the input string
    for char in input_string:
        # If the character is already in the dictionary, increment its count
        if char in freq_counts:
            freq_counts[char] += 1
        else:
            # Otherwise, add the character to the dictionary with a count of 1
            freq_counts[char] = 1

    # Return the updated frequency count dictionary
    return freq_counts
```
