```python
def group_letters(input):
    # Create an empty list to store the groups of letters
    groups = []

    # Iterate over each letter in the input string
    for i, letter in enumerate(input):

        # Check if the current letter has already been seen before
        if letter not in groups[0]:
            # If it hasn't, create a new group and add the current letter to it
            groups.append([letter])

        else:
            # Otherwise, append the current letter to the existing group
            groups[0].append(letter)

    # Return the list of groups
    return groups
```
