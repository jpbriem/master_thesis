To solve this problem, we can use a hash table to keep track of the frequency of each character in the input string. We iterate through the string and update the frequency count for each character.
Then, we iterate through the string again and create a new string by replacing each character with its corresponding frequency count. If the frequency count is 0, we skip that character.
Here's the Python code to implement this solution:

```python
def group_chars(input_str):
    # Create an empty dictionary to store the frequency counts
    freq_counts = {}
    
    # Iterate through the input string and update the frequency count for each character
    for char in input_str:
        if char in freq_counts:
            freq_counts[char] += 1
        else:
            freq_counts[char] = 1
    
    # Create a new string by replacing each character with its corresponding frequency count
    output_str = ''
    for char in input_str:
        if freq_counts[char] > 0:
            output_str += str(freq_counts[char]) + char
    
    return output_str
```
