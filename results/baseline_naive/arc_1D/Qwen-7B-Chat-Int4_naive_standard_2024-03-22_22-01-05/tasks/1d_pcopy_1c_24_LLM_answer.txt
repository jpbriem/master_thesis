To solve this problem, we can use a dictionary to keep track of the frequency of each character. We iterate through the input string and for each character, we check if it is already in the dictionary. If it is not, we add it to the dictionary with a frequency count of 1. If it is already in the dictionary, we increment its frequency count by 1.

Here's the Python code to implement this solution:

```python
def process_string(input_str):
    freq_dict = {}
    output_str = []
    for char in input_str:
        if char in freq_dict:
            freq_dict[char] += 1
        else:
            freq_dict[char] = 1
    for char, freq in freq_dict.items():
        if freq == len(input_str) // 2:
            output_str.append(char)
    return output_str
```
