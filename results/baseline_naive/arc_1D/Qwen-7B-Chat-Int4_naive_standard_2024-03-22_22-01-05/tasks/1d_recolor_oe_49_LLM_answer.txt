To solve this problem, we can use a hash table to count the frequency of each character. We iterate through the input string and update the counts in the hash table accordingly. Then, we iterate through the hash table again and replace characters that appear more than twice with their corresponding two-letter code.

Here is the Python code for the solution:

```python
def compress_string(s):
    # Create an empty dictionary to store the counts
    freq = {}

    # Iterate through the string and update the counts in the dictionary
    for char in s:
        if char not in freq:
            freq[char] = 0
        freq[char] += 1

    # Create an empty list to store the compressed string
    compressed_str = []

    # Iterate through the dictionary and append characters to the compressed string
    for char, count in sorted(freq.items()):
        if count > 1:
            compressed_str.append(f'{char[0]}{char[1]}')

    return ''.join(compressed_str)
```
