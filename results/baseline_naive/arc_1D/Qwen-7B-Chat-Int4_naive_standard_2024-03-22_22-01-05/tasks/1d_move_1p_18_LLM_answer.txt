To solve this problem, we can use a simple approach where we iterate over the input array and keep track of two pointers, one at the beginning of the array and one at the end. We will compare the characters at these positions and move the pointer towards the middle if they are different. If they are equal, we will move both pointers towards the middle until they meet or cross each other.

Here's the Python code to implement this algorithm:

```python
def group_chars(input):
    # Initialize pointers at the beginning and end of the array
    left = 0
    right = len(input) - 1

    # Keep moving pointers towards the middle until they meet or cross each other
    while left < right:
        # Compare characters at current pointers
        char_left = input[left]
        char_right = input[right]

        # Move left pointer towards the middle if character is different
        if char_left != char_right:
            left += 1
        else:
            # Move right pointer towards the middle if characters are same
            right -= 1

    # Extract groups of characters from the array
    output = [input[i:i+min(left, right)+1] for i in range(min(left, right))]

    return output
```
