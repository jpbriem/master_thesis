To solve this problem, we can use a hash set to keep track of the characters that have already been seen. We iterate through each character in the input array and add it to the hash set if it has not already been seen. If a character is added to the hash set, we remove all occurrences of that character from the output array. Finally, we return the modified output array.
Here's an implementation in Python:

```python
def unique_chars(input):
    seen = set()
    output = []
    for char in input:
        if char not in seen:
            seen.add(char)
            output.append(char)
        else:
            for i in range(len(output)):
                if output[i] == char:
                    output.pop(i)
                    break
    return output
```
