```python
class Solution:
    def groupAnagrams(self, words: List[str]) -> List[List[str]]:
        anagram_dict = {}
        for word in words:
            sorted_word = ''.join(sorted(word))
            if sorted_word not in anagram_dict:
                anagram_dict[sorted_word] = [word]
            else:
                anagram_dict[sorted_word].append(word)
        
        result = []
        for key, value in anagram_dict.items():
            result.append(value)
            
        return result
```

Explanation:

  * We create a dictionary `anagram_dict` to store the sorted versions of each word as keys and the original words as values.
  * For each word in `words`, we sort its characters and use this sorted version as the key in our dictionary. If the key already exists in the dictionary, we append the current word to the corresponding list of values (since it is also an anagram). Otherwise, we add the current word as a new key with a single value.
  * Finally, we iterate over the dictionary and collect all the lists of values into one final result.

This solution has a time complexity of O(n log n), where n is the length of the input list `words`. This is because sorting each word takes O(n) time, and iterating over the dictionary takes O(m log m), where m is the number of unique sorted words. Since there are at most n unique sorted words, the total time complexity is O(n log n + m log m) = O(n log n).
