We iterate over the string and update the count for each character in the dictionary. Here is an example implementation in Python:

```python
def remove_consecutive_duplicates(input_string):
    # Create a dictionary to store the frequency of each character
    char_count = {}
    
    # Iterate over the input string and update the count for each character
    for char in input_string:
        if char in char_count:
            char_count[char] += 1
        else:
            char_count[char] = 1
    
    # Iterate over the dictionary and return the characters with a count greater than or equal to 4
    output_string = []
    for char, count in char_count.items():
        if count >= 4:
            output_string.append(char)
    
    return output_string
```
