To solve this problem, we can use a hash set to keep track of the characters that have already been seen. We iterate through each character in the input string and check if it has already been seen. If not, we add it to the hash set and append it to the output string. Finally, we return the output string.

Here is an implementation of the solution in Python:

```
def group_chars(input_string):
    output_string = []
    char_set = set()
    
    for char in input_string:
        if char not in char_set:
            char_set.add(char)
            output_string.append(char)
            
    return ''.join(output_string)
```
