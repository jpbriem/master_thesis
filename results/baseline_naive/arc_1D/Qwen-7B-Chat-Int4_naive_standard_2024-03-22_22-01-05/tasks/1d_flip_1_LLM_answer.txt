```python
def longest_common_subsequence(input):
    if not input:
        return []
    
    n = len(input)
    dp = [[0] * (n + 1) for _ in range(n + 1)]
    
    for i in range(1, n + 1):
        for j in range(1, n + 1):
            if input[i - 1] == input[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    
    result = []
    i, j = n, 0
    while i > 0 and j > 0:
        if input[i - 1] == input[j]:
            result.append(input[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] >= dp[i][j - 1]:
            i -= 1
        else:
            j -= 1
    
    return result[::-1]
```

The time complexity of this algorithm is O(n^2), where n is the length of the input. The space complexity is also O(n^2).
