To solve this problem, we can use a hash table to count the frequency of each character in the input string. Then, we can iterate through the hash table and return only those characters that have a frequency greater than or equal to 5.

Here is the Python code to implement this solution:

```
def group_chars(input):
    char_count = {}
    for char in input:
        if char not in char_count:
            char_count[char] = 0
        char_count[char] += 1
    
    output = []
    for char, count in char_count.items():
        if count >= 5:
            output.append(char)
    
    return output
```
