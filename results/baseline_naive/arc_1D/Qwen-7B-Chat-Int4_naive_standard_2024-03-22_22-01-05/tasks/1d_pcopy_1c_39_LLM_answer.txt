To solve this problem, we can use a hash table to keep track of the frequency of each character in the input string. We iterate through the input string and update the frequency count for each character in the hash table. If a character appears more than once in a row, we append it to the output string.

Here's the Python code to implement this approach:

```python
def longest_consecutive_substring(s):
    # Initialize an empty dictionary to store the frequency of characters
    freq = {}
    
    # Initialize an empty list to store the consecutive subsequence
    consecutive_subseq = []
    
    # Iterate through the input string
    i = 0
    while i < len(s):
        # Check if the current character has appeared before
        if s[i] in freq:
            # Update the frequency count for the current character
            freq[s[i]] += 1
        else:
            # Add the current character to the frequency count
            freq[s[i]] = 1
        
        # Check if the current character is repeated consecutively
        j = i + 1
        while j < len(s) and s[j] == s[i]:
            # Append the current character to the consecutive subsequence
            consecutive_subseq.append(s[i])
            
            # Increment the index of the next character
            j += 1
            
        # Move on to the next character
        i = j
    
    # Return the length of the consecutive subsequence
    return len(consecutive_subseq)
```
