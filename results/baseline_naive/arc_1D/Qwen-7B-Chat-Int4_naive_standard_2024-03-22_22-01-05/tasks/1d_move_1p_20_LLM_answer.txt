To solve this problem, we can use a dictionary to count the frequency of each character in the input string. Then, we can iterate through the dictionary and add characters with a frequency greater than or equal to 2 to the output list.
Here is an implementation of this algorithm in Python:

```
def remove_duplicates(input_string):
    # Create a dictionary to count the frequency of each character
    char_count = {}
    for char in input_string:
        if char in char_count:
            char_count[char] += 1
        else:
            char_count[char] = 1
    
    # Iterate through the dictionary and add characters with a frequency greater than or equal to 2 to the output list
    output_list = []
    for char, count in char_count.items():
        if count >= 2:
            output_list.append(char)
    
    return output_list
```
