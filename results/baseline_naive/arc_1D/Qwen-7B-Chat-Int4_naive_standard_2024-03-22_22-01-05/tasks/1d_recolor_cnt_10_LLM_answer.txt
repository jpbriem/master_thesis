To solve this problem, we can use a dictionary to store the frequency of each character in the input string. We can iterate over the characters in the string and update the count for each character in the dictionary. Then, we can iterate over the dictionary and return the characters that have the highest frequency.
Here's the Python code to implement this approach:

```
def group_chars(input_str):
    # Create an empty dictionary to store the frequency of each character
    freq_dict = {}

    # Iterate over the characters in the string
    for char in input_str:
        # Update the count for the current character in the dictionary
        if char in freq_dict:
            freq_dict[char] += 1
        else:
            freq_dict[char] = 1

    # Sort the keys of the dictionary by their values (frequency)
    sorted_keys = sorted(freq_dict.keys(), key=lambda x: freq_dict[x], reverse=True)

    # Initialize an empty list to store the grouped characters
    output_list = []

    # Iterate over the sorted keys and append the corresponding characters to the output list
    for char in sorted_keys:
        output_list.append(char)

    return output_list
```
