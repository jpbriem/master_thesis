To solve this problem, we can use a dictionary to keep track of the frequency of each character in the input string. We iterate over the string and for each character, if it is not already in the dictionary, we add it with a count of 1. If it is already in the dictionary, we increment its count.

After iterating over all characters, we check if the count of any character is greater than or equal to 2. If so, we append that character to the output list. Otherwise, we append an empty string.

Here's the Python code to implement this approach:

```python
def longest_common_subsequence(input):
    char_freq = {}
    for char in input:
        if char not in char_freq:
            char_freq[char] = 0
        char_freq[char] += 1

    output = []
    for char in input:
        if char_freq[char] >= 2:
            output.append(char)

    return output
```
