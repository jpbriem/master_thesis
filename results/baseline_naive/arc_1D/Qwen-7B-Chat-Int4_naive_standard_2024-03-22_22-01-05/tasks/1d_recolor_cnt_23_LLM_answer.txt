To solve this problem, we can use a sliding window approach. We will maintain two pointers, one at the beginning of the array and another at the end of the array. At each step, we will check if the character at the current pointer is different from the previous characters in the window. If it is different, we will update the window by adding the new character to the window and moving both pointers forward. Otherwise, we will remove the first character from the window and move both pointers forward.
Here's the Python code for solving this problem:

```python
def longest_substring(input):
    n = len(input)
    if n == 0:
        return []
    
    start = 0
    max_length = 0
    
    # Create an empty string to store the result
    result = ""
    
    # Initialize two pointers
    left = right = start
    
    while right < n:
        
        # Check if the character at the right pointer is different from the previous characters in the window
        if input[right] != input[left]:
            result += input[left]
            
            # Move the left pointer forward
            left += 1
            
            # Update the maximum length if necessary
            if len(result) > max_length:
                max_length = len(result)
                
        else:
            # Remove the first character from the window and move the right pointer forward
            result = result[:-1]
            right += 1
            
    return result[:max_length]
```
