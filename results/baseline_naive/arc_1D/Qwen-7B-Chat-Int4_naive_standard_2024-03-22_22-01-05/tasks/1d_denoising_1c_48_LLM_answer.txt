To solve this problem, we can use a set to keep track of the unique characters in each string. We can iterate through each string and add each character to the corresponding set if it is not already present. After iterating through all strings, we can convert the sets back into lists and concatenate them.

Here's the Python code for this solution:

```python
def group_strings(strings):
    # Initialize an empty dictionary to store the groups of characters
    groups = {}

    # Iterate through each string
    for string in strings:
        # Convert the string to lowercase
        string = string.lower()

        # Create an empty list to store the characters in the current group
        group = []

        # Iterate through each character in the string
        for char in string:
            # If the character is already in the current group, skip it
            if char in group:
                continue

            # Add the character to the current group
            group.append(char)

        # Check if the current group has at least two characters
        if len(group) >= 2:
            # If so, create a new group with the characters in the current group
            key = ''.join(sorted(group))
            if key not in groups:
                groups[key] = []
            groups[key].append(string)

    # Convert the groups dictionary into a list of lists
    result = [list(g) for g in groups.values()]

    return result
```
