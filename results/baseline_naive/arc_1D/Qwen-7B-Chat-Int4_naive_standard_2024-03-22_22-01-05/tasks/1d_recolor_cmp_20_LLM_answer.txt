To solve this problem, we can use a hash table to count the frequency of each character in the input string. We can iterate over the input string and update the counts in the hash table accordingly.
Here's an example implementation in Python:

```python
def group_chars(input_str):
    # Create an empty dictionary to store the character frequencies
    char_freq = {}

    # Iterate over the input string and update the counts in the dictionary
    for char in input_str:
        if char in char_freq:
            char_freq[char] += 1
        else:
            char_freq[char] = 1

    # Initialize two empty lists to store the groups of characters
    group_a = []
    group_b = []

    # Iterate over the characters and add them to the appropriate group
    for char in sorted(char_freq.keys()):
        if char_freq[char] % 2 == 0:
            group_a.append(char)
        else:
            group_b.append(char)

    return ''.join(group_a), ''.join(group_b)
```

This solution has a time complexity of O(n log n) because it iterates over the input string twice (once to compute the character frequencies and once to group the characters). The space complexity is also O(n) because it stores the character frequencies in a dictionary.
