```python
def longest_common_subsequence(input):
    n = len(input)
    dp = [[0] * (n + 1) for _ in range(n + 1)]

    for i, x in enumerate(input):
        for j, y in enumerate(input):
            if x == y:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    result = []
    i, j = n, 0
    while i > 0 and j > 0:
        if input[i-1] == input[j]:
            result.append(input[i-1])
            i -= 1
            j -= 1
        elif dp[i-1][j] >= dp[i][j-1]:
            i -= 1
        else:
            j -= 1

    return ''.join(result[::-1])
```

Once the matrix is built, we can iterate through it backwards to find the longest common subsequence.
