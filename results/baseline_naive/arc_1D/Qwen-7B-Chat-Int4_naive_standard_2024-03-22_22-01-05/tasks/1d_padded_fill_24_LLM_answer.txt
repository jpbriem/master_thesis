```python
class Solution:
    def groupAnagrams(self, words: List[str]) -> List[List[str]]:
        d = {}
        
        for word in words:
            key = ''.join(sorted(word))
            if key not in d:
                d[key] = []
            d[key].append(word)
            
        return [d[k] for k in sorted(d.keys())]
```

Explanation:

We can use a dictionary to store the anagram groups. We iterate over each word and convert it into a string of unique characters using the `sorted` function. This gives us a lexicographically sorted version of the word that we can use as a key in our dictionary.

If the key is already present in the dictionary, we append the word to its corresponding list. If the key is not present, we create a new empty list for this key and append the word to it.

Finally, we convert the dictionary keys into a list and sort them lexicographically. Then, we return a list containing the lists of anagrams grouped by their key.
