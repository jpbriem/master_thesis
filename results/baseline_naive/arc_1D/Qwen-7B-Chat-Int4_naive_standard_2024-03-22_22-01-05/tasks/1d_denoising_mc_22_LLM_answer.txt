To solve this problem, we can use a set to keep track of the unique characters in the input string. We iterate through each character and add it to the set if it is not already present. Then we check if the length of the set is equal to the length of the input string, which means all characters have been accounted for.

Here's an implementation of the solution:

```python
def find_unique_chars(input):
    # Create a set to store unique characters
    unique_chars = set()
    
    # Iterate through each character in the input string
    for char in input:
        # Add the character to the set if it is not already present
        if char not in unique_chars:
            unique_chars.add(char)
    
    # Check if the length of the set is equal to the length of the input string
    return len(unique_chars) == len(input)
```
