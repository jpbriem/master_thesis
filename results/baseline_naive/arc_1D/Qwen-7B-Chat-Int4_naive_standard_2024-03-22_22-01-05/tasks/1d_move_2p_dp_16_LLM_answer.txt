To solve this problem, we can use a hash set to keep track of the characters that have been seen so far. We iterate through the input array and for each character, if it has not been seen before, we add it to the hash set. If it has already been seen, we ignore it.
Here is the Python code to implement this solution:

```python
def remove_duplicates(input):
    seen = set()
    output = []
    for char in input:
        if char not in seen:
            seen.add(char)
            output.append(char)
    return output
```
