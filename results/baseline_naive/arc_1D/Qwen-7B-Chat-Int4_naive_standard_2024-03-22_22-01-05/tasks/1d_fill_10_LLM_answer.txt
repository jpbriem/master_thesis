```
def longest_common_subsequence(input):
    result = []
    seen = set()
    for char in input:
        if char not in seen:
            seen.add(char)
            result.append(char)
    return ''.join(result)
```

The outer loop iterates through each character in the input string. Inside the loop, we check if the current character is already in the `seen` set. If it is not, we add it to the set and append it to the `result` list. Finally, we join the elements of the `result` list into a single string using the `join()` method and return the resulting string.
