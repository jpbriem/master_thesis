To solve this problem, we can use a set to keep track of the characters that have been seen so far. We iterate through the input array and for each character, if it has already been seen before, we add it to the output array. If not, we add it to both the output array and the set.

Here is an implementation in Python:

```python
def unique_chars(input):
    output = []
    seen = set()

    for char in input:
        if char not in seen:
            output.append(char)
            seen.add(char)

    return output
```
