```python
def longest_common_subsequence(input):
    n = len(input)
    dp = [[0] * (n + 1) for _ in range(n + 1)]

    # Initialize the first row and column with 0's
    for i in range(n + 1):
        dp[i][i] = 0

    # Fill in the rest of the table using dynamic programming
    for i in range(1, n+1):
        for j in range(i, n+1):
            if input[j-1] == input[i-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    # The length of the longest common subsequence is the maximum value in the bottom-right corner
    return dp[-1][-1]
```

We initialize the first row and column with 0's and then fill in the rest of the table by comparing characters at corresponding positions in the input. Finally, we return the maximum value in the bottom-right corner of the matrix, which gives us the length of the longest common subsequence.
