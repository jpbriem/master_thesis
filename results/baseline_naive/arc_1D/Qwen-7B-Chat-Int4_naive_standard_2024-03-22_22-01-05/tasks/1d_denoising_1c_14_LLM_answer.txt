To solve this problem, we can use a hash set to keep track of the characters that have already been encountered in the input string. We iterate over the input string and for each character, if it has not been encountered before, we add it to the hash set and append it to the output string. If it has been encountered before, we do nothing.
Here's the Python code to implement this approach:

```python
def longest_common_subsequence(input):
    seen = set()
    result = []
    for char in input:
        if char not in seen:
            seen.add(char)
            result.append(char)
        else:
            result[-1] += char
    return ''.join(result)
```
