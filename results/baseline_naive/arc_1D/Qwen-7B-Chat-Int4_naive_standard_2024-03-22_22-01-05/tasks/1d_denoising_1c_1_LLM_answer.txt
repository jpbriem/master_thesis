To solve this problem, we can use a simple approach called "bucket sort". In bucket sort, we divide the input array into several buckets and then sort each bucket individually. Finally, we concatenate all the sorted buckets together to get the final output.
Here is an implementation of the algorithm in Python:

```python
def bucket_sort(arr):
    # Find the maximum element in the array
    max_val = max(arr)
    
    # Create a list of buckets with length equal to (max_val + 1) / bucket_size
    buckets = [[] for _ in range((max_val + 1) // bucket_size)]
    
    # Place each element in its corresponding bucket based on its value
    for elem in arr:
        index = (elem - min(arr)) // bucket_size
        buckets[index].append(elem)
        
    # Concatenate all the buckets together
    return [item for bucket in buckets for item in bucket]
```
