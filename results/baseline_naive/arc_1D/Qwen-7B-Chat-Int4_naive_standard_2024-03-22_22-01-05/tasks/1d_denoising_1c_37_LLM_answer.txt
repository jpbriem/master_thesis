```python
def group_chars(input):
    # Initialize an empty list to store the groups of characters
    groups = []
    
    # Iterate through each character in the input string
    for char in input:
        # If a new group has been formed, append it to the list of groups
        if len(groups) == 0 or groups[-1][0] != char:
            groups.append([char])
        
        # Otherwise, add the current character to the last group
        else:
            groups[-1].append(char)
            
    # Return the list of groups
    return groups
```

This function takes an input string as its argument and returns a list of lists, where each inner list represents a group of consecutive characters. The groups are sorted by their first element (the character that starts them).
