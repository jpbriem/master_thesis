Here is an implementation of the solution in Python:

```python
def group_unique_chars(strings):
    # Initialize empty sets for each string
    sets = [set() for _ in range(len(strings))]

    # Iterate through each character in each string
    for i in range(len(strings)):
        for char in strings[i]:
            # Add character to corresponding set if not already there
            sets[i].add(char)

    # Convert each set back into a list and join them with commas
    output = ', '.join([''.join(sorted(set)) for set in sets])

    return output
```
