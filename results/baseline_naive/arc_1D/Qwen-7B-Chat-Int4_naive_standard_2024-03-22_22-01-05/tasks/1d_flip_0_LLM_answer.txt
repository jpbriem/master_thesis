To solve this problem, we can use a hash set to keep track of the characters that have been seen so far. We iterate through the input string and for each character, if it has not been seen before, we add it to the hash set. If it has been seen before, we do nothing.

At the end, we return an array containing all the unique characters in the order they were first encountered.

Here is the Python code that implements this solution:

```python
def remove_duplicates(input):
    seen = set()
    result = []
    for char in input:
        if char not in seen:
            seen.add(char)
            result.append(char)
    return result

# Test cases
```
