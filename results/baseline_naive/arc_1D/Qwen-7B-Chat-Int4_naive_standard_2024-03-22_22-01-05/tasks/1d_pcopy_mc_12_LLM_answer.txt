To solve this problem, we can use a simple approach. We will iterate through the input string and keep track of the frequency of each character using a dictionary. Then, we will iterate through the dictionary again to find all characters that appear more than once.
Here's an implementation in Python:

```python
def count_unique_chars(input_str):
    # Initialize an empty dictionary to store the frequency of each character
    freq_dict = {}

    # Iterate through the input string and update the frequency dictionary
    for char in input_str:
        if char in freq_dict:
            freq_dict[char] += 1
        else:
            freq_dict[char] = 1

    # Initialize an empty list to store unique characters
    unique_chars = []

    # Iterate through the frequency dictionary and add characters with frequency > 1 to the list
    for char, freq in freq_dict.items():
        if freq > 1:
            unique_chars.append(char)

    return unique_chars
```
