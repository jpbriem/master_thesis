To solve this problem, we can use a simple approach where we iterate through the input array and keep track of the last character that was encountered. If the current character is different from the last character, we append it to the output array. Otherwise, if the current character matches the last character, we move on to the next character.

Here's the Python code for the solution:

```python
def longest_common_subsequence(input):
    # Initialize an empty list to store the result
    result = []

    # Keep track of the last character encountered
    last_char = None

    # Iterate through the input array
    for char in input:
        # If the current character is different from the last character,
        # append it to the result
        if char != last_char:
            result.append(char)

        # Update the last character
        last_char = char

    return result
```
