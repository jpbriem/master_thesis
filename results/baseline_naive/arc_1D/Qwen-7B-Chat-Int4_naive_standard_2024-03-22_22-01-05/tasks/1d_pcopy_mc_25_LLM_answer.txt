To solve this problem, we can use a hash set to keep track of the unique characters in each string. We iterate through the input strings and add each character to the corresponding hash set if it is not already present. Finally, we create an output array that contains only the unique characters from all the input strings.
Here's the Python code to implement this solution:

```
def find_unique_chars(input_strings):
    # Create an empty dictionary to store the hash sets for each input string
    hash_sets = {}
    
    # Iterate through the input strings
    for i, string in enumerate(input_strings):
        # Initialize an empty list to store the unique characters in this string
        unique_chars = []
        
        # Iterate through the characters in this string
        for char in string:
            # Check if this character is already in the hash set for this string
            if char not in hash_sets.get(string, []):
                # If not, add it to the hash set and the list of unique characters
                hash_sets[string].append(char)
                unique_chars.append(char)
                
        # Add the list of unique characters to the output array
        output_strings.append(unique_chars)
    
    return output_strings
```
