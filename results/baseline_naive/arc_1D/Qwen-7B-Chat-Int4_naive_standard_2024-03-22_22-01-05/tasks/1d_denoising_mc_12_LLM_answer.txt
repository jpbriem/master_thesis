To solve this problem, we can use a hash table to count the frequency of each character in the input string. Then, we iterate through the input string and return only the characters that appear more than once.

```python
def remove_duplicates(input_string):
    # Create a dictionary to store the frequency of each character
    freq_dict = {}
    
    # Iterate through the input string and count the frequency of each character
    for char in input_string:
        if char in freq_dict:
            freq_dict[char] += 1
        else:
            freq_dict[char] = 1
    
    # Initialize an empty output string
    output_string = ""
    
    # Iterate through the input string again and add only the characters with frequency greater than one
    for char in input_string:
        if freq_dict[char] > 1:
            output_string += char
    
    return output_string
```
