To solve this problem, we can use a hash table to count the frequency of each character in the input string. We will iterate through the input string and update the count for each character in the hash table.

Here's the Python code that implements this solution:

```python
def remove_duplicates(input_string):
    # Create an empty hash table
    char_count = {}
    
    # Iterate through the input string
    for char in input_string:
        # If the character is already in the hash table, increment its count
        if char in char_count:
            char_count[char] += 1
        else:
            # Otherwise, add the character to the hash table with a count of 1
            char_count[char] = 1
    
    # Initialize an empty output string
    output_string = ""
    
    # Iterate through the hash table
    for char, count in char_count.items():
        # Add the character to the output string as many times as its count
        for _ in range(count):
            output_string += char
    
    return output_string
```
