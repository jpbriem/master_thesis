We iterate through the input string and for each character, we check if it is already in the hash table. If it is not, we add it with a frequency of 1. If it is already in the hash table, we increment its frequency by 1.

Once we have counted the frequency of each character, we iterate through the input string again and remove all characters that occur more than once. To do this, we create an empty list `output` and then iterate through the input string. For each character, we check if its frequency is greater than or equal to 2. If it is, we append it to the output list only if it has occurred at least twice.

Finally, we return the output list.

Here's the Python code that implements this solution:

```python
def count_chars(input_str):
    freq = {}
    for char in input_str:
        if char in freq:
            freq[char] += 1
        else:
            freq[char] = 1

    output = []
    for char in input_str:
        if freq[char] >= 2:
            if char not in output:
                output.append(char)

    return output
```
