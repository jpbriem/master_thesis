To solve this problem, we can use a greedy approach. We will iterate through the input array and for each character, we will check if it is equal to any of its adjacent characters. If it is, we will replace it with an "e" character. Otherwise, we will keep the original character.
Here's the Python code that implements this algorithm:

```python
def find_unique_chars(input):
    output = []
    i = 0

    while i < len(input):
        # Check if current character is equal to any of its adjacent characters
        j = i - 1
        k = i + 1

        if j >= 0 and input[i] == input[j]:
            continue

        if k < len(input) and input[i] == input[k]:
            continue

        output.append(input[i])

        # Move on to the next character
        i += 1

    return output
```
