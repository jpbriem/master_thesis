To solve this problem, we can use a set to keep track of the unique characters in the input string. We iterate through each character in the string and add it to the set if it's not already there. Finally, we convert the set back into a list and return it.

Here is the Python code that implements this solution:

```python
def remove_duplicates(input_string):
    unique_chars = set()
    result = []
    for char in input_string:
        if char not in unique_chars:
            unique_chars.add(char)
            result.append(char)
    return ''.join(result)
```
