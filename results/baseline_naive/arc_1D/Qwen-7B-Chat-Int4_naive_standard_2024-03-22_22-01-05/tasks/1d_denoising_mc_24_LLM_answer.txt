Here's the Python code that implements this algorithm:

```
def longest_common_subsequence(input):
    output = ''
    prev_char = None
    count = 0

    for char in input:
        if char != prev_char:
            if char.isalpha():
                count += 1
            else:
                count = 0
            output += char * count
        prev_char = char
    return output
```

In this code, `prev_char` keeps track of the previous character seen in the input string. `count` keeps track of the number of times the current character has been seen in a row. The `if char.isalpha()` condition checks if the current character is a letter or not. If it is a letter, we increment the count by 1. Otherwise, we reset the count back to 0. Finally, we append the current character multiplied by its count to the output string.
