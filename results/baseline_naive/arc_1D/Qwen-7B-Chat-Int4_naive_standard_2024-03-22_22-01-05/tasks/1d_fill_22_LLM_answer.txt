To solve this problem, we can use a hash set to keep track of the characters that have already been seen. We iterate through the input string and for each character, if it has not been seen before, we add it to the hash set and append it to the output string. If the character has already been seen, we do nothing.
Here is an implementation in Python:

```python
def unique_chars(s):
    seen = set()
    result = []
    for char in s:
        if char not in seen:
            seen.add(char)
            result.append(char)
    return result
```
