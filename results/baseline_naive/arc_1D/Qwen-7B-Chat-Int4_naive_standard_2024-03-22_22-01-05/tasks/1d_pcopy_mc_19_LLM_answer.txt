To solve this problem, we can use a set to keep track of the unique characters in each word. We iterate through the input array and for each word, add its characters to the corresponding set. If a character is already present in that set, it means that it appears multiple times in that word, so we remove it from the output list.

Here's an implementation in Python:

```python
def longest_common_subsequence(input):
    n = len(input)
    dp = [[0] * (n + 1) for _ in range(n + 1)]
    
    for i in range(1, n+1):
        for j in range(1, n+1):
            if input[i-1] == input[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
                
    result = []
    i, j = n, n
    while i > 0 or j > 0:
        if i > 0 and dp[i-1][j] >= dp[i][j]:
            result.append(input[i-1])
            i -= 1
        elif j > 0 and dp[i][j-1] >= dp[i][j]:
            result.append(input[j-1])
            j -= 1
        
    return ''.join(result[::-1])
```

This solution has a time complexity of O(n^2), where n is the length of the input array. It also has a space complexity of O(n), which is required to store the dp table.
