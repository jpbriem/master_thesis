To solve this problem, we can use a hash set to keep track of the characters that have been seen so far. We iterate through the input string and for each character, if it has already been seen before, we remove it from the output string. Otherwise, we add it to the output string.
Here is the Python code to implement this solution:

```
def unique_chars(input):
    output = []
    char_set = set()
    for char in input:
        if char not in char_set:
            output.append(char)
            char_set.add(char)
    return output
```
