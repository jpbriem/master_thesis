To solve this problem, we can iterate through the input array and keep track of the current character. If the current character is equal to the previous character, we add it to the output array. Otherwise, we reset the output array to an empty string.
Here's a Python function that implements this approach:

```python
def longest_common_subsequence(arr):
    n = len(arr)
    if n == 0 or n == 1:
        return arr

    max_len = 1
    prev_char = arr[0]
    output = [arr[0]]

    for i in range(1, n):
        curr_char = arr[i]

        # If the current character is different from the previous character,
        # update the maximum length and start building the output array
        if curr_char != prev_char:
            max_len += 1
            output.clear()

        # Add the current character to the output array
        output.append(curr_char)

        # Update the previous character
        prev_char = curr_char

    return output[:max_len]
```
