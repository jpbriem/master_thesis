Here is an implementation of the solution in Python:

```python
def unique_chars(input_string):
    # Create an empty hash set to store the characters that have been seen
    seen = set()

    # Iterate through the input string
    for i, char in enumerate(input_string):
        # Add the character to the hash set if it hasn't been seen before
        if char not in seen:
            seen.add(char)
            output_string.append(i)

    # Return the output string
    return ''.join(map(str, output_string))
```
