To solve this problem, we can iterate over the input array and keep track of two variables: `prev` which stores the previous character seen in the array and `curr` which stores the current character being processed. If `curr` is equal to `prev`, then we add it to the output array. Otherwise, we reset both `prev` and `curr`. Here's the Python code that implements this approach:

```python
def remove_duplicates(input):
    prev = None
    output = []
    for char in input:
        if char != prev:
            prev = char
            output.append(char)
    return output
```
