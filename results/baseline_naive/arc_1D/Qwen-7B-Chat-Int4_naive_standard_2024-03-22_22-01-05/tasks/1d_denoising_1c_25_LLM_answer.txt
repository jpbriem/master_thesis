To solve this problem, we can iterate through the input list and keep track of the last occurrence of each character. If a character is encountered again before its previous occurrence, it means that it has been repeated in the string, so we append it to the output list.

Here's an implementation in Python:

```python
def longest_repeated_subsequence(input):
    output = []
    last_occurrence = {}
    
    for char in input:
        if char not in last_occurrence:
            last_occurrence[char] = 0
        last_occurrence[char] += 1
    
    for i in range(len(input)):
        for j in range(i+1, len(input)+1):
            if input[i:j] in last_occurrence:
                last_occurrence[input[i:j]] += 1
                
    for char, count in last_occurrence.items():
        while count > 0:
            output.append(char)
            count -= 1
            
    return output[::-1]
```

For each pair of indices `(i, j)`, it checks if the substring between them appears in the `last_occurrence` dictionary. If it does, it increments the count of occurrences of that substring.

After iterating through all pairs of indices, it iterates through the `last_occurrence` dictionary again and appends the characters with more than one occurrence to the `output` list.

Finally, it returns the `output` list reversed to get the longest repeated subsequence in descending order of length.
