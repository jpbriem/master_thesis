Once we have counted all the characters, we iterate over the hash table again and for each key-value pair, if the value is greater than or equal to 2, we append it to the output string. This way, we ensure that only unique characters are added to the output string, and we also avoid adding duplicate characters.
Here's an implementation in Python:

```python
def group_chars(input_string):
    char_counts = {}
    output_string = []

    # Count the frequency of each character
    for char in input_string:
        if char in char_counts:
            char_counts[char] += 1
        else:
            char_counts[char] = 1

    # Append characters to the output string if they appear more than once
    for char, count in char_counts.items():
        if count >= 2:
            output_string.append(char)

    return output_string
```
