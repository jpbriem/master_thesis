```python
def group_chars(input):
    # Create an empty list to store the output
    output = []
    
    # Loop through each character in the input string
    for char in input:
        # If the current character has not been seen before, append it to the output and start a new group
        if char not in output:
            output.append(char)
        
        # Otherwise, append the character to the current group
        else:
            output[-1] += char
    
    return output
```

This function takes an input string as its argument and returns a list of characters that are grouped together. It works by iterating over each character in the input string and checking whether it has been seen before. If the character has not been seen before, it is added to the output list and a new group begins. If the character has already been seen, it is appended to the current group.
