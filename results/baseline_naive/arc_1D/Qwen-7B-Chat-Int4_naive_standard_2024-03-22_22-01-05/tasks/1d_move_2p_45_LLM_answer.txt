To solve this problem, we can use a hash set to keep track of the characters that have been seen so far. We iterate over the input string and for each character, if it has not been seen before, we add it to the hash set. If it has been seen before, we ignore it.

Here's an implementation in Python:

```python
def remove_duplicates(input):
    seen = set()
    result = []
    for char in input:
        if char not in seen:
            seen.add(char)
            result.append(char)
    return ''.join(result)
```

This solution has a time complexity of O(n), where n is the length of the input string. This is because we only need to iterate over the input once to check if each character has been seen before or not. The space complexity is also O(n) since we are storing all the unique characters in the hash set.
