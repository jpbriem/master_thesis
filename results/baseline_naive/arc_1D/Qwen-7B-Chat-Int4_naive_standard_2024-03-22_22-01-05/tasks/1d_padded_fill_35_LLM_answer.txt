To solve this problem, we can use a hash table to count the frequency of each character in the input string. We iterate over the string and for each character, we increment its corresponding count in the hash table. If the count is greater than 0, it means that the character has appeared multiple times in the string, so we append it to the output list.

Here's the Python code to implement this solution:

```python
def group_chars(s):
    counts = {}
    result = []
    for char in s:
        if char not in counts:
            counts[char] = 0
        counts[char] += 1
        if counts[char] > 0:
            result.append(char)
    return result
```
