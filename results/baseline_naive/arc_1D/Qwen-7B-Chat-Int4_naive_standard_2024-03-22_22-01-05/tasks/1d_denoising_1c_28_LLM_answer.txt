To solve this problem, we can use a hash set to keep track of the characters that have already been seen in the input string. We iterate over the input string and add each character to the hash set if it has not already been added. If a character is encountered more than once, we remove all occurrences except for the first one.

Here's an implementation in Python:

```python
def find_unique_chars(input_string):
    unique_chars = set()
    result = []
    for char in input_string:
        if char not in unique_chars:
            unique_chars.add(char)
            result.append(char)
    return ''.join(result)
```
