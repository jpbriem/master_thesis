To solve this problem, we can use a hash set to keep track of the unique characters in each string. We iterate through each string and for each character, if it is not already in the hash set for that string, we add it to the set and update the output string by appending the character. If the character is already in the set for that string, we do nothing.
Here's the Python code to implement this solution:

```python
def find_unique_chars(strings):
    result = []
    for s in strings:
        seen = set()
        new_string = ""
        for c in s:
            if c not in seen:
                seen.add(c)
                new_string += c
        result.append(new_string)
    return result
```
