To solve this problem, we can use a hash set to keep track of the characters that have appeared so far. We iterate through the input string and add each character to the hash set if it hasn't already been seen. If a character has already been seen, we remove it from the hash set and append it to the output string.

Here is the Python code for the solution:

```python
def find_unique_chars(s):
    # Initialize an empty hash set
    seen = set()

    # Iterate through the input string
    for char in s:
        # Add the character to the hash set if it hasn't already been seen
        if char not in seen:
            seen.add(char)

    # Create an empty list to store the unique characters
    unique_chars = []

    # Iterate through the hash set and append each character to the list
    for char in seen:
        unique_chars.append(char)

    return ''.join(unique_chars)
```
