To solve this problem, we can iterate through the input array and keep track of the current character in a variable. If the current character is equal to the previous character, we append it to the output array. Otherwise, we reset the current character and add the previous character and its count to the output array.

```python
def longest_common_subsequence(arr):
    n = len(arr)
    if not arr or n == 0:
        return []
    
    # Initialize an empty list to store the result
    result = []
    
    # Iterate through the input array
    current_char = None
    prev_count = 0
    
    for i in range(n):
        # Check if the current character is different from the previous one
        if arr[i] != current_char:
            # Add the previous character and its count to the result
            result.append((current_char, prev_count))
            
            # Reset the current character and increment the count
            current_char = arr[i]
            prev_count = 1
        else:
            # Increment the count of the previous character
            prev_count += 1
            
    # Add the last character and its count to the result
    result.append((current_char, prev_count))
    
    return result
```
