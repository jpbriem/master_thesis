To solve this problem, we can use a hash set to keep track of the characters that have already been seen. We iterate over the input string and add each character to the hash set if it hasn't been seen before. If a character has already been seen, we skip it.
Here's the Python code to implement this approach:

```python
def group_chars(input):
    seen = set()
    output = []
    for char in input:
        if char not in seen:
            output.append(char)
            seen.add(char)
    return output
```
