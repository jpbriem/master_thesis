```python
def longest_common_subsequence(input):
    n = len(input)
    dp = [[0] * (n + 1) for _ in range(n + 1)]

    # fill the first row and column with 0s
    for i in range(1, n + 1):
        dp[i][0] = 0

    for j in range(1, n + 1):
        dp[0][j] = 0

    # fill the rest of the matrix by comparing characters
    for i in range(1, n + 1):
        for j in range(1, n + 1):
            if input[j - 1] == input[i - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    result = []
    i, j = n, n
    while i > 0 and j > 0:
        if input[i - 1] == input[j - 1]:
            result.append(input[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] > dp[i][j - 1]:
            i -= 1
        else:
            j -= 1

    return ''.join(result[::-1])
```

The time complexity of this algorithm is O(mn), where m and n are the lengths of the two input strings. The space complexity is also O(mn).
