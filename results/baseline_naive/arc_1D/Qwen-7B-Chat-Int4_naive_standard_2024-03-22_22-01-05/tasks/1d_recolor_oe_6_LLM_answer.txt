Here's the Python code to implement this solution:

```python
def replace_chars(input_array):
    output_array = []
    for i in range(len(input_array)):
        if input_array[i] == "e":
            if i > 0 and input_array[i-1] == "e":
                output_array.append("a")
            else:
                output_array.append("g")
        elif input_array[i] == "g":
            if i > 0 and input_array[i-1] == "g":
                output_array.append("a")
            else:
                output_array.append("e")
        else:
            output_array.append(input_array[i])
    return output_array
```
