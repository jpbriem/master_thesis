To solve this problem, we can use a hash set to keep track of the characters that have already been encountered in the input string. We iterate through each character in the input string and check if it is already in the hash set. If not, we add it to the hash set and append it to the output string. This way, we ensure that all unique characters are included in the output string.

Here's an implementation of the solution in Python:

```python
def group_chars(input):
    seen = set()
    output = []
    for char in input:
        if char not in seen:
            seen.add(char)
            output.append(char)
    return output
```
