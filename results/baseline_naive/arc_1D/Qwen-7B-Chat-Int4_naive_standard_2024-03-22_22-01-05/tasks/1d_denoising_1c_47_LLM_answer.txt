We iterate through each character in the input string and check if it is already in the hash set. If not, we add it to the hash set and append it to the output list. ```python
def group_chars(input):
    char_set = set()
    output = []
    for char in input:
        if char in char_set:
            continue
        else:
            char_set.add(char)
            output.append(char)
    return output
