To solve this problem, we can use a hash set to keep track of the characters that have been seen so far. We iterate through each character in the input string and add it to the hash set if it has not been seen before. If a character has already been seen, we simply ignore it.

Here is an implementation in Python:

```python
def remove_duplicates(input_string):
    seen = set()
    output = []
    for char in input_string:
        if char not in seen:
            seen.add(char)
            output.append(char)
    return output
```

This function takes an input string as its argument and returns a new list with all duplicates removed. The time complexity of this solution is O(n), where n is the length of the input string, because we only need to iterate through the string once.
