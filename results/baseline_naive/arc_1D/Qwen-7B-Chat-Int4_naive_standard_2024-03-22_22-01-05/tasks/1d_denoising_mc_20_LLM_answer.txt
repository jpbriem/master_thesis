```python
def longest_common_subsequence(input):
    # Create a matrix of size (len(input) + 1) x (len(input[0]) + 1)
    m = [[0] * (len(input[0]) + 1) for _ in range(len(input) + 1)]
    
    # Fill the matrix with the length of the longest common subsequence
    for i in range(1, len(input)):
        for j in range(1, len(input[0])):
            if input[i - 1] == input[j - 1]:
                m[i][j] = m[i - 1][j - 1] + 1
            else:
                m[i][j] = max(m[i - 1][j], m[i][j - 1])
                
    # Find the index of the maximum value in the last row and column of the matrix
    start = max(range(len(input)), key=lambda i: m[-1][i])
    end = min(range(len(input[0])), key=lambda i: m[i][-1])
    
    # Return the longest common subsequence from the start index to the end index
    return input[start - 1:end]
```

The time complexity of this solution is O(nm), where n is the number of elements in the first array and m is the length of the second array. The space complexity is also O(nm).
