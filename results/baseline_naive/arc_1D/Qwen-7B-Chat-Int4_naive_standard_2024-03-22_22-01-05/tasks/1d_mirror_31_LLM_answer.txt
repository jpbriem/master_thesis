To solve this problem, we can use a hash set to keep track of the characters that have already been encountered. We iterate through each character in the input string and add it to the hash set if it hasn't been seen before. If the character has already been seen, we simply skip it.

Here's an implementation in Python:

```python
def group_unique_chars(input):
    output = []
    seen = set()
    
    for char in input:
        if char not in seen:
            output.append(char)
            seen.add(char)
            
    return output
```

This solution has a time complexity of O(n), where n is the length of the input string. The space complexity is also O(n) because we need to store all the unique characters in the output list.
