To solve this problem, we can use a hash table to keep track of the frequency of each character in the input string. We will iterate through the input string and update the count for each character in the hash table.
Here is the Python code that implements this approach:

```python
def group_chars(input_string):
    # Create an empty hash table
    char_counts = {}

    # Iterate through the input string
    for char in input_string:
        # If the character is already in the hash table, increment its count
        if char in char_counts:
            char_counts[char] += 1
        else:
            # Otherwise, add the character to the hash table with a count of 1
            char_counts[char] = 1

    # Sort the characters by their counts in descending order
    sorted_chars = sorted(char_counts.items(), key=lambda x: x[1], reverse=True)

    # Return the sorted characters as a list
    return [char[0] for char in sorted_chars]
```
