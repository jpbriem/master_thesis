```python
def removeDuplicates(input):
    char_count = {}
    output = []
    
    for char in input:
        if char not in char_count:
            char_count[char] = 0
            
        char_count[char] += 1
    
    for char, count in char_count.items():
        if count >= 2:
            output.append(char)
            
    return output
```

Then, we iterate through the input string using a for loop. For each character, we check if it is already in the `char_count` dictionary. If it is not, we add it to the dictionary with a count of 0. Otherwise, we increment its count by 1.

After updating the counts in the `char_count` dictionary, we iterate through the dictionary again using another for loop. This time, we check if the count of each character is greater than or equal to 2. If it is, we append it to the `output` list.

Finally, we return the `output` list containing only the unique characters found in the input string.
