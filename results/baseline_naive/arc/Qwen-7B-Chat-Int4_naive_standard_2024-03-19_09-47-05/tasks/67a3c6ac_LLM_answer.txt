```python
class Solution:
    def groupAnagrams(self, strs: List[List[str]]) -> List[List[str]]:
        d = {}
        
        for s in strs:
            # Convert string to set of characters and sort them
            chars = sorted(set(s))
            
            # Check if the character set already exists in dictionary
            if chars in d:
                # If it does, append the string to existing list
                d[chars].append(s)
            else:
                # Otherwise, create a new list and add the string
                d[chars] = [s]
                
        return [[d[c]] for c in d]
```

Explanation:

  * We initialize an empty dictionary `d`.
  * For each input string `s`, we convert it to a set of unique characters and sort them.
  * Then we check if the sorted character set is already present in the dictionary. If it is, we append the string to the existing list associated with that character set. If not, we create a new list and add the string to it.
  * Finally, we return a list of lists, where each inner list contains all the strings with the same sorted character set.

This solution has a time complexity of O(n*mlogm), where n is the number of input strings and m is the maximum length of any input string. The space complexity is also O(m).
