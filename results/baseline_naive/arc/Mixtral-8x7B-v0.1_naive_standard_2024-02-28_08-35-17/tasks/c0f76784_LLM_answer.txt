If path exists then we update our result array by adding new value at indexes corresponding to current position in original matrix. We do this until there are no other valid moves or when we reach end point which means that solution was found successfully!

Here is my implementation using python language :

```
def solve(matrix):
    m = len(matrix) # number rows
    n = len(matrix[0]) #number columns
    res = [] * m*n #result list initialized empty
    def dfs(i,j,path):
        if i == m-1 and j==n-1:#if reached last cell return true else continue searching next move
            print("found")
            res[i][j] += int(path[-1])+int(matrix[i][j])
            return True
        elif i>=m or j >=n or matrix[i][j]!= '.':return False
        else:
            res[i][j]=int(path[-1]) + int(matrix[i][j])
            for k in range(-1,2):
                for l in range(-1,2):
                    if abs(k)+abs(l)>1:continue
                    if dfs(i+k,j+l,"{}{}".format(path,-1)):break
            return True
    dfs(0,0,'')
    return res
print(solve([['.','.','.','.'],['.','.','.','.'],['.','.','.','.'],['.','.','.','.']]))
