## Answer (3)

You can use a recursive function to find all possible paths from top-left corner to bottom right. If you reach at any point where there are no mines in its adjacent cells then mark that cell as visited and call the same function with new coordinates. You need to keep track of maximum number of consecutive zeros encountered so far while traversing through these paths. Finally return this max count when we have reached the destination.

Here's how it looks like -

```
def dfs(i, j, grid):
    if i == len(grid)-1 and j == len(grid[0])-1: # base condition
        return 1
    elif i >= len(grid) or j >= len(grid[0]):
        return 0
    else:
        count = 0
        if grid[i][j]!= '*':
            grid[i][j] = '.'
            count += dfs(i+1, j, grid) + dfs(i, j+1, grid)
            grid[i][j] = 'x'
        return count

def maxZeroes(grid):
    m = float('-inf')
    for i in range(len(grid)):
        for j in range(len(grid[0])):
            if grid[i][j]!= '*':
                grid[i][j] = 'x'
                m = max(m, dfs(i, j, grid))
                grid[i][j] = 'x'
    return m
