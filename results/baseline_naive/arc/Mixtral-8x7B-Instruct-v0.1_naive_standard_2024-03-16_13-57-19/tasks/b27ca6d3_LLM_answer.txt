## Answer (4)

You can use `numpy.pad()` to pad all sides of the array with zeros and then iterate over each cell in the original matrix using a nested loop. If you find a non-zero value at that position, increment its corresponding row and column indices by one. Then set the new values as the current cell's value.

Here is an example implementation:

```
import numpy as np

def floodfill(matrix):
    m = len(matrix)
    n = len(matrix[0])
    padded = np.pad(matrix, ((1, 1), (1, 1)), 'constant')
    for i in range(m):
        for j in range(n):
            if matrix[i][j]!= 0:
                padded[i + 1][j + 1] += max(padded[i][j + 1], padded[i + 1][j])
    return padded[:-1, :-1]
```
