I think you can simplify this a bit by using `numpy`. Here I use `np.pad()` to pad the array with zeros and then iterate over each row in the original matrix. For each row we find all indices of non-zero elements and add them to our result at their corresponding position. We also need to make sure that if there are multiple non-zero values on one row they don't overlap. This can be done by adding an offset equal to the number of previous non-zero values.

```
import numpy as np

def flood_fill(matrix):
    m = len(matrix)
    n = len(matrix[0])
    padded = np.pad(matrix, ((1, 1), (1, 1)), mode='constant')
    res = np.zeros((m + 2, n + 2))
    for i in range(1, m + 1):
        idx = []
        for j in range(1, n + 1):
            if matrix[i - 1][j - 1]!= 0:
                idx.append(j)
        if not idx:
            continue
        idx = sorted(idx)
        start = idx[0]
        end = idx[-1]
        res[i, start:end+1] = 3
        for k in range(len(idx)-1):
            res[i, idx[k]+1:idx[k+1]] = 3
    return res
```
