## Answer (4)

You can use `numpy.where()` to find where the water should be placed and then fill it with a single call using `numpy.fill_diagonal()`. Here's an example implementation that works on all of your examples:

```
import numpy as np

def pourWater(heights):
    h = np.array(heights)
    w = np.zeros((len(h), len(h)))
    # Find indices where there are higher blocks above
    idx = np.argmax(np.triu(h[::-1]), axis=0)[::-1]
    # Fill in those spots
    np.fill_diagonal(w[:idx[-1]+1][::-1], 1)
    return heights if not np.any(w) else list(map(list, w))
```

This solution has time complexity O(N^2) because we need to search through N rows for each column. However, since you have specified that N <= 100, this will still run very quickly.
