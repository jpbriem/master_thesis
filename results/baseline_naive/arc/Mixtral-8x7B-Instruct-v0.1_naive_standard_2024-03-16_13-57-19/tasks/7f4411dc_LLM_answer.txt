## Answer (4)

You can use a recursive function to check if there are any non-zero elements in each of the four directions. If so, then you have not reached an edge and should continue searching. Otherwise, return True since we've found an edge. Here's how that would look like:

```
def hasEdge(grid, x, y):
    # Base cases - out of bounds or zero value
    if x < 0 or y < 0 or x >= len(grid) or y >= len(grid[x]) or grid[x][y] == 0:
        return False

    # Check all four directions
    return hasEdge(grid, x + 1, y) or \
           hasEdge(grid, x - 1, y) or \
           hasEdge(grid, x, y + 1) or \
           hasEdge(grid, x, y - 1)

def fillEdges(grid):
    n = len(grid)
    m = len(grid[0])

    # Fill edges with zeros
    for i in range(n):
        if hasEdge(grid, i, 0):
            break
        grid[i][0] = 0

    for j in range(m):
        if hasEdge(grid, 0, j):
            break
        grid[0][j] = 0

    for i in reversed(range(n)):
        if hasEdge(grid, i, m - 1):
            break
        grid[i][m - 1] = 0

    for j in reversed(range(m)):
        if hasEdge(grid, n - 1, j):
            break
        grid[n - 1][j] = 0
```

## Answer (1)

I am assuming that the grid is rectangular. In other words, its length and width are both positive integers. Also, I assume that the grid contains only nonnegative integers.

The following code uses two passes through the grid. First, it finds the minimum number of rows and columns required to contain all the nonzero entries. Then, it fills the outermost rows and columns with zeros until they reach the desired size.

```
def min_rows(grid):
    """Return the minimum number of rows needed to hold all nonzero values."""
    num_nonzero = sum(sum(row) for row in grid)
    return max(num_nonzero // cols + int(bool(num_nonzero % cols)), 1)

def min_cols(grid):
    """Return the minimum number of columns needed to hold all nonzero values."""
    num_nonzero = sum(sum(row) for row in grid)
    return max(num_nonzero // rows + int(bool(num_nonzero % rows)), 1)

def fill_edges(grid):
    """Fill the edges of the grid with zeros."""
    rows = min_rows(grid)
    cols = min_cols(grid)
    for row in grid[:rows]:
        row[:cols] = [0] * cols
    for col in zip(*grid)[:cols]:
        col[:rows] = [0] * rows
```

This algorithm runs in O(mn) time where m and n are the dimensions of the grid.

--------------------

If you prefer a more functional style, you might enjoy the following version which also takes advantage of Python's matrix multiplication operator.

```
import numpy as np

def min_rows(grid):
    """Return the minimum number of rows needed to hold all nonzero values."""
    num_nonzero = sum(sum(row) for row in grid)
    return max(num_nonzero // cols + bool(num_nonzero % cols), 1)

def min_cols(grid):
    """Return the minimum number of columns needed to hold all nonzero values."""
    num_nonzero = sum(sum(row) for row in grid)
    return max(num_nonzero // rows + bool(num_nonzero % rows), 1)

def fill_edges(grid):
    """Fill the edges of the grid with zeros."""
    rows = min_rows(grid)
    cols = min_cols(grid)
    arr = np.array(grid)
    arr[:, :cols] = 0
    arr[:rows, :] = 0
    grid[:rows] = arr[:rows].tolist()
    grid[:] = [list(col) for col in zip(*arr)]
```
