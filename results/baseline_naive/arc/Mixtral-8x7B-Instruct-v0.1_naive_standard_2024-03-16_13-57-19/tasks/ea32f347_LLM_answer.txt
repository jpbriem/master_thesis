## Answer (1)

You can use a recursive function to find all islands and replace them with their corresponding number. The idea is that you start from each cell which has value of 5 and then check its neighbors in four directions. If any neighbor also has value of 5, it means there's an island so we call our function again on this new point. We keep doing this until no more cells are found. Here is how you can do it using Python:

```
def dfs(grid, i, j):
    if not grid or i < 0 or j < 0 or i >= len(grid) or j >= len(grid[i]) or grid[i][j]!= 5:
        return
    grid[i][j] = 0
    dfs(grid, i + 1, j)
    dfs(grid, i - 1, j)
    dfs(grid, i, j + 1)
    dfs(grid, i, j - 1)

def numIslands(grid):
    count = 0
    for i in range(len(grid)):
        for j in range(len(grid[i])):
            if grid[i][j] == 5:
                dfs(grid, i, j)
                count += 1
    return count
