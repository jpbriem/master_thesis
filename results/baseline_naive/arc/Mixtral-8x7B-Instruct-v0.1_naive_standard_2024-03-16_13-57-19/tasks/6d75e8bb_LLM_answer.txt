## Answer (4)

You can use a recursive function to flood fill. The idea is that you start at some point and then look in all directions from there until you find an edge or have visited every cell around your current one. If you've found an edge, you mark it as such and return. Otherwise, if you haven't visited any cells yet, you call yourself with those new coordinates. Here's how I would implement this:

```
def flood(grid, x, y):
    # Base cases - we're out of bounds or on an edge
    if not grid[x][y] or x < 0 or x >= len(grid) or y < 0 or y >= len(grid[0]):
        return

    # Mark our starting position as visited
    grid[x][y] = 2

    # Recursively visit neighbors
    flood(grid, x + 1, y)
    flood(grid, x - 1, y)
    flood(grid, x, y + 1)
    flood(grid, x, y - 1)

# Start by visiting the first non-edge cell
for i, row in enumerate(grid):
    if 8 in row:
        j = row.index(8)
        flood(grid, i, j)
        break
```

This will modify the original array so make sure to copy it beforehand if you need to keep the original intact. Also note that this assumes that the input is valid; specifically, that there are no invalid indices. You might want to add checks for these conditions depending on where your data comes from.
