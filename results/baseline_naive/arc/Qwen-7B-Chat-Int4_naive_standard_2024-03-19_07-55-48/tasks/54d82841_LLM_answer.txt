```python
class Solution:
    def islands(self, grid: List[List[int]]) -> int:
        # check if any island is connected to the boundary
        for i in range(len(grid)):
            for j in range(len(grid[0])):
                if grid[i][j] == 1 and (i != 0 or j != 0):
                    return self.islands(grid[:i+1]+grid[i+1:j-1]+grid[j-1:])
        
        # initialize visited matrix with False
        visited = [[False]*len(grid) for _ in range(len(grid[0]))]
        
        # mark all cells that are part of an island as True
        for i in range(len(grid)):
            for j in range(len(grid[0])):
                if grid[i][j] == 1:
                    visited[i][j] = True
        
        # count number of islands
        count = 0
        for i in range(len(grid)):
            for j in range(len(grid[0])):
                if visited[i][j]:
                    count += 1
                    
        return count
```

This solution uses a depth-first search approach to find all islands in the given grid. Then it counts the number of True values in the visited matrix, which gives the total number of islands. This solution has a time complexity of O(n*m), where n and m are the dimensions of the grid.
