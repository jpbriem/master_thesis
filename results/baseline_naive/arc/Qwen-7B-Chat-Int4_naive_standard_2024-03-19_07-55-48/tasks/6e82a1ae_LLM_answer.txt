In this solution, we use recursion to solve the problem. We first check if the length of the input list is 1, which means there's only one element left to be permuted. In that case, we simply append the single element to the result list.

If the length of the input list is greater than 1, we iterate through each element in the list. For each element, we remove it from the list, then recursively permute the remaining elements. After permuting the remaining elements, we insert the removed element back into its original position in the list, and then permute the new list with the remaining elements. Finally, we extend the result list with the permuted list.

This solution has a time complexity of O(n!), where n is the length of the input list. The space complexity is also O(n!) because we need to store all possible permutations of the input list.
