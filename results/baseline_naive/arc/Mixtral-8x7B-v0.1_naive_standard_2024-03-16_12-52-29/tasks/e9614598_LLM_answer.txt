## Answer (4)

You can use `itertools.groupby()`, which will group consecutive elements of an iterable by their value. In this case we want groups where all values are zeroes; if there's any nonzero element in the group then we know they must be connected together somehow. We also need to keep track of how many rows each island spans - for example, consider these two islands:

```
[[0, 0, 0]
 [0, 1, 0]
 [0, 0, 0]]
```

and

```
[[0, 0, 0]
 [0, 1, 0]
 [0, 0, 1]]
```

The first one has three zeros above and below its only cell, while the second one has four because it extends into another row. So when processing our list of lists, we count upwards from both ends until we find either a nonzero number *or* reach the end of the matrix. This gives us the height of the island at that point. Then we do the same thing horizontally. Finally, since we don't care about individual cells but rather whole islands, we replace every single cell within those bounds with the maximum size found anywhere inside them. Here's what I came up with:

```
from collections import defaultdict
import itertools as IT

def max_island(matrix):
    # Find out how big each "island" of zeroes is
    heights = []
    widths = []
    for i, row in enumerate(matrix):
        h = len([x for x in IT.takewhile(lambda y: sum(y), zip(*row))])
        w = len([x for x in IT.takewhile(lambda y: sum(y), reversed(zip(*row)))])
        heights += [h]*w + [-1]*(len(row)-w)
        widths += [w]*h + [-1]*(len(row[i])-h)

    # Replace everything on the map with the largest island containing it
    return [[max((heights[j]+widths[k]), key=abs) for k, j in enumerate(r)] for r in matrix]
```
