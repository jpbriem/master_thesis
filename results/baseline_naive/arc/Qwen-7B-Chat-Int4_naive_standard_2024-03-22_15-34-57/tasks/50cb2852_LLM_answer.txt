```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        if not strs:
            return []
        
        d = {}
        
        for s in strs:
            sorted_s = "".join(sorted(s))
            if sorted_s in d:
                d[sorted_s].append(s)
            else:
                d[sorted_s] = [s]
                
        groups = []
        
        for k, v in d.items():
            groups.append(v)
            
        return groups
```

This solution uses a dictionary to store the anagram groups. It iterates over each string in `strs`, sorts its characters, and adds it to the dictionary with its sorted version as the key. If the key already exists in the dictionary, it appends the string to the list associated with that key; otherwise, it creates a new list containing the string.

After all strings have been processed, the solution returns the list of lists representing the anagram groups. Note that this solution has a time complexity of O(n * m), where n is the number of strings and m is the length of the longest string. This is because sorting each string takes O(m) time, and iterating over all strings takes O(n).
