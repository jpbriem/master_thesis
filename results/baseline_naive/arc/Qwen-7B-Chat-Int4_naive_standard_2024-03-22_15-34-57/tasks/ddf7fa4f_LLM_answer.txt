```python
def group_characters(input):
    groups = []
    for i, row in enumerate(input):
        if len(row) == 0 or row[0] != input[i-1][0]:
            groups.append([row])
        else:
            groups[-1].append(row)
    
    return [group for group in groups if group]
```

This function takes a list of lists as input and returns a list of lists containing the grouped characters. The idea is to iterate over each character in the input and check whether it matches the first character in the previous group. If not, create a new group with that character. If yes, append the current character to the previous group.

The time complexity of this algorithm is O(n*m), where n is the number of rows in the input and m is the maximum length of any row. This is because we need to iterate over all characters in the input once, and for each character, we may need to iterate over some (or all) of the previous groups. The space complexity is also O(m*n), since we need to store the output groups in memory.
