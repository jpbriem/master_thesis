You can use `map()` to apply a function to each element of an array, then `reduce()` to combine them into a single result.

Here's one way you could do it:

```
function transform(arr) {
  return arr.map(row => row.map(x => x + 1))
            .reduce((a, b) => [...a,...b]);
}
```
