The problem statement asks to remove all the islands of characters that are connected, meaning they share a side or corner. To solve this task, we can use Depth First Search (DFS) algorithm to traverse the grid and mark visited cells as we go along.

Here's an implementation in Python:

```python
def removeIslands(grid):
    def dfs(i, j, c):
        if i < 0 or j < 0 or i >= len(grid) or j >= len(grid[0]) or grid[i][j] != c or visited[i][j]:
            return

        visited[i][j] = True
        for dx, dy in [(0, -1), (-1, 0), (0, 1), (1, 0)]:
            dfs(i + dx, j + dy, c)

    m, n = len(grid), len(grid[0])
    visited = [[False] * n for _ in range(m)]

    # Mark all land cells as visited
    for i in range(m):
        for j in range(n):
            if grid[i][j] != '.':
                dfs(i, j, grid[i][j])

    # Remove unvisited land cells
    for i in range(m):
        for j in range(n):
            if not visited[i][j]:
                grid[i][j] = '.'

    return grid
```
