You can use a recursive function to fill the grid with `'a'`. The idea is that if there are no more `'e'`, then we return the current state of the grid, otherwise we find an index where `'e'` exists in the first row, replace it by `'a'`, and call the same function on the sub-grid obtained by removing the first line and column from the original one. Here is how this could be implemented:

```
def flood(grid):
    def _flood(g, i=0, j=0):
        while 0 <= i < len(g) and 0 <= j < len(g[i]) and g[i][j]!= 'e':
            i += 1; j += 1
        if i == len(g) or j == len(g[i]): # No more e found
            return g
        else:
            g[i][j] = 'a'; g[0][j] = 'a'
            return _flood([row[:j]+row[j+1:] for row in (g[1:] + [[None]])[1:]], 1, 0)
    return _flood(grid)
```
