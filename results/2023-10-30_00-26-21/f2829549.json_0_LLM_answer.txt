Answer: You can use `np.argmax` to get the index of the maximum value for a given axis. For example, if we have an array with shape `(4, 6)` then `np.argmax(array, axis=0)` will return the indices of the maximum values along the rows (i.e., the columns). Similarly, `np.argmax(array, axis=1)` will return the indices of the maximum values along the columns (i.e., the rows):

\begin{code}
import numpy as np

a = np.random.randint(0, 9, size=(4, 6))
print(a)
# [[8 2 7 3 8]
#  [8 2 7 3 8]
#  [8 2 7 3 8]
#  [8 2 7 3 8]]

print(np.argmax(a, axis=0))
# [1 1 1 1]

print(np.argmax(a, axis=1))
# [0 0 0 0]
\end{code}
