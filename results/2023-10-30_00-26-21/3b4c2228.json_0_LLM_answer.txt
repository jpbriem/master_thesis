Answer: You are using `np.random` which returns a random number in each call. This means that your training data will be different every time you run this program. If you want to use the same training data for multiple runs, then you need to store them somewhere (e.g., as a numpy array).

\begin{code}
import numpy as np
from sklearn import datasets

X = []
y = []
for i in range(1000):

    y.append(np.random.randint(0, 3))

y = np.array(y)
