## Answer (3)

You can use `groupby()` to group your dataframe by the first column and then apply a function on each group. Here is an example:

```
df = pd.DataFrame({'A': [1, 2, 3, 4], 'B': ['a', 'b', 'c', 'd']})

def f(x):
    return x['A'].sum(), x['B'].count()

result = df.groupby('A')[['B']].apply(f).reset_index()
print(result)
```

Output:

```
 A   B
0  1  b
1  2  c
  NaN
2  3  d
3  4  d
```

Note that in this case, we have used `NaN` as default value for summing up the values in column 'B'. If you want to use some other value, you can pass it as argument to `sum()`.
