```
import numpy as np

arr = np.array([[0, 2, 2, 0], [2, 0, 0, 2], [0, 2, 0, 2], [2, 2, 2, 2]])
threshold = 5
new_arr = np.where(arr > threshold, -1, arr)
print(new_arr)
```
