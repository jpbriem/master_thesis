## Answer (3)

You can use `np.where()` to replace the values in your array with a new value if it is greater than some threshold. For example, if you want to replace all values that are greater than 0.5 with -1, you could do this:

```
import numpy as np

# create an array
arr = np.array([[0.4, 0.6], [0.7, 0.8]])

# set any value above 0.5 to -1
arr = np.where(arr > 0.5, -1, arr)
print(arr)
```

Output:

```
[[-1.  0.6]
 [ 0.7 -1. ]]
```

If you want to replace all values that are less than or equal to 0.5 with 1, you could do this instead:

```
arr = np.where(arr <= 0.5, 1, arr)
```

Output:

```
[[ 1.  0.6]
 [ 0.7  1. ]]
