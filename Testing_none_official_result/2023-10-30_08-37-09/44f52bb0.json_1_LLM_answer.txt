You can use `numpy` to reshape your data and then use `scipy.stats.ttest_ind`. Here is an example:

```
import numpy as np
from scipy.stats import ttest_ind

# create some random data
data = np.random.randint(0, 4, size=(5, 3))
labels = np.random.randint(0, 2, size=5)

# reshape data
data = data.reshape(-1, 3)

# perform t-test
tstatistic, pvalue = ttest_ind(data[:, labels == 0], data[:, labels == 1])
print(pvalue)
```

```
[0.98611111e+30]
```
