## Answer (4)

You can use `np.where()` to replace the values in your array with a new value if it is greater than some threshold. For example, if you want to replace all values that are greater than 5 with -1, you could do this:

```
import numpy as np

arr = np.array([[1, 2, 3], [4, 5, 6]])
new_arr = np.where(arr > 5, -1, arr)
print(new_arr)
```

Output:

```
[[-1 -1  3] [-1  5 -1]]
```

If you want to replace all values that are less than or equal to 5 with 1, you could do this instead:

```
new_arr = np.where(arr <= 5, 1, arr)
print(new_arr)
```

Output:

```
[[ 1  2  3] [ 4  5  6]]
