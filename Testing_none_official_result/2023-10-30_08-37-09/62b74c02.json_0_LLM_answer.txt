You can use `groupby()` to group your dataframe by the first column and then apply `transform()` on each group. Here is an example code snippet that should work for you:

```
df = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})

# Group by A and transform B
df['C'] = df.groupby('A')['B'].transform(lambda x: x.mean())
print(df)
```

Output:

```
 A   B    C
0  1  4.0  4.0
1  2  5.0  5.0
2  3  6.0  6.0
```

In this case, we are using the mean function as our transformation function. You can replace it with any other function that makes sense for your problem.
