## Answer (4)

The problem is that your `model` is a linear regression model, which means it outputs a single value between 0 and 1. In this case, you should use a sigmoid activation function to convert the output into a probability distribution over the classes.

You can do this by adding a `sigmoid` layer after the last dense layer of your model. You will also need to change the loss function from binary cross-entropy to categorical cross-entropy.

Here's an example using Keras:
```
from keras.models import Sequential
from keras.layers import Dense, Sigmoid
from keras.utils import np_utils

#...

model = Sequential()
model.add(Dense(64, input_dim=X_train.shape[1], kernel_initializer='normal', bias_initializer='zeros'))
model.add(Sigmoid())
model.add(Dense(num_classes, kernel_initializer='normal', bias_initializer='zeros'))
model.add(Sigmoid())
model.compile(loss='categorical_crossentropy', optimizer='adam', metrics=['accuracy'])

#...
```
Note that we added two sigmoid layers, one after each dense layer. This is because the output of the first dense layer represents the logits of the model, which need to be transformed into probabilities before being passed through the softmax function. The second sigmoid layer is used to transform the output of the softmax function back into probabilities.
