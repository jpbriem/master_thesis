## Answer (1)

You can use `apply()` to apply a function on each row and then `groupby()` to group them by their labels.

Here is an example:

```
df = pd.DataFrame({'label': ['A', 'B', 'C', 'D'],
                   'input': [['a','b','c','d','e','f'],
                             ['g','h','i','j','k','l'],
                             ['m','n','o','p','q','r'],
                             ['s','t','u','v','w','x']]})

def f(row):
    return row['input'].tolist()

df_grouped = df.groupby('label')['input'].apply(f).reset_index()
print(df_grouped)
```

Output:

```
 label         input
0     A  [a, b, c, d, e, f]
1     B  [g, h, i, j, k, l]
2     C  [m, n, o, p, q, r]
3     D  [s, t, u, v, w, x]
```
