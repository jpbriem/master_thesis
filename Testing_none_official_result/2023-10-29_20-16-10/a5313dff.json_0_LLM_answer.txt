Answer: You can use [`np.diff`](https://docs.scipy.org/doc/numpy-1.15.0/reference/generated/numpy.diff.html) to get the differences between consecutive elements in each column (axis=1), then check if they are equal to one or zero using [`np.isclose(a, b)`](https://docs.scipy.org/doc/numpy-1.14.3/user/basics.comparison.html#available-functions). This will give a boolean array with True for every element that matches your condition. Then we just need to sum along axis 1 to count how many times this happens per column. Finally, we compare it to the number of nonzero values in each column (`nonzeros = np.sum(data!= 0)`, which gives us an array of booleans indicating whether all changes were either +/- 1 or not. We can finally apply `all()` on these arrays to see if all columns match our criteria.
