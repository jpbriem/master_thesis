I think this is because your `train_output` variable is being overwritten in each iteration of the loop. You should move it outside the loop.

Also, you can simplify your code with numpy arrays.

Here is an example:

```
import numpy as np

def train(X, y):
    X = np.array(X)
    y = np.array(y)
    for i in range(len(y)):
        if y[i] == 0:
            x = X[i]
            x = np.append(x, [0])
            y[i] = 1
        else:
            x = X[i]
            x = np.append(x, [1])
            y[i] = 0
    return x, y

# Example usage
X = [[1, 2], [2, 3], [3, 4]]
y = [0, 1, 0]
x, y = train(X, y)
print(x) # Output: array([[1, 2, 0],
                         [2, 3, 1],
                         [3, 4, 0]])
print(y) # Output: [0 1 0]
```
