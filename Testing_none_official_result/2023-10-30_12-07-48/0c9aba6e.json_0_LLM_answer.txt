## Answer (1)

You can use `np.where()` to create a mask for each column and then multiply it with your original dataframe.

Here is an example:

```
import numpy as np

df = pd.DataFrame({'A': [0, 2, 2, 0], 'B': [2, 0, 0, 0]})
masks = [[True if x > 0 else False for x in df['A']],
         [True if y > 0 else False for y in df['B']]]

df_new = np.where(masks[0], df['A'], 0).values[:, np.newaxis].T \
       .join(np.where(masks[1], df['B'], 0).values[:, np.newaxis])

print(df_new)
```
