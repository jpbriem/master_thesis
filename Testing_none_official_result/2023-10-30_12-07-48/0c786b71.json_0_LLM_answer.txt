I think this is because your `train_output` variable is being overwritten in each iteration of the loop. You should append to it instead of replacing it.

Try this:

```
def train(X, y):
    train_output = []
    for i in range(len(y)):
        X_train = np.array([[X[i][0], X[i][1]]])
        y_train = y[i]
        if y_train == 0:
            train_output.append([0, 0])
        else:
            train_output.append([1, 1])
    return train_output
```
