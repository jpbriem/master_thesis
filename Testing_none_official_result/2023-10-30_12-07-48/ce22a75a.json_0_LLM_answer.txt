The problem is that you are using `np.array()` to convert your numpy array into a list. This will create a new list with the same values as the numpy array but in a different order. You can fix this by using `numpy.tolist()` instead.

Here is an example:

```
import numpy as np

# Create numpy array
arr = np.array([[0, 0], [0, 0]])

# Convert numpy array to list
lst = arr.tolist()
print(lst) # Output: [[0, 0], [0, 0]]

# Convert numpy array to list without converting first
lst = np.array([[0, 0], [0, 0]])[:].tolist()
print(lst) # Output: [[0, 0], [0, 0]]
```
