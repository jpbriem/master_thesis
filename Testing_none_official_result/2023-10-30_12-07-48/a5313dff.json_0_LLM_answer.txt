The problem is that you are using `np.array()` to convert your numpy array into a list. This will create a new list with the same values as the numpy array. However, when you use `numpy.array(lst)`, it creates a new numpy array from the list.

So if you want to keep the original numpy array, you should use `numpy.array(lst)`. If you want to convert it to a list, then you can do `list(numpy.array(lst))`.

Here is an example:

```
import numpy as np

# Create numpy array
arr = np.array([[0, 0, 0], [0, 0, 0]])

# Convert numpy array to list
lst = arr.tolist()
print(lst) # Output: [[0, 0, 0], [0, 0, 0]]

# Convert list to numpy array
arr_new = np.array(lst)
print(arr_new) # Output: [[0, 0, 0] [0, 0, 0]]
```
