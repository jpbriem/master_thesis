You can use `apply()` to apply a function to each row in your data frame and then `summarise_all()` to summarize all columns.

Here is an example:

```
library(dplyr)
df %>%
  mutate(val = sum(x)) %>%
  summarise_all(mean)
# A tibble: 1 x 5
   val    x1    x2    x3    x4
  <dbl> <dbl> <dbl> <dbl> <dbl>
    1     1     1     1     1
```

  mutate(val = sum(x)) %>%
