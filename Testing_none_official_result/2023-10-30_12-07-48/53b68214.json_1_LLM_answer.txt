## Answer (4)

You can use `apply()` to apply a function to each row in your data frame and then `summarise_all()` to summarize all columns.

Here is an example using `dplyr`.

```
library(dplyr)
df %>%
  mutate(val = sum(x)) %>%
  summarise_all(mean)
# A tibble: 1 x 5
   val    x1    x2    x3    x4
  <dbl> <dbl> <dbl> <dbl> <dbl>
    1     1     1     1     1
```

If you want to keep the original rows as well, you can use `bind_rows()` instead of `summarise_all()`.

```
df %>%
  mutate(val = sum(x)) %>%
  bind_rows(data.frame(val = mean(x)))
# A tibble: 2 x 6
   val    x1    x2    x3    x4    x5
