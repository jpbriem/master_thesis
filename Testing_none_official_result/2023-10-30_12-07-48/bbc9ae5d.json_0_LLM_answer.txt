The problem is that you are using `np.array()` to convert your data into numpy arrays. This will create a new array with the same values as the original list. However, numpy arrays have their own memory management and are not directly comparable to lists.

To compare two numpy arrays you need to use `numpy.equal(a, b)`.

Here is an example:

```
import numpy as np

# Create some lists
lst1 = [1, 2, 3]
lst2 = [1, 2, 3]

# Convert them to numpy arrays
arr1 = np.array(lst1)
arr2 = np.array(lst2)

# Compare them
print(np.equal(arr1, arr2)) # True
```

If you want to compare lists in python you can use `==` operator. Here is an example:

```
lst1 = [1, 2, 3]
lst2 = [1, 2, 3]

print(lst1 == lst2) # True
```
