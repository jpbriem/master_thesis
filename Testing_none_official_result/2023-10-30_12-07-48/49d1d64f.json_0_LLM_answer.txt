I think this is because your `train_output` variable is being overwritten each time you call `predict`. You should create a new list for it instead.

Also, you can simplify your code with some list comprehensions:

```
def predict(X):
    train_output = []
    for i in range(len(X)):
        if X[i][0] == 1:
            train_output.append([0, 1])
        else:
            train_output.append([1, 0])
    return train_output
