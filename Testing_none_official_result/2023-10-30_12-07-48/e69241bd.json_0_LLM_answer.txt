The problem is that you are using `np.array()` to convert your lists into numpy arrays. This will create new arrays every time you call it. You should use `np.array()` only once for each list.

You can do this with a single line of code:

```
arr = np.array([[...],...])
```

This creates an array from all the elements in the list.

Also, you need to make sure that the number of columns in the input array matches the number of features in your model. In your case, you are using `X_train.shape[1]`, which means that the number of columns in the input array must be equal to the number of features in your model.
