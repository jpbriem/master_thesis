Here is an example:

```
import numpy as np

# Create some random data
data = np.random.rand(5,4)
masks = np.array([[True, False], [False, True]]) # You need to adjust this according to your needs

# Multiply masks with data
result = np.multiply(data, masks[:, :, None])
print(result)
```

```
