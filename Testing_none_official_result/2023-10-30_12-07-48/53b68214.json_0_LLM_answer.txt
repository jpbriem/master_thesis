The problem is that your `predict_proba` function returns an array with probabilities for each class. In this case, since there are only two classes, it will return an array with two elements. You can then use `argmax` to get the index of the highest probability.

However, in your test data, some samples have more than two classes. For instance, sample 5 has three classes. So when you call `predict_proba`, it will return an array with three elements. This means that `argmax` will always return 0, which corresponds to the first class.

To fix this, you need to modify your `predict_proba` function so that it returns an array with one element per class instead of an array with one element per sample. Here's how you could do it:

```
def predict_proba(X):
    y = model.predict(X)
    probs = np.zeros((y.shape[0], num_classes))
    for i in range(y.shape[0]):
        if y[i] == 0:
            probs[i][0] = 1 - y[i] / y.sum()
        else:
            probs[i][1-y[i]] = 1 - y[i] / y.sum()
    return probs
```
