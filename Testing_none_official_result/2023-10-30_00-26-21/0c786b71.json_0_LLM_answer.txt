Answer: \begin{blockquote}

I am trying to find out if there is any way to count the number of times that the same number occurs consecutively in a list.
\end{blockquote}

You can use `itertools.groupby` and `sum`:

\begin{code}
from itertools import groupby

def consecutive_count(lst):
    return sum((len(g) for k, g in groupby(enumerate(lst), key=lambda x:x[1])))
\end{code}
