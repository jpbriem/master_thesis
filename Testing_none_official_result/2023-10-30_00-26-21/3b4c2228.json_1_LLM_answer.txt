Answer: You can do this with a simple loop over all possible values for each feature and then check if they are present in the training set or not. For example:

\begin{code}
import numpy as np
from sklearn import datasets

iris = datasets.load_iris()
X = iris['data']
y = iris['target']

# create a list of all possible features
features = [x for x in range(len(X[0]))]

for f in features:
    # get all instances where feature == f
    indices = np.where(f == X[:, f])
    # count how many times we find that index
    n = len(indices)
    print('feature %s occurs %d times in training set' % (f, n))
\end{code}
