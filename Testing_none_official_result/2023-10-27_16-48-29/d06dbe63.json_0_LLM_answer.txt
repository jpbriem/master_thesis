```
## C[edit]

This is a simple implementation that uses the same approach as the Python version above. It does not use any external libraries and it's very slow (~1000x slower than the Python version).

```
#include <stdio.h>
#include <string.h>
#include <math.h>

int main(void) {
    int i;
    double x = 0.0;
    for (i=0; i<20; ++i) {
        printf("input %d:\n", i);
        for (int j=0; j<20; ++j) {
            scanf("%lf", &x);
            printf("%lf ", x);
        }
        printf("\n");
        printf("output:\n");
        for (int j=0; j<20; ++j) {
            scanf("%lf", &x);
            printf("%lf ", x);
        }
        printf("\n\n");
    }
}
