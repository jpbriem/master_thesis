```
## C[edit]

This is a simple implementation that uses the `rand()` function to generate random numbers between 0 and 9. It then generates a random number between 0 and 5 for each digit in the number. The result is printed as an integer with commas separating every three digits from right to left.

```
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void) {
    srand((unsigned int) time(NULL));
    printf("Random number generator\n");
    while (1) {
        int i;
        int n = rand();
        if (!n) break;
        for (i=0; i<6; ++i) {
            int d = rand() % 10;
            if (d == 0) --;
            else printf("%d", d);
            if ((i+1)%3==0 && i!=5) printf(", ");
        }
        putchar('\n');
    }
}
